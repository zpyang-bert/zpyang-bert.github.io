## 一、均衡技术基础

### 1.1 为什么需要均衡

高速信号在信道中传输时会受到损耗和色散影响，导致：

```
发送信号:    ■ ■ ■ ■ ■ ■ ■ ■    (理想眼图)
            │
            ▼ 经过信道
接收信号:    ▼▼▼▼▼▼▼    (ISI拖尾，眼图闭合)
```

**码间串扰 (ISI)**：相邻符号的能量重叠，导致判决错误

### 1.2 均衡的目标

**均衡目标**：
- **频域**：将频率响应平坦化至奈奎斯特频率，消除信道对各频率成分的不等增益
- **时域**：消除符号间干扰，使脉冲响应满足Nyquist准则
- **目标**：最大化眼图张开度，最小化误码率

```
信道均衡目标示意：
         均衡前                    均衡后
      ┌─────────┐             ┌─────────┐
  增益 │  ╱╲     │         增益 │─────────│
      │ ╱  ╲    │             │         │
      │╱    ╲   │             │         │
      └──────────────→       └──────────────→
             f                      f
      (不平坦，有衰减)         (平坦化)
```

### 1.3 均衡器分类

```
均衡技术
├── 线性均衡 (Linear Equalization)
│   ├── TX FFE (预加重/去加重)
│   ├── RX FIR (前向均衡)
│   └── CTLE (连续时间线性均衡)
├── 非线性均衡 (Non-linear Equalization)
│   ├── DFE (判决反馈均衡)
│   └── MLSE (最大似然序列估计)
└── 混合均衡
    └── FFE + DFE 联合
```

### 1.4 均衡有效性分析

**观察结果**（基于10Gbps实验）：

| 均衡策略 | 性能提升 |
|----------|---------|
| 2-tap TX EQ | 初始性能大幅提升 |
| 2-tap vs 4-tap TX EQ | 差距不大 |
| RX均衡（特别是DFE） | 进一步显著提升 |
| 仅TX均衡 | 提升有限 |

**注意事项**：由于关键时序路径限制，高速DFE实现困难

## 二 TX FFE (发送前向均衡器)

### 2.1 TX FIR 滤波器原理

TX FIR滤波器对传输脉冲进行预失真，以抵消信道失真，代价是发送信号衰减（去加重）：

```
        ┌─────────────────────────┐
数据 ──▶│ c[-1]    c[0]    c[1] │──▶ 发送
        │  pre    main   post   │
        │  tap     tap    tap   │
        └─────────────────────────┘
```

**传递函数**：
$$W(z) = c_{-1} \cdot z^{+1} + c_0 \cdot z^0 + c_{-1} \cdot z^{-1}$$

**时域解释**：
- 低频响应（所有Taps同极性）：直流增益降低
- 奈奎斯特频率响应（交替极性）：高频直通

### 2.2 去加重 (De-emphasis) 设计实例

典型3-tap FFE系数设计（10Gb/s，Refined Server Channel）：

$$W(z) = -0.131 + 0.595 \cdot z^{-1} - 0.274 \cdot z^{-2}$$

| 参数 | 值 |
|------|-----|
| Pre-tap | -0.131 |
| Main-tap | 0.595 |
| Post-tap | -0.274 |
| 低频增益 | -14.4dB（衰减） |
| 奈奎斯特增益 | 0dB（直通） |
| 频率峰值 | 14.4dB |

**频率响应**：
- 直流（f=0）：$W(f=0) = 0.190 \Rightarrow -14.4dB$
- 奈奎斯特（$f = 1/(2T_s)$）：$W(f_{Nyq}) = -1 \Rightarrow 0dB$

### 2.3 TX FIR系数选择：MMSE算法

**最小均方误差（MMSE）算法**是设计TX FIR系数的常用方法。

**系统方程**：
$$\mathbf{y} = \mathbf{H} \cdot \mathbf{W}_{ls}$$

其中：
- $\mathbf{y}$：信道输出向量
- $\mathbf{H}$：信道脉冲响应矩阵（$h_k$）
- $\mathbf{W}_{ls}$：MMSE最优TX均衡器抽头

**误差**：
$$E = \mathbf{Y} - \mathbf{Y}_{des} = \mathbf{H}\mathbf{W} - \mathbf{Y}_{des}$$

**最优抽头求解**（最小化误差范数平方）：
$$\frac{d}{d\mathbf{W}}\|E\|^2 = 2\mathbf{W}^T\mathbf{H}^T\mathbf{H} - 2\mathbf{Y}_{des}^T\mathbf{H} = 0$$

**MMSE最优抽头**：
$$\boxed{\mathbf{W}_{ls} = (\mathbf{H}^T\mathbf{H})^{-1}\mathbf{H}^T\mathbf{Y}_{des}}$$

**归一化**（为TX FIR实现）：
$$\boxed{\mathbf{W}_{ls,norm}(n) = \frac{\mathbf{W}_{ls}(n)}{\sum_{i=1}^{n}|\mathbf{W}_{ls}(i)|}}$$

### 2.4 TX FIR抽头分辨率

实际TX DAC分辨率通常限制在**4-6位**。

**实例**（10Gb/s Refined Server Channel）：

未归一化系数：
$$W_{ls} = [-0.8180, 0.7245, 0.5949, -1.7184]^T$$

归一化系数（直流增益=-14.4dB）：
$$W_{ls,norm} = [-0.1307, 0.5949, 0.5949, -0.2745]^T$$

**结论**：将均衡系数映射到有限分辨率会略微影响性能

### 2.5 TX FIR电路架构

#### 2.5.1 Direct FIR（直接FIR）

- 并联输出驱动器驱动各抽头
- 每个并联驱动器需按最大可能电流设计
- **优点**：低功耗、低复杂度
- **缺点**：输出电容较大

#### 2.5.2 Segmented DAC（分段数模转换器）

- 最小尺寸输出晶体管仅处理峰值总电流
- **优点**：最低输出电容、最灵活均衡
- **缺点**：最高功耗和复杂度、需要映射表（RAM）
- 各抽头在数字域合并，非输出端

**工作原理**：4-bit模式发送6-bit数值（并行滤波bit @ 6-bit分辨率）

#### 2.5.3 Voltage-Mode TX FIR驱动器

**问题**：由于串联阻抗，电压模式实现FIR均衡较困难

**解决方案**：输出分压器配合GND分流路径实现不同电平，同时维持阻抗控制

**设计1**（Wong JSSC 2004）：
- 输出分压器实现FIR电平
- 缺点：预分频逻辑复杂、VREF供电电流随时间变化

**设计2**（Sredojevic JSSC 2011）：
- 增加信道分流路径
- 实现FIR电平、维持阻抗控制、恒定VREF电流
- 缺点：输出分割预驱动逻辑更复杂

#### 2.5.4 混合电压模式驱动器（电流模式均衡）

**架构特点**：
- 主驱动器抽头采用电压模式（给定电压摆幅下降低电流）
- 并联电流模式驱动器实现高分辨率预加重均衡抽头（最小预驱动复杂度）
- 动态电流变化比原始VM TX FIR #1小

### 2.6 阻抗调制均衡

**特性**：
- 去加重增加时信号功耗降低
- 转换位（transition bit）保持50Ω阻抗
- 较长游程长度数据具有更高阻抗

## 三、RX FIR (接收前向均衡器)

### 3.1 工作原理

延迟模拟输入信号并乘以均衡系数：

```
模拟延迟线 ──▶ 乘法器 ──┐
                       ├─▶ 加法器 ──▶ 输出
数据延迟 ──▶ 乘法器 ──┘
```

### 3.2 优缺点分析

**优点**：
- 若动态范围足够，可放大高频内容（而非衰减低频）
- 可消除前标和超出滤波器范围的后标ISI
- 滤波器抽头系数可无需回传信道自适应调谐

**缺点**：
- 放大噪声和串扰
- 模拟延迟线实现困难
- 抽头精度限制

### 3.3 噪声增强问题

**关键问题**：线性RX均衡器不区分信号、噪声和串扰

- 虽然信失真比（ISI比）改善，但**SNR保持不变**
- 原因：均衡器对信号和噪声同等处理

### 3.4 模拟RX FIR均衡器实例

**5抽头均衡器，抽头间距$T_b/2$**（Hernandez-Garduno & Silva-Martinez ISSCC 2007）：

| 参数 | 指标 |
|------|------|
| 数据率 | 1 Gb/s |
| 工艺 | CMOS |
| 抽头数 | 5 |
| 间距 | $T_b/2$ |
| 滤波器类型 | 3阶延迟单元 |

**核心电路**：
- 浮电感仿真器和有源负载
- 3阶延迟单元实现

**实验结果**：23米CAT5e电缆，均衡后眼图显著张开

### 3.5 数字RX FIR均衡器

**架构**：高速低/中等分辨率ADC数字化输入，在数字域进行均衡

**组成**：数字延迟、乘法器、加法器

**限制**：ADC分辨率限制性能

**功耗**：由于高速ADC和数字滤波器，功耗可能较高

**实例**（Harwood ISSCC 2007）：

| 参数 | 指标 |
|------|------|
| ADC速度 | 12.5 GS/s |
| ADC分辨率 | 4.5-bit Flash |
| 工艺 | 65nm CMOS |
| FFE抽头 | 2 |
| DFE抽头 | 5 |
| 收发机功耗 | 330mW |
| 模拟功耗 | 245mW |
| 数字功耗 | 85mW |

## 四、CTLE (连续时间线性均衡器)

### 4.1 基本原理

无源R-C（或L）可实现高通传递函数以补偿信道损耗：

$$H(s) = \frac{R_2}{R_1+R_2} \cdot \frac{1+R_1C_1s}{1+\frac{R_1R_2}{R_1+R_2}(C_1+C_2)s}$$

**零点频率**：$\omega_z = \frac{1}{R_1C_1}$
**极点频率**：$\omega_p = \frac{1}{\frac{R_1R_2}{R_1+R_2}(C_1+C_2)}$

**直流增益**：$\frac{R_2}{R_1+R_2}$
**高频增益**：$\frac{C_1}{C_1+C_2}$

**峰值增益**：
$$\text{Peaking} = \frac{\omega_p}{\omega_z} = \frac{R_1+R_2}{R_2} \cdot \frac{C_1}{C_1+C_2}$$

### 4.2 无源CTLE

无源结构提供优异线性度，但在奈奎斯特频率无增益

**特点**：
- 无源RC网络实现
- 线性度好
- 无放大能力

### 4.3 有源CTLE（RC退化放大器）

**电路结构**：带RC退化的输入放大器提供频率峰值和奈奎斯特增益

**传递函数**：
$$H(s) = \frac{g_m}{C_p} \cdot \frac{s + \frac{1}{R_sC_s}}{(s + \frac{1+g_mR_s/2}{R_sC_s})(s + \frac{1}{R_DC_p})}$$

**零点**：$\omega_z = \frac{1}{R_sC_s}$
**第一极点**：$\omega_{p1} = \frac{1+g_mR_s/2}{R_sC_s}$
**第二极点**：$\omega_{p2} = \frac{1}{R_DC_p}$

**直流增益**：$\frac{g_mR_D}{1+g_mR_S/2}$
**理想峰值增益**：$g_mR_D$

**理想峰值**：
$$\text{Ideal Peaking} = 1 + \frac{g_mR_s}{2}$$

**设计考虑**：
- 放大器增益带宽积限制
- 需考虑输入线性范围
- 常与TX均衡配合（TX提供部分低频衰减）
- 对PVT变化敏感，调谐困难
- 通常限于1阶补偿

### 4.4 CTLE调谐方法

**调谐退化电阻和电容**：调节零频和1st极点设置峰值和直流增益

| 调节方式 | 效果 |
|----------|------|
| 增加$C_s$ | 零点+1st极向低频移动，峰值不变 |
| 增加$R_s$ | 零点向低频移动、增加峰值（降低直流增益），对1st极点影响小 |

### 4.5 CTLE优缺点总结

**优点**：
- 低功耗、低面积开销提供增益和均衡
- 可消除前标和长尾ISI
- 提供增益（这点与无源CTLE不同）

**缺点**：
- 通常限于1阶补偿
- 放大噪声和串扰
- PVT敏感性
- 难以调谐

## 五、DFE (判决反馈均衡器)

### 5.1 基本原理

DFE是非线性均衡器——判决器对输入符号进行量化（判决），然后通过反馈FIR滤波器直接从输入信号中减去ISI：

$$z_k = y_k - w_1 d_{k-1} - w_2 d_{k-2} - ... - w_{n-1} d_{k-(n-1)} - w_n d_{k-n}$$

```
判决器
  │
  ▼
d_k ──▶ 反馈抽头w₁, w₂... ──┐
                           │
输入 y_k ──────────────────┤──▶ 加法器 ──▶ z_k（判决前信号）
```

### 5.2 DFE优缺点

**优点**：
- 不放大噪声和串扰的高频增强
- 滤波器抽头系数可无需回传信道自适应调谐

**缺点**：
- 不能消除前标ISI（前标必须由其他均衡器处理）
- 错误传播风险（实际链路的BER=10⁻¹²概率很低）
- **关键反馈时序路径**：ISI消除的时序会复杂化CDR相位检测
- 关键时序路径：DFE加法器必须在1个比特周期内完成

### 5.3 DFE抽头系数设定

**纯DFE均衡**：DFE抽头系数应等于未均衡信道脉冲响应值
$$[w_1, w_2] = [a_1, a_2]$$

**有其他均衡器时**：DFE抽头系数应等于DFE前脉冲响应值
$$[w_1, w_2] = [a_1^{pre-DFE}, a_2^{pre-DFE}]$$

DFE提供灵活性——可优化TX均衡器而不必担心DFE会处理的ISI项

### 5.4 直接反馈DFE实例（TI）

| 参数 | 指标 |
|------|------|
| 数据率 | 6.25 Gb/s |
| DFE抽头 | 4 |
| 架构 | 半速率 |
| 算法 | 自适应抽头算法 |

**关键设计**：在½ UI内完成第1抽头时序收敛（同时收敛自适应均衡抽头值和CDR）

### 5.5 直接反馈DFE关键路径

**时序约束**：必须在1比特周期内完成数据解析和反馈

TI设计实际上在½ UI内完成（为CDR考虑）

### 5.6 DFE环路展开 (Loop Unrolling)

**原理**：不是1 UI内反馈和消除ISI，而是：

1. 展开环路
2. 预计算2种可能性（1-tap DFE）
3. 带可调 slicer 阈值

**数学表达**：
$$\tilde{d}_k = \begin{cases} \text{sgn}(y_k - w_1) & \text{if } \tilde{d}_{k-1} = 1 \\ \text{sgn}(y_k + w_1) & \text{if } \tilde{d}_{k-1} = -1 \end{cases}$$

**代价**：随抽头数增加，比较器数量以$2^{\#taps}$增长

### 5.7 DFE求和器设计

**重要性**：求和器性能对DFE运行至关重要

**要求**：求和器必须在一定精度水平（>95%）内建立以消除ISI

**权衡**：求和器输出摆幅 vs 建立时间

**结果**：输入和抽头可能需要大偏置电流

### 5.8 DFE积分求和器

**改进**：将电流积分到负载电容上，消除RC建立时间

**优势**：由于$\Delta T/C > R$，给定输出摆幅下偏置电流可减小

**典型改善**：偏置电流降低3倍

### 5.9 带反馈FIR滤波器的DFE

**限制**：带2-tap FIR反馈的DFE只消除前两个后标ISI

**"平滑信道"问题**：
- 带FIR反馈的DFE需要很多抽头来消除ISI
- 平滑信道的长尾ISI可近似为指数衰减
- 实例：片上线缆和硅载线缆

### 5.10 带IIR反馈的DFE

**设计**（Liu ISSCC 2009）：

| ISI类型 | 消除方式 |
|---------|----------|
| 大1st后标$H_1$ | 正常FIR反馈抽头 |
| 平滑长尾ISI（2nd后标之后） | 低通IIR反馈滤波器 |

**注意**：信道需平滑（无多径反射），此方法才有效

**架构要点**：
- 2 UI延迟通过mux实现（从2nd后标开始消除）
- 低通响应由$R_D$和$C_D$实现
- 幅度由$R_D$和$I_D$控制

### 5.11 合并求和器与部分Slicer

**设计**：带再生PMOS器件的积分求和器实现部分slicer操作

### 5.12 合并Mux与IIR滤波器

**IIR实现**：
- 低通响应（时间常数）由$R_D$和$C_D$实现
- 幅度由$R_D$和$I_D$控制
- 2 UI延迟通过mux实现（在2nd后标开始消除）

## 六、均衡器自适应技术

### 6.1 自适应均衡的必要性

**固定均衡**：针对特定系统优化，依赖实验室测量，对制造和环境变化敏感

**自适应均衡**：允许针对不同信道、环境条件、数据速率优化均衡器

### 6.2 自适应均衡关键问题

1. **提取均衡校正（误差）信号**
2. **自适应算法和硬件开销**
3. **将校正信息传达给均衡器电路**

### 6.3 TX FIR自适应误差提取

**问题**：TX FIR自适应时，需在接收机输入端测量响应

**误差提取方法**：比较接收机输入与期望符号电平$d_{Level}$

**附加要求**：接收机需要额外采样器（带可编程阈值电平）

### 6.4 TX FIR自适应算法：Sign-Sign LMS

**符号-符号LMS算法**（因实现简单而被广泛使用）：

$$w_{n+1}^k = w_n^k + \Delta_w \cdot \text{sign}(d_{n-k}) \cdot \text{sign}(e_n)$$

其中：
- $w_n^k$：抽头系数，时间n，抽头索引k
- $d_n$：接收数据
- $e_n$：相对于期望数据电平的误差

**附加**：由于期望数据电平取决于发射机摆幅和信道损耗，期望电平也需自适应

$$d_{Level,n+1} = d_{Level,n} - \Delta_{d_{Level}} \cdot \text{sign}(e_n)$$

### 6.5 TX FIR回传信道

**问题**：TX FIR抽头更新信息需传回TX，需回传信道

**方案1：低速共模信令**
- 约10 Mb/s共模信令从RX到TX（同差分信道）
- 实现简单

**方案2：高速TX信道编码**
- RX侧利用高速TX信道传数据回TX
- 8B/10B编码灵活性允许低数据率抽头自适应信息传递（无数据率开销）

### 6.6 CTLE调谐：PSD测量法

**方法**：比较随机数据低频和高频频谱内容

**理论**：对于理想随机数据，低频功率与高频功率之间有可预测比例

$$\int_0^{f_m} s_x(f)df = \int_{f_m}^{\infty} s_x(f)df = \frac{1}{4}$$

其中 $f_m \approx \frac{0.28}{T_b}$

**误差**：两种频谱分量之间的误差送入伺服环调谐CTLE

### 6.7 CTLE调谐：输出幅度测量法

**方法**：比较低频和高频平均幅度

**理论**：将均衡数据近似为正弦波，低频和和高频平均幅度之间存在可预测比例

**调谐目标**：调整均衡设置直至高频峰峰值摆幅匹配低频峰峰值

### 6.8 CTLE调谐：数据边沿分布监测

**方法**：利用数据边沿分布的宽度和形状可靠校准均衡器

**实现**：对数据位进行子周期精度过采样

**目标**：最大化眼图张开度，即最小化边沿分布的标准差

### 6.9 DFE调谐：FIR反馈

**方法**：在数据边沿处2x过采样均衡信号，可提取DFE和CDR环路自适应信息

**算法**：Sign-Sign LMS算法自适应DFE抽头值

### 6.10 DFE调谐：IIR反馈

**RC时间常数控制** → 等效抽头数量

**幅度控制** → 等效抽头权重

**自适应规则**（Hidaka ISSCC 2007）：

| 情况 | 条件 | 动作 |
|------|------|------|
| Case 1: D2,D1,D0=110 | $1S_1^{E0} > 0$ | 幅度减小 |
| Case 2: D3,D2,D1,D0=1110 | $1S_1^{E0} > 0$ | 增加时间常数 |
| Case 2: D3,D2,D1,D0=1110 | $1S_1^{E0} < 0$ | 减小时间常数 |

## 七、高级调制与均衡技术

### 7.1 高级调制减少均衡复杂度

**核心思想**：减少频率依赖损耗，将奈奎斯特频率移至更低值

**方法**：
- PAM4（或更高阶PAM）
- 双二进制（duo-binary）

### 7.2 多音调信令 (Multi-tone Signaling)

**思想**：
- 不将信道均衡至基带奈奎斯特频率
- 将信道分成多个频带（每个频带频率依赖损耗更小）
- 每子频带均衡复杂度降低
- 需要上/下变频

**应用**：DSL调制解调器（极挑战信道），低数据率允许高性能DSP

**近期趋势**：在高速链路研究原型中开始出现

## 附录：均衡器传递函数汇总

**TX FIR传递函数**：
$$W(z) = c_{-1}z^{+1} + c_0z^0 + c_1z^{-1}$$

**CTLE传递函数（无源）**：
$$H(s) = \frac{R_2}{R_1+R_2} \cdot \frac{1+R_1C_1s}{1+\frac{R_1R_2}{R_1+R_2}(C_1+C_2)s}$$

**CTLE传递函数（有源）**：
$$H(s) = \frac{g_m}{C_p} \cdot \frac{s + \frac{1}{R_sC_s}}{(s + \frac{1+g_mR_s/2}{R_sC_s})(s + \frac{1}{R_DC_p})}$$

**DFE时域方程**：
$$z_k = y_k - \sum_{i=1}^{n} w_i d_{k-i}$$

**MMSE最优TX抽头**：
$$\mathbf{W}_{ls} = (\mathbf{H}^T\mathbf{H})^{-1}\mathbf{H}^T\mathbf{Y}_{des}$$

**Sign-Sign LMS更新**：
$$w_{n+1}^k = w_n^k + \Delta_w \cdot \text{sign}(d_{n-k}) \cdot \text{sign}(e_n)$$
