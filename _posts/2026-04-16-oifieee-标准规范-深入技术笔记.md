## 1. OIF 与 IEEE 标准体系

### 1.1 OIF 简介

**OIF (Optical Internetworking Forum)** 是光网络行业的主要标准组织，成立于 1998 年，成员包括运营商、设备商、器件商。

**主要工作**:
- 制定光网络 Implementation Agreements (IA)
- 推动多厂商互操作性
- 定义物理层和链路层规范

### 1.2 OIF vs IEEE 802.3 对比

| 方面 | OIF | IEEE 802.3 |
|------|-----|------------|
| 焦点 | 光网络互联 | 以太网标准 |
| 层次 | 物理层/链路层 | 物理层/PHY |
| 灵活性 | 更灵活 | 更正式 |
| 速度 | 快速迭代 | 标准化周期长 |
| 互补 | 提供 CEI 等电气规范 | 定义系统标准 |

### 1.3 OIF 主要技术领域

```
OIF 技术领域:
├── 高速电气接口 (CEI)
├── 光模块规范
├── FlexE/FlexO
├── 互通测试 (Interop)
└── 运营商需求
```

### 1.4 标准关系图

```
IEEE 802.3 (以太网物理层)
     ↓
OIF CEI (电气接口规范)
     ↓
MSA (多源协议 - QSFP-DD/OSFP)
     ↓
光模块厂商实现
```

## 3. CEI 关键电气参数

### 3.1 差分阻抗

```
阻抗要求:
- 差分阻抗: 100 Ω ± 10%
- 单端阻抗: 50 Ω ± 10%
- 回波损耗: > 10 dB @ 目标频率
```

### 3.2 眼图要求

**CEI-112G-VSR 眼图**:
```
眼高: > 50 mV (典型)
眼宽: > 0.3 UI
过渡时间: < 20 ps
抖动: < 0.15 UI RMS
```

### 3.3 CDR 抖动规范

| 参数 | CEI-56G | CEI-112G |
|------|---------|----------|
| 抖动产生 | < 0.15 UI | < 0.12 UI |
| 抖动容限 | > 0.3 UI | > 0.3 UI |
| 抖动带宽 | < 10 MHz | < 10 MHz |

### 3.4 光模块电气接口

**QSFP-DD 电气接口**:
```
引脚定义:
差分对:
- 8 发送对 (TX)
- 8 接收对 (RX)

控制信号:
- SCL/SDA (I2C)
- MODPRSn (存在检测)
- RESETn (复位)
- INTn (中断)
```

**功耗限制**:
| 类型 | 功耗 |
|------|------|
| QSFP-DD 400G | < 12 W |
| QSFP-DD 800G | < 24 W |

## 4. FlexE / FlexO 规范

### 4.1 FlexE 概述

**FlexE (Flexible Ethernet)** 是 OIF 制定的灵活以太网接口规范，实现多厂商以太网设备的互联。

### 4.2 FlexE 架构

```
FlexE 架构:

┌─────────────────────────────────────────────┐
│                   FlexE Shim                 │
├─────────────────────────────────────────────┤
│                                              │
│  40G/100G/200G/400G 以太网                   │
│                                              │
│  ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐         │
│  │ 子通道 │  │ 子通道 │  │ 子通道 │  │ 子通道 │         │
│  │  0   │  │  1   │  │  2   │  │  3   │         │
│  └─────┘  └─────┘  └─────┘  └─────┘         │
│                                              │
└─────────────────────────────────────────────┘
```

### 4.3 FlexE 版本

| 版本 | 说明 | 通道化 |
|------|------|--------|
| FlexE 1.0 | 基础规范 | 支持 |
| FlexE 2.0 | 增强特性 | 支持 |
| FlexE 3.0 | 800G 支持 | 支持 |

### 4.4 FlexE 应用

**速率绑定**:
```
2x 100G → 200G
4x 100G → 400G
8x 100G → 800G
```

**子速率映射**:
- 将多个低速率服务映射到高速率链路
- 增强灵活性

### 4.5 FlexO vs FlexE

| 特性 | FlexE | FlexO |
|------|-------|-------|
| 层级 | MAC 层 | PHY 层 |
| 介质 | 铜缆/光纤 | 光纤 |
| 应用 | 以太网互联 | 光网络互联 |
| 绑定 | 电气绑定 | 光层绑定 |

## 5. CEI-Linear 电气接口

### 5.1 CEI-Linear 概述

**CEI-Linear** 是 OIF 针对 Linear Drive (非重定时) 光模块新定义的电气接口规范。

### 5.2 Linear vs Retimed 对比

| 特性 | CEI-112G (Retimed) | CEI-112G-Linear |
|------|---------------------|------------------|
| 重定时 | 是 | 否 |
| CDR | 模块内含 CDR | 主机侧 CDR |
| 功耗 | 高 (~15pJ/b) | 低 (~5pJ/b) |
| 延迟 | ~100ns | ~0 |
| 眼图 | CDR 恢复后 | 线性放大后 |

### 5.3 CEI-Linear 合规测试点

```
主机(TX)              光模块              主机(RX)
   │                    │                    │
   │    TP1 (电气)     │    TP2 (光)       │
   ▼                    ▼                    ▼
┌──────────────────────────────────────────────┐
│  CEI-Linear TX spec   │   Optical spec    │
└──────────────────────────────────────────────┘
   ▲                    ▲                    ▲
   │    TP4 (电气)     │    TP3 (光)       │
   │                    │                    │
```

| 测试点 | 位置 | 说明 |
|--------|------|------|
| TP1 | 主机 ASIC 输出 | 发射电气接口 |
| TP2 | 模块输入 (光) | 光发射机输入 |
| TP3 | 模块输出 (光) | 光接收机输出 |
| TP4 | 模块输出 (电气) | 接收电气接口 |

### 5.4 CEI-Linear 目标应用

| 应用 | 距离 | 功耗目标 |
|------|------|---------|
| 前面板可插拔 | <500m | <8W/800G |
| CPO/NPO | <2m | <5W/800G |
| Die-to-Die | <50mm | <2W/800G |

## 7. 以太网速率与物理层规范

### 7.1 为什么从 NRZ 转向 PAM4

| 方面 | NRZ @ 50G | PAM4 @ 50G |
|------|-----------|------------|
| 带宽需求 | 25 GHz | 12.5 GHz |
| 编码效率 | 1 bit/sym | 2 bits/sym |
| SNR 要求 | 低 | 高 (~9dB) |
| 眼图 | 1 眼 | 3 眼 |

### 7.2 RS-FEC 配置

**Reed Solomon FEC**:
```
RS(544,514)
- 544 符号块
- 514 数据符号
- 30 纠错符号
- 可纠正: 15 符号错误
```

**性能提升**:
```
无 FEC:  BER ~ 1e-5
有 FEC:  BER ~ 1e-12
编码增益: ~6 dB
```

### 7.3 以太网物理层编码

| 编码 | 效率 | 应用 |
|------|------|------|
| 8b/10b | 80% | 早期 10G |
| 64b/66b | 97% | 10G+ 当前 |
| 128b/132b | ~97% | 800G/1.6T |

### 7.4 关键物理层接口

**QSFP28 (100G)**:
```
- 4x25G
- 功耗 < 3.5W
- 间距: MR/LR/SR
```

**QSFP-DD (400G/800G)**:
```
- 8x50G (400G)
- 8x100G (800G)
- 向后兼容 QSFP28
```

**OSFP (800G)**:
```
- 8x100G
- 更高密度
- 热管理改进
```

## 8. 800G/1.6T/224G 标准进展

### 8.1 802.3dj (800G/1.6T Ethernet)

**目标**:
```
- 800 Gbps (8x100G)
- 1.6 Tbps (16x100G 或 8x200G)
- PAM4 调制
```

**物理层规范**:
| 规范 | 距离 | 介质 |
|------|------|------|
| 800GBASE-SR8 | 100m | MMF |
| 800GBASE-LR8 | 10km | SMF |
| 800GBASE-FR8 | 2km | SMF |
| 1.6TBASE-SR16 | 100m | MMF |
| 1.6TBASE-LR16 | 10km | SMF |

### 8.2 802.3df (224G SerDes)

**关键参数**:
| 参数 | 值 |
|------|-----|
| 单通道 | 224 Gbps |
| 调制 | PAM4 |
| 波特率 | 112 Gbaud |

**应用**:
```
- 1.6T/3.2T Ethernet
- 支持更长距离
- 芯片到芯片互联
```

### 8.3 未来路线图
