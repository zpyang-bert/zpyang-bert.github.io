## 概述

IEEE P802.3ck/D2.1 是 IEEE 802.3 标准的一份草案修正案，于2021年6月17日发布，用于对 IEEE Std 802.3-2018 进行修订。该修正案主要定义了基于 100 Gb/s  signaling 的 100 Gb/s、200 Gb/s 和 400 Gb/s 电气接口的物理层规范和管理参数。

核心新增内容：
- Clause 136：50GBASE-CR, 100GBASE-CR2, 200GBASE-CR4 的 PMD 规范
- Clause 137：（保留用于backplane）
- Clause 162：100GBASE-CR1, 200GBASE-CR2, 400GBASE-CR4 的 PMD 规范（铜缆）
- Clause 163：100GBASE-KR1, 200GBASE-KR2, 400GBASE-KR4 的 PMD 规范（背板）
- Clause 161：RS-FEC-Int（码字交织RS-FEC）子层
- Annex 120F/120G：100GAUI-1/200GAUI-2/400GAUI-4 芯片到芯片/模块接口规范

## 2. Clause 162：100GBASE-CR1/200GBASE-CR2/400GBASE-CR4 PMD

### 2.1 PMD服务接口

Clause 162定义的铜缆PMD使用PAM4调制，信道速率53.125 GBd，每个通道承载100Gb/s。

**支持的PHY类型关联：**

| PMD类型 | 关联Clause |
|---------|-----------|
| 100GBASE-CR1 | Clause 82 (PCS), Clause 91 (RS-FEC), Clause 135 (PMA) |
| 200GBASE-CR2 | Clause 119 (PCS 200/400G), Clause 120 (PMA), 可选RS-FEC |
| 400GBASE-CR4 | Clause 119 (PCS 400G), Clause 120 (PMA), 可选RS-FEC |

### 2.2 PMD功能块

```
发射器功能：
- 通过PMD:IS_UNITDATA_i.request原语接收数据
- 信令速率：53.125 GBd（标称值）
- PAM4调制

接收器功能：
- 通过PMD:IS_UNITDATA_i.indication原语传递数据
- 信令速率：53.125 GBd（标称值）

可选功能：
- 全局/逐通道信号检测
- 全局/逐通道发射禁止(Tx disable)
- 发射/接收故障检测
```

### 2.3 延迟约束

Clause 162定义了严格的延迟约束，确保系统时序性能。

## 4. Clause 161：RS-FEC-Int（码字交织RS-FEC）子层

### 4.1 概述

RS-FEC-Int是802.3ck为100GBASE-P PHY引入的新型前向纠错编码，采用码字交织技术提升纠错性能。

**位置：** 位于100GBASE-P子层中，在PCS和PMA之间

**主要功能：**
- 64B/66B到256B/257B转码
- 对齐标记映射和插入
- Pre-FEC分布
- Reed-Solomon编码
- 符号分布
- 传输位排序

### 4.2 发送功能

```
功能块图：
1. 通道块同步（Lane block synchronization）
2. 对齐锁定和解偏斜（Alignment lock and deskew）
3. 通道重排序（Lane reorder）
4. 对齐标记移除（Alignment marker removal）
5. 64B/66B到256B/257B转码器
6. 对齐标记映射和插入
7. Pre-FEC分布
8. Reed-Solomon编码器
9. 符号分布
10. 传输位排序
```

### 4.3 接收功能

```
功能块图：
1. 对齐锁定和解偏斜
2. 通道重排序和去交织
3. Reed-Solomon解码器
   - 旁路错误指示（可选）
   - FEC降级SER（可选）
4. Post FEC交织
5. 对齐标记移除
6. 256B/257B到64B/66B转码器
7. 块分布
8. 对齐标记映射和插入
```

### 4.4 MDIO/MANagement接口

定义了丰富的MDIO寄存器用于RS-FEC-Int控制和管理：

| 寄存器功能 | 说明 |
|-----------|------|
| fec_align_status | FEC对齐状态 |
| FEC_corrected_cw_counter | 纠错码字计数器 |
| FEC_uncorrected_cw_counter | 未纠错码字计数器 |
| FEC_lane_mapping | 通道映射 |
| fec_align_status | 对齐状态 |
| block_lock | 块锁定状态 |
| am_lock | 对齐标记锁定 |
| FEC_degraded_SER_ability | 降级SER能力 |
| FEC_degraded_SER | 降级SER状态 |
| FEC_cw_counter | 码字计数器 |

## 6. 电气接口特性对比

### 6.1 CR（铜缆）vs KR（背板）

| 特性 | 100GBASE-CR1/KR1 | 200GBASE-CR2/KR2 | 400GBASE-CR4/KR4 |
|------|-------------------|------------------|-------------------|
| 通道数 | 1 | 2 | 4 |
| 单通道速率 | 100 Gb/s | 100 Gb/s | 100 Gb/s |
| 总速率 | 100 Gb/s | 200 Gb/s | 400 Gb/s |
| 调制 | PAM4 | PAM4 | PAM4 |
| 符号率 | 53.125 GBd | 53.125 GBd | 53.125 GBd |
| 典型传输距离 | 短距离铜缆 | 短距离铜缆 | 短距离铜缆/背板 |

### 6.2 BER要求

**100GBASE-CR1/KR1：**
- 目标帧丢失率：< 6.2×10⁻¹⁰ (64字节帧，最小IPG)
- 所需输入BER：< 2.4×10⁻⁴（经FEC处理后）
- FEC：Clause 91 RS-FEC 或 Clause 161 RS-FEC-Int

**200GBASE-CR2/KR2 和 400GBASE-CR4/KR4：**
- 目标帧丢失率：< 6.2×10⁻¹¹ (64字节帧，最小IPG)
- 所需输入BER：< 2.4×10⁻⁴
- PCS处理后帧丢失率：< 9.2×10⁻¹³

## 8. 调制与编码

### 8.1 PAM4调制

802.3ck采用PAM4（四电平脉冲幅度调制）实现100Gb/s/lane：

| 特性 | 值 |
|------|-----|
| 调制阶数 | 4电平 |
| 符号率 | 53.125 GBd |
| 每符号比特数 | 2 |
| 频谱效率 | 2 bits/symbol |

**PAM4编码：**
- Gray映射用于PAM4编码通道
- 可选的1/(1+D)预编码

### 8.2 前向纠错（FEC）

**RS(544,514)编码：**
- 适用于100GBASE-P PHY
- Clause 91：传统RS-FEC
- Clause 161：RS-FEC-Int（码字交织版本）

**关键特性：**
- 纠错能力：15符号错误/码字
- 可选的降级SER检测和报告
- 丰富的MDIO管理接口

## 11. 关键技术创新总结

### 11.1 100Gb/s/lane PAM4电气接口

802.3ck首次在电气接口层面实现100Gb/s/lane，核心创新：

1. **53.125 GBd PAM4调制**：每通道实现100Gb/s
2. **多通道聚合**：支持1/2/4通道配置，覆盖100G/200G/400G应用
3. **灵活的AUI接口**：支持芯片到芯片和芯片到模块多种变体

### 11.2 RS-FEC-Int（码字交织RS-FEC）

为100GBASE-P PHY引入的新型FEC方案：

1. **256B/257B转码**：提高编码效率
2. **码字交织**：增强突发错误纠错能力
3. **可选的降级SER监控**：支持链路质量预警

### 11.3 综合测试规范

- 完整的发送器/接收器电气特性测试
- 通道运行裕量(COM)计算方法学
- 压力眼图测试（stress eye）
- 抖动容限测试
