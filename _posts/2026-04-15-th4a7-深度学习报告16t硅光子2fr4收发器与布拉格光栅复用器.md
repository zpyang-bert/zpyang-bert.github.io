## 二、2×FR4架构

### 2.1 FR4标准

**CWDM4波长分配**：
```
2×FR4 = 2 × 4波长

第一组 (Lane 0-3):
- CH1: 1271 nm
- CH2: 1291 nm
- CH3: 1311 nm
- CH4: 1331 nm

第二组 (Lane 4-7):
- CH5: 1271 nm
- CH6: 1291 nm
- CH7: 1311 nm
- CH8: 1331 nm

通道间隔: 20 nm
```

**为什么FR4？**
```
优势：
- 使用SMF (G.652)
- 适合数据中心
- 成本低于LWDM
- 波长间隔宽 → 容错性好

挑战：
- 带宽受限 (O-band)

```

## 三、布拉格光栅原理

### 3.1 布拉格反射原理

**光栅方程**：
```
布拉格条件：
m·λ_B = 2·n_eff·Λ

其中：
- m = 衍射级数 (通常 m=1)
- λ_B = 布拉格波长
- n_eff = 有效折射率
- Λ = 光栅周期

设计参数：
- n_eff ≈ 2.15 (Si waveguide)
- λ_B = 1291 nm
- Λ ≈ 300 nm
```

**反射谱特性**：
```
反射带宽 Δλ:

Δλ = λ_B × (Δn_eff / n_eff) × (N / N_eff)

其中：
- Δn_eff = 折射率调制幅度
- N = 光栅周期数
- N_eff = 有效折射率加权

典型设计：
- 反射带宽: 15-20 nm
- 旁瓣抑制: >15 dB
- 纹波: <0.5 dB
```

### 3.2 均匀布拉格光栅

**折射率调制**：
```
n(z) = n₀ + Δn · cos(2πz/Λ)

其中：
- n₀ = 平均折射率
- Δn = 折射率调制幅度 (~10⁻⁴ - 10⁻³)
- Λ = 光栅周期

耦合系数 κ:
κ = π · Δn / λ_B

典型值：
κ ≈ 50-200 cm⁻¹
```

### 3.3 Apodized (渐变) 光栅

**为什么要Apodization？**
```
均匀光栅问题：
- 高旁瓣
- 纹波大
- 频谱不平坦

Apodization方案：
n(z) = n₀ + Δn(z) · cos(2πz/Λ)

Δn(z) 分布：
- 汉明窗: Δn(z)·sin(πz/L)
- 余弦平方: Δn(z)·cos²(πz/L)
- 高斯: Δn(z)·exp(-(z/σ)²)

效果：
- 旁瓣抑制 >20 dB
- 平顶带宽最大化
- 纹波 <0.2 dB
```

## 十一、深度技术问题

### 11.1 光栅制造容差

**工艺波动影响**：
```
光栅周期误差:
ΔΛ/Λ = Δλ/λ

λ = 1300nm, Δλ = 1nm:
ΔΛ/Λ ≈ 0.08%

挑战:
- 纳米级控制
- 工艺均匀性
- CD偏差补偿
```

### 11.2 级联串扰

**通道间串扰机制**：
```
串扰来源:
1. MUX泄漏
   - 旁瓣泄露
   - -25dB (Bragg)

2. 端面反射
   - Fresnel反射
   - 抗反射涂层

3. 偏振相关
   - TE/TM差异
   - 偏振分集解决
```

### 11.3 眼图闭合

**TDECQ计算**：
```
TDECQ = 10·log₁₀(σᵣₑc²/σ₀²)

其中:
- σᵣₑc = 接收机等效噪声
- σ₀ = 理想眼图噪声

眼图恶化因素:
- 带宽限制
- 码间干扰 (ISI)
- 非线性失真
- 噪声
```

> 🔍 深度说明：本图展示布拉格光栅MUX对应4个波长通道的特征光谱示意图，属于原理性示意而非实测数据。图中有4个离散的特征峰（λ1~λ4），分别对应4个通道的谐振反射波长，峰以外区域透射率接近100%（对应基线水平）。这种离散线状光谱是布拉格光栅的典型特征——与AWG的连续通带不同，布拉格光栅的反射峰非常锐利（高Q值），适合窄间隔波长选择。工程背景：SiPh平台上制作布拉格光栅通常用电子束曝光（EBL）或深紫外曝光（DUV）刻蚀周期性波纹结构，周期精度需控制在±0.5nm以内才能保证中心波长准确；相比之下AWG的波长精度由波导几何尺寸决定，对工艺波动较不敏感。工程价值：对于CWDM系统（通道间隔5nm），布拉格光栅的高Q值特性意味着通道间串扰可以做得极低（<-30dB），为未来的4×400G甚至4×800G升级预留充足余量；而传统AWG的串扰通常在-25dB量级。落地注意：布拉格光栅的温度敏感性（dn/dT ~1.8×10⁻⁴/K）会导致波长漂移约0.01nm/K，对于户外光模块需要温控或温度补偿设计；SiPh的温度相关损耗（TCL）也需要在链路预算中留足余量。

> 🔍 深度说明：本图展示8个CWDM通道在70°C高温环境下的误码率（BER）随光调制幅度（OMA）的变化曲线（Fig.3）。横轴OMA范围-7dBm到3dBm，纵轴BER从1E-2到1E-13（对数刻度），8条曲线对应8个独立通道（CH1-CH8）。两条红色参考线：垂直线OMA=-5.5dBm为典型接收灵敏度参考点，水平线BER=1E-4为IEEE 802.3标准的前向纠错（FEC）可纠正误码率门限。所有通道在OMA>-5dBm时可实现BER<1E-4（满足FEC门限），随OMA增大BER持续下降；70°C高温下8通道一致性良好，曲线基本重合，说明器件封装设计对温度的敏感性控制到位。研究背景：800G 2×FR4光模块的高温（70°C）BER测试是IEEE 802.3cu和MSA规范要求的必测项目，用于验证光模块在数据中心热环境下的工作可靠性；BER vs OMA曲线是光模块灵敏度特性的直接表征，是制定光链路功率预算的核心依据。工程价值：光模块的光功率预算（link power budget）=发射光功率-接收灵敏度，其中接收灵敏度=OMA_thr+接收器损耗；-5.5dBm@BER=1E-4的灵敏度要求，意味着在总链路损耗10dB的场景下，发射光功率需要>-5.5+10=+4.5dBm才能正常工作，这对DFB激光器的输出功率提出了明确要求（通常需要>6dBm）。落地注意：FEC门限的选择需平衡编码增益和延迟开销——KP4 FEC（纠错能力~1E-15）的净编码增益约9.5dB，但编码延迟约100ns；CR5 FEC的延迟更低（约10ns），但编码增益只有约7dB，在TDECQ>2.5dB的高性能光模块中需要仔细权衡。
