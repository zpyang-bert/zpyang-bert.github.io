## 二、时域反射计(TDR)

### 2.1 TDR系统架构

**TDR原理框图**：

![](lecture3_ee720_tdr_spar_images/img-001.ppm)

> 🔍 深度说明：
> 【研究背景】TDR（时域反射计）是高速信道表征的基本仪器，通过发射快沿阶跃并观测反射来定位阻抗不连续。该架构图展示了标准TDR系统的连接方式。
> 【核心结论】TDR的上升沿时间（约35ps）决定了空间分辨率（约5mm）；通过反射波的幅度和时间延迟，可以计算出不连续点的阻抗值和位置。
> 【工程价值】TDR是信道故障诊断的首选工具，可以快速识别过孔、连接器、背板等位置的问题；与VNA互补，TDR提供时域直观性，VNA提供频域精度。
> 【落地注意】TDR校准需要使用标准开路/短路/负载件；探头的接地电感会影响测量精度，在高频率时需使用探头deskew技术。

### 2.2 TDR阻抗计算

**反射系数定义**：
```
ρ_r(t) = V_r(t)/V_i = [Z_T(t) - Z_0]/[Z_T(t) + Z_0]

时变阻抗提取:
Z_T(t) = Z_0 · (1 + ρ_r(t))/(1 - ρ_r(t))

若入射电压 V_i = 1V (阶跃):
→ V_r(t) = ρ_r(t)
→ V_t(t) = V_i + V_r = 1 + ρ_r(t)

因此:
Z_T(t) = Z_0 · V_t(t)/(2 - V_t(t))

空间位置:
x = v_p · t/2    (往返时间 → 单程距离)
v_p = 1/√(LC)   (无损情况)
```

### 2.3 TDR波形特征

**典型终端条件波形**：

![](lecture3_ee720_tdr_spar_images/img-002.jpg)

> 🔍 深度说明：
> 【研究背景】该图展示了开路、短路、匹配三种终端条件下TDR的典型响应波形。通过观测反射波形可以快速判断终端匹配状态。
> 【核心结论】开路时ρ=+1，电压稳定在2V；短路时ρ=-1，电压稳定在0V；匹配时ρ=0，电压稳定在0.5V且无反射。
> 【工程价值】TDR波形是信道故障诊断的第一手资料——电压高于0.5V表示阻抗偏高，低于0.5V表示阻抗偏低。
> 【落地注意】TDR显示的电压是入射波和反射波的叠加，需要理解反射系数的物理含义才能正确解读波形。

**不连续性波形特征**：
```
┌─────────────────────────────────────────────┐
│ 并联电容不连续 (Shunt C):                    │
│ • 阻抗骤降 → 电压下冲                        │
│ • 然后指数恢复                               │
│ • 时间常数 τ = C·Z₀/2                       │
├─────────────────────────────────────────────┤
│ 串联电感不连续 (Series L):                   │
│ • 阻抗骤增 → 电压上冲                        │
│ • 然后指数恢复                               │
│ • 时间常数 τ = L/(2Z₀)                      │
├─────────────────────────────────────────────┇
│ 幅值:                                       │
│ ΔV/V = (τ/t_r) · 指数项最大值                │
│ 其中 t_r 为TDR上升时间                       │
└─────────────────────────────────────────────┘
```

### 2.4 TDR分辨率与限制

```
空间分辨率:
Δx > t_r · v_p

其中 t_r 为TDR阶跃上升时间

限制因素:
1. 信道传播对上升沿的滤波
   • 趋肤效应导致高频衰减
   • 色散展宽脉冲

2. 集总不连续性的低通效应
   • stub等效于谐振腔
   • 限制了L/C值估算精度

有趣发现:
• 信道滤波有时会补偿集总不连续性的尖峰
• 这被称为"Channel filtering compensation"
```

```
TDR可分析多重反射:

时序分析:
┌──────────────────────────────────────────────────────┐
│ 不连续点 A → B → C → Load                           │
│                                                       │
│ 反射路径序列:                                          │
│ • A处: 首次反射                                      │
│ • BAB: A→B→A 的二次反射                             │
│ • BCB: B→C→B                                         │

│ • CAB: C→A→B→C                                      │
│ • CBCBC: 多重往返                                     │
└──────────────────────────────────────────────────────┘

通过格型图追踪:
• 每次反射的时间
• 每次反射的幅度(由各点反射系数决定)
• 最终收敛值(终端阻抗)
```

### 2.6 时域传输(TDT)

```
TDT测量:
• 观测传输到负载另一侧的信号
• 与TDR互补(TDR看反射，TDT看过孔)

传递函数测量:
H(jω) = V_out(jω)/V_in(jω)

问题:
• 难以分离阻抗不连续的影响
• 反射叠加在上升沿上
• 难以区分各个不连续点

TDR+TDT联合使用:
• TDR: 定位阻抗不连续
• TDT: 测量端到端传递函数
• 联合分析得到完整信道特性
```

## 三、网络分析仪与S参数

### 3.1 频域测量原理

**网络分析仪架构**：

![](lecture3_ee720_tdr_spar_images/img-019.ppm)

> 🔍 深度说明：
> 【研究背景】网络分析仪（VNA）是高速信道表征的核心设备，通过扫频正弦波测量S参数。该架构图展示了典型VNA的硬件连接和信号流。
> 【核心结论】VNA通过定向耦合器分离前向波和反射波，分别测量S₁₁（回波损耗）和S₂₁（插入损耗）；测量精度取决于校准方法（SOLT/TRL）和参考面设定。
> 【工程价值】VNA是信道S参数测量的标准工具，可以表征从DC到50GHz的宽频带响应；级联多个组件的S参数需要转换为ABCD参数再进行矩阵乘法。
> 【落地注意】VNA校准是测量精度的关键——TRS（Thru-Reflect-Line）校准适合高损耗信道，SOLT（Short-Open-Load-Thru）适合通用场景；探头和电缆的相位稳定性也需关注。

### 3.2 S参数定义

**为什么用S参数？**：
```
相比Y、Z参数的优势:
• Y、Z参数需要开路/短路条件 ← 实际难实现
• S参数在50Ω终端下测量 ← 真实系统条件
• 高频下开路短路难以实现良好的参考平面

核心概念:
• a: 入射波(归一化到√Z₀)
• b: 反射波(归一化到√Z₀)

归一化功率波:
a = V⁺/√Z₀
b = V⁻/√Z₀
功率 = |a|² = |b|²
```

**二端口S参数**：
```
S参数矩阵:
[b₁]   [S₁₁  S₁₂] [a₁]
[b₂] = [S₂₁  S₂₂]·[a₂]

物理含义:
• S₁₁ = b₁/a₁ |a₂=0  — 输入端反射系数(端口2匹配)
• S₂₁ = b₂/a₁ |a₂=0  — 正向传输系数(端口2匹配)
• S₁₂ = b₁/a₂ |a₁=0  — 反向传输系数(端口1匹配)
• S₂₂ = b₂/a₂ |a₁=0  — 输出端反射系数(端口1匹配)

幅度解释:
|S₁₁|² = 反射功率/入射功率 (输入回波损耗)
|S₂₁|² = 传输功率/入射功率 (插入损耗)
```

### 3.3 S参数级联理论

**关键问题**：
```
S参数不能直接级联！
原因: S参数定义基于各端口的入射/反射波
级联时内部端口边界条件不匹配

解决方案:
必须转换为ABCD参数(或T参数)进行级联
然后再转换回S参数
```

**ABCD参数定义**：
```
ABCD参数(传输矩阵):
[v₁]   [A  B] [v₂]
[i₁] = [C  D]·[i₂]

其中定义:
A = v₁/v₂ | i₂=0  (电压比，开路)
B = v₁/i₂ | v₂=0  (转移阻抗，短路)
C = i₁/v₂ | i₂=0  (转移导纳，开路)
D = i₁/i₂ | v₂=0  (电流比，短路)

级联特性:
[v₁]     [A₁  B₁] [A₂  B₂] [v₂]
[i₁]_orig = [C₁  D₁]·[C₂  D₂]·[i₂]_new
              = [A₁A₂+B₁C₂  A₁B₂+B₁D₂] [v₂]
                [C₁A₂+D₁C₂  C₁B₂+D₁D₂] [i₂]_new
```

**S ↔ ABCD转换表**：

![](lecture3_ee720_tdr_spar_images/img-032.ppm)

> 🔍 深度说明：
> 【研究背景】S参数和ABCD参数的相互转换是信道级联分析的基础。该转换公式基于归一化功率波定义，需要参考阻抗Z₀进行计算。
> 【核心结论】ABCD参数可以进行矩阵级联（[A₁ B₁;C₁ D₁] × [A₂ B₂;C₂ D₂]），这是计算多段信道总响应的标准方法；S参数不能直接级联。
> 【工程价值】在实际信道建模中，需要将芯片封装、PCB走线、连接器、背板等各部分的实测S参数转换为ABCD参数后级联，再转换回S参数进行仿真。
> 【落地注意】转换过程中的数值精度需要注意，特别是当S₂₁接近0时（高损耗信道）；建议使用双精度浮点数进行计算。

### 3.4 级联示例

**Via + 传输线案例**：
```
应用场景:
芯片封装 → 过孔 → PCB走线 → 连接器 → 背板

逐级级联:
[整体信道] = [Via模型] × [传输线模型] × [连接器模型]

实际测量:
• 各组件单独测量S参数
• 转换为ABCD参数
• 矩阵乘法完成级联
• 再转换回S参数
```

### 3.5 差分S参数

**四端口差分模型**：
```
单端口测量 → 差分表示

差分激励:
• 奇模激励: V₁ = -V₂ (差分驱动)

## 四、信道脉冲响应

### 4.1 S参数→脉冲响应

**理论基础**：
```
系统响应: Y(ω) = H(ω)·X(ω)

时域卷积: y(t) = h(t) ⊛ x(t)

脉冲响应: h(t) = F⁻¹{H(ω)}
        = F⁻¹{S₂₁(ω)}  (传输通道)

逆傅里叶变换步骤:
1. S参数通常只有正频率数据
2. 需要构造负频率: S(-f) = S*(f)
3. 拼接正负频率数据
4. 执行ifft
5. 得到时域脉冲响应
```

### 4.2 频率分辨率提升

**问题**：直接ifft的分辨率受限
```
最高频率 f_max 决定脉冲响应时间分辨率:
Δt = 1/(2·f_max)

如果 f_max = 15GHz
→ Δt ≈ 33.3ps

这对高速链路分析可能不够精细
```

**解决：零填充(Zero Padding)**：
```
方法:
• 将频率数据补零扩展到更高频率
• 例如: 扩展到 ±500GHz

1ps分辨率目标:
Δt = 1ps = 1/(2·f_max)
→ f_max = 500GHz

实际操作:
• 原始数据: DC to 15GHz
• 零填充: 15GHz to 500GHz
• IFFT获得更高时间分辨率
```

### 4.3 脉冲响应验证

** sanity check**：
```
验证方法:
1. 对脉冲响应做FFT
2. 比较FFT结果与原始S参数
3. 两者应高度吻合
4. 任何差异表明建模/计算错误

典型信道脉冲响应特征:
• 主峰(cursor): 对应符号时刻
• 前标(pre-cursor): 主峰之前的响应(前标ISI)
• 后标(post-cursor): 主峰之后的响应(后标ISI)
• 衰减包络: 反映信道损耗
```

### 4.4 不同信道脉冲响应对比

```
三种典型信道对比:

## 五、眼图与码间干扰

### 5.1 眼图形成

```
眼图原理:
• 重叠绘制所有可能的比特组合
• 观察眼图"张开度"
• 表征系统性能

眼图指标:
┌─────────────────────────────────────────────────┐
│ 眼高 (Eye Height):                               │
│ • 垂直张开度                                     │
│ • 表征SNR/电压裕量                               │
│ • 眼高越小 → 噪声余量越小                         │
├─────────────────────────────────────────────────┤
│ 眼宽 (Eye Width):                                 │
│ • 水平张开度                                     │
│ • 表征抖动/时序裕量                              │
│ • 眼宽越小 → 时序余量越小                         │
├─────────────────────────────────────────────────┤
│ 眼图模板 (Eye Mask):                              │
│ • 定义合法区域                                   │
│ • 任何信号不得进入模板区域                         │
│ • 保证互操作性和兼容性                            │
└─────────────────────────────────────────────────┘
```

### 5.2 ISI成因分析

**码间干扰(Inter-Symbol Interference)**：
```
ISI来源:
1. 反射: 阻抗不连续导致多次反射
2. 信道谐振: stub等形成谐振腔
3. 信道损耗(色散): 不同频率分量衰减不同

信道脉冲响应视角:
• 每个比特对后续比特造成干扰
• 主峰周围的pre-cursor和post-cursor贡献ISI
• 长尾响应积累造成显著干扰

信道输入: 眼图开阔
信道输出: 眼图闭合(ISI导致)
```

### 5.3 眼图案例分析(IBM 10Gb/s SerDes)

**测试配置**：
```
IBM 90nm CMOS SerDes @ 10Gb/s

信道组成:
• 16英寸背板走线
• 线卡走线
• 边缘连接器
• 过孔stub

眼图演变:
┌─────────────────────────────────────────────────────┐
│ 位置          │ 眼高        │ 观察                  │
├─────────────────────────────────────────────────────┤
│ TX输出        │ ~600mV      │ 眼图开阔              │
│ 经过封装       │ 轻微恶化     │ 封装走线损耗          │
│ 经过线卡走线   │ 进一步恶化   │ 板级损耗累积          │
│ 经过连接器     │ 明显恶化     │ 连接器不连续          │
│ 经过背板       │ 眼图闭合     │ 主要损伤源            │

└─────────────────────────────────────────────────────┘

眼图随距离恶化过程:
TX输出(开阔) → 550mV眼高 → 接收端(ISI闭合)
```

## 六、瞬态仿真与信道建模

### 6.1 传递函数建模

```
信道传递函数测量:
H(ω) = V_out(ω)/V_in(ω)

在频域:
V_out(ω) = H(ω) · V_in(ω)

时域卷积:
v_out(t) = ∫ h(τ)·v_in(t-τ)dτ

其中 h(t) = F⁻¹{H(ω)}
```

### 6.2 冲激响应生成流程

```
完整流程:
┌──────────────────────────────────────────────────────┐
│ 1. S参数测量 (VNA)                                   │
│    • 频域数据: S₁₁(f), S₂₁(f)                       │
│    • 频率范围: 50MHz - 15GHz                        │
├──────────────────────────────────────────────────────┤
│ 2. 构造正负频率数据                                   │
│    • S(-f) = S*(f)                                 │
│    • 保证实数时域信号                                 │
├──────────────────────────────────────────────────────┤
│ 3. 零填充 (提升分辨率)                                │
│    • 填充至 ±500GHz                                 │
│    • 获得 ~1ps 时间分辨率                            │
├──────────────────────────────────────────────────────┤
│ 4. 逆傅里叶变换                                       │
│    • h(t) = IFFT{S₂₁(f)}                           │
├──────────────────────────────────────────────────────┤
│ 5. 验证                                              │
│    • FFT{h(t)} vs 原始S₂₁(f)                        │
│    • 高度吻合 → 模型正确                              │
└──────────────────────────────────────────────────────┘
```
