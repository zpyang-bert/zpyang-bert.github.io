## 三、高速电气链路系统

### 3.1 系统架构全貌

**电气背板链路**：
```
发送端IC → 封装 → 板级传输线 → 连接器 → 背板 → 连接器 → 板级传输线 → 封装 → 接收端IC
    Tx          Pkg        Trace      Conn    Backplane    Conn     Trace    Pkg          Rx
```

**核心损伤因素**：
```
1. 频率选择性损耗
   • 趋肤效应
   • 介质损耗

2. 反射
   • 阻抗不连续
   • 多重反射

3. 串扰
   • 远端串扰 (FEXT)
   • 近端串扰 (NEXT)
```

### 3.2 IBM 10Gb/s SerDes案例分析

**芯片规格** (ISSCC 2006):
|| 参数 | 数值 |
||------|------|
|| 数据率 | 10 Gb/s |
|| 工艺 | 90nm CMOS |
|| 发射机 | 4-tap FFE |
|| 接收机 | 5-tap DFE + 自适应 |
|| 时钟 | LC-VCO PLL |
|| 功耗 | ~70mW |

![](lecture1_ee720_intro_images/img-005.jpg)

> 🔍 深度说明：
> 【研究背景】IBM该SerDes发表于ISSCC 2006，是90nm工艺节点的高速链路代表作。该设计首次系统性地展示了FFE+DFE联合均衡在实际芯片中的实现，对后续SerDes架构影响深远。
> 【核心结论】4-tap FFE（1前标+1主标+2后标）和5-tap DFE的组合，在10Gb/s下实现了~500mV眼高，功耗仅70mW，是当时最优能效比。
> 【工程价值】半速率CML发射机设计降低了时钟频率要求，4-tap FIR结构可通过数字控制实现自适应预加重，对SerDes transmitter设计有直接参考意义。
> 【落地注意】实际商用SerDes需支持多种信道条件，tap权重需要可编程；90nm工艺的ESD防护设计需特别关注，防止Latch-up影响可靠性。

**发射机架构**：
```
关键特性:
• 半速率CML设计
• 4-tap FFE (1前标 + 1主标 + 2后标)
• Tap极性控制
• ESD保护

FFE抽头配置:
┌──────────┬──────────┐
│ Pre-cursor   │ 25%      │ 4 bits   │
│ Cursor       │ 100%     │ 6 bits   │
│ 1st Post    │ 50%      │ 5 bits   │
│ 2nd Post    │ 25%      │ 4 bits   │
└──────────┴──────────┘
```

### 3.3 眼图分析

**有无FFE的眼图对比**：

![](lecture1_ee720_intro_images/img-003.jpg)

> 🔍 深度说明：
> 【研究背景】该眼图对比是ECEN720课程的核心教学材料，展示了FFE均衡对高速链路的决定性影响。左侧无FFE眼图呈现严重的 ISI（码间干扰），右侧4-tap FFE后眼图明显张开。
> 【核心结论】无FFE时眼高仅~400mV且眼宽受压缩；启用4-tap FFE后眼高提升至~500mV，眼宽改善约20%，系统BER从10^-6级提升至10^-12以下。
> 【工程价值】在10Gb/s以上速率，FFE几乎是必备的TX均衡手段；通过pre-cursor可以补偿前标ISI，post-cursor补偿后标ISI，cursor维持主信号幅度。
> 【落地注意】FFE tap权重设计需考虑信道特性——不同信道长度和损耗曲线需要不同的预加重策略；建议预留可编程tap以适应多种应用场景。

```
无FFE @ 10Gb/s:
• 眼高: ~400mV
• 眼宽: 受限
• 码间干扰(ISI)严重

有FFE (4-tap) @ 10Gb/s:
• 眼高: ~500mV
• 眼宽: 改善
• 预加重补偿信道损耗
```

## 四、信道损伤深度分析

### 4.1 频率依赖损耗

**损耗来源**：
```
趋肤效应:
• 高频电流集中在导体表面
• 等效串联电阻随√f增加
• 损耗 ∝ √f

介质损耗:
• 电介质极化滞后
• 损耗 ∝ f
• 在高频更显著

总损耗:
IL(f) = IL_conductor(f) + IL_dielectric(f)
```

### 4.2 反射机理

**阻抗不连续点**：
```
• 连接器接头
• 板级过孔
• 线宽变化
• 层叠变化

反射系数:
ρ = (Z_L - Z_0) / (Z_L + Z_0)

多次反射:
• 导致符号间干扰(ISI)
• 恶化眼图质量
```

### 4.3 串扰类型

**FEXT vs NEXT**：
```
远端串扰 (FEXT):
• 干扰信号从相邻通道远端耦合
• 与发射方向相同
• 通常小于NEXT

近端串扰 (NEXT):
• 干扰信号从相邻通道近端耦合
• 与发射方向相反
• 能量更大
```

## 五、均衡技术概述

### 5.1 均衡必要性

**信道频率响应不平坦**：
```
输入信号频谱:
┌─────────────────────┐
│                      │ 平坦
└─────────────────────┘

经过信道后:
┌─────────────────────┐
│  高频衰减            │
└─────────────────────┘
         ↓
均衡器补偿
```

### 5.2 发射机均衡 (TX EQ)

**前馈均衡器(FFE)**：

![](lecture1_ee720_intro_images/img-000.jpg)

> 🔍 深度说明：
> 【研究背景】该FFE结构图展示了经典的FIR（有限冲激响应）滤波器结构在SerDes发射机中的应用。早期高速链路使用无源模拟抽头，现在的SerDes通常采用电流模式DAC实现可编程权重。
> 【核心结论】FFE通过预加重高频分量补偿信道衰减，tap数量（4-8tap）和权重精度决定了均衡能力；pre-cursor通常25-50%权重，post-cursor可达50-100%。
> 【工程价值】TX FFE是开环均衡，优点是不放大噪声且对时钟要求低；设计时需平衡功耗（更多tap意味着更大的driver）和均衡效果。
> 【落地注意】实际设计中tap权重需考虑温度和工艺漂移；建议使用10-bit精度DAC配合数字校准，以实现<2%的权重误差。

```
原理:
• 在发射端预加重高频分量
• 补偿信道高频衰减
• 不增加接收机复杂度

抽头结构:
y[n] = Σ w_k · x[n-k]
  = w_0·x[n] + w_1·x[n-1] + w_2·x[n-2] + ...

典型配置:
• Pre-cursor: 补偿前标 ISI
• Cursor: 主信号
• Post-cursor: 补偿后标 ISI
```

### 5.3 接收机均衡 (RX EQ)

**连续时间线性均衡 (CTLE)**：
```
• 可变增益
• 高频提升
• 直流增益控制
```

**判决反馈均衡 (DFE)**：
```
原理:
• 消除已判定符号引起的ISI
• 非线性处理
• 无噪声放大

结构:
┌─────────────────────────────────┐
│   前向滤波 → 判决 → 反馈滤波   │
└─────────────────────────────────┘
```

## 六、时钟系统

### 6.1 PLL架构

**核心模块**：

![](lecture1_ee720_intro_images/img-001.jpg)

> 🔍 深度说明：
> 【研究背景】该PLL架构图是高速SerDes时钟恢复系统的经典实现方案。LC-VCO的选择在噪声和频率调节范围之间做了权衡，是高端SerDes的常用配置。
> 【核心结论】LC-VCO相位噪声优于Ring-VCO约15dB，但调谐范围窄（<10%），需要多级VCO覆盖宽频段；PFD采用鉴频鉴相器消除死区，CP提供充放电电流。
> 【工程价值】PLL环路带宽设计需匹配CDR要求——带宽太宽会传递VCO噪声，太窄则跟不上参考时钟抖动；建议采用type-II PLL实现零静态相位误差。
> 【落地注意】实际商用PLL需支持SSC（扩频时钟）以满足EMI要求；LC电感受工艺偏差影响，Q值偏差±20%会导致相位噪声恶化。

```
鉴相器 (PFD) → 电荷泵 (CP) → 环路滤波器 (LF) → VCO → 分频器
    ↑                                              │
    └────────────────────────────────────────────┘
```

### 6.2 LC-VCO vs Ring-VCO

| 特性 | LC-VCO | Ring-VCO |
|------|---------|----------|
| 相位噪声 | 优 | 较差 |
| 调谐范围 | 窄 | 宽 |
| 面积 | 大(电感) | 小 |
| 功耗 | 中等 | 低 |

## 七、系统性能指标

### 7.1 误码率(BER)

**定义**：
```
BER = 错误比特数 / 总传输比特数

典型要求:
• 有线通信: < 10^-12
• 数据中心: < 10^-15
• 存储: < 10^-16
```

### 7.2 眼图指标

```
眼高 (Eye Height):
• 垂直张开度
• SNR指标

眼宽 (Eye Width):
• 水平张开度
• 抖动指标

眼图模板:
• 定义合法区域
• 保证兼容性
```
