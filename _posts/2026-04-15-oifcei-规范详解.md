## 1. OIF 概述

### 1.1 OIF 简介

**OIF (Optical Internetworking Forum)** 是光网络行业的主要标准组织之一，成立于 1998 年，成员包括运营商、设备商、器件商。

**主要工作**:
- 制定光网络 Implementation Agreements (IA)
- 推动多厂商互操作性
- 定义物理层和链路层规范

### 1.2 OIF 与 IEEE 的关系

| 方面 | OIF | IEEE 802.3 |
|------|-----|------------|
| 焦点 | 光网络互联 | 以太网标准 |
| 层次 | 物理层/链路层 | 物理层/PHY |
| 灵活性 | 更灵活 | 更正式 |
| 速度 | 快速迭代 | 标准化周期长 |
| 互补 | 提供 CEI 等电气规范 | 定义系统标准 |

### 1.3 OIF 主要技术领域

```
OIF 技术领域:
├── 高速电气接口 (CEI)
├── 光模块规范
├── FlexE/FlexO
├── 互通测试 (Interop)
└── 运营商需求
```

## 2. CEI (Common Electrical Interface)

### 2.1 CEI 概述

**CEI** 定义了光模块与系统设备之间的电气接口规范，是光网络互联的关键标准。

### 2.2 CEI 版本演进

```
CEI 演进时间线:

2006: CEI-3G (3 Gbps)
     └── 8b/10b 编码

2010: CEI-10G (10 Gbps)
     └── 64b/66b 编码

2014: CEI-28G (28 Gbps)
     └── NRZ

2018: CEI-56G (56 Gbps)
     └── PAM4

2020: CEI-112G (112 Gbps)
     └── PAM4

2024: CEI-224G (224 Gbps)
     └── PAM4 (研究中)
```

### 2.3 主要 CEI 规范

#### CEI-28G

| 类型 | 说明 | 距离 |
|------|------|------|
| CEI-28G-VSR | Very Short Reach | < 2m |
| CEI-28G-MR | Medium Reach | < 10m |
| CEI-28G-LR | Long Reach | < 20km |

**关键参数**:
- 速率: 28 Gbps/lane
- 调制: NRZ
- 编码: 64b/66b

#### CEI-56G

| 类型 | 说明 | 距离 |
|------|------|------|
| CEI-56G-VSR | Very Short Reach | < 2m |
| CEI-56G-MR | Medium Reach | < 10m |
| CEI-56G-LR | Long Reach | < 40km |

**关键参数**:
- 速率: 56 Gbps/lane
- 调制: PAM4 (28 Gbaud)
- 编码: 64b/66b

#### CEI-112G

| 类型 | 说明 | 距离 |
|------|------|------|
| CEI-112G-XSR | Extra Short Reach | < 0.2m |
| CEI-112G-VSR | Very Short Reach | < 2m |
| CEI-112G-MR | Medium Reach | < 10m |
| CEI-112G-LR | Long Reach | < 40km |

**关键参数**:
- 速率: 112 Gbps/lane
- 调制: PAM4 (56 Gbaud)
- 编码: 64b/66b

#### CEI-224G (研究中)

| 类型 | 说明 | 距离 |
|------|------|------|
| CEI-224G-XSR | Extra Short Reach | < 0.1m |
| CEI-224G-VSR | Very Short Reach | < 2m |

**关键参数**:
- 速率: 224 Gbps/lane
- 调制: PAM4 (112 Gbaud)
- 目标: 1.6T/3.2T 系统

## 3. CEI 关键电气参数

### 3.1 差分阻抗

```
阻抗要求:
- 差分阻抗: 100 Ω ± 10%
- 单端阻抗: 50 Ω ± 10%
- 回波损耗: > 10 dB @ 目标频率
```

### 3.2 眼图要求

**CEI-112G-VSR 眼图**:
```
眼高: > 50 mV (典型)
眼宽: > 0.3 UI
过渡时间: < 20 ps
抖动: < 0.15 UI RMS
```

### 3.3 抖动规范

**CDR 要求**:
| 参数 | CEI-56G | CEI-112G |
|------|---------|----------|
| 抖动产生 | < 0.15 UI | < 0.12 UI |
| 抖动容限 | > 0.3 UI | > 0.3 UI |
| 抖动带宽 | < 10 MHz | < 10 MHz |

## 4. 光模块电气接口

### 4.1 QSFP-DD 电气接口

**引脚定义**:
```
差分对:
- 8 发送对 (TX)
- 8 接收对 (RX)

控制信号:
- SCL/SDA (I2C)
- MODPRSn (存在检测)
- RESETn (复位)
- INTn (中断)
```

**功耗限制**:
| 类型 | 功耗 |
|------|------|
| QSFP-DD 400G | < 12 W |
| QSFP-DD 800G | < 24 W |

### 4.2 OSFP 电气接口

**特点**:
- 8 通道
- 更高密度
- 改进散热

### 4.3 CFP2/DCO 相干模块

**电气接口**:
```
CEI-112G 兼容
差分 100 Ω
功耗: < 30 W (相干)
```

## 5. FlexE (Flexible Ethernet)

### 5.1 FlexE 概述

**FlexE (Flexible Ethernet)** 是 OIF 制定的灵活以太网接口规范，实现多厂商以太网设备的互联。

### 5.2 FlexE 架构

```
FlexE 架构:

┌─────────────────────────────────────────────┐
│                   FlexE Shim                 │
├─────────────────────────────────────────────┤
│                                              │
│  40G/100G/200G/400G 以太网                   │
│                                              │
│  ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐        │
│  │ 子通道 │  │ 子通道 │  │ 子通道 │  │ 子通道 │        │
│  │  0   │  │  1   │  │  2   │  │  3   │        │
│  └─────┘  └─────┘  └─────┘  └─────┘        │
│                                              │
└─────────────────────────────────────────────┘
```

### 5.3 FlexE 版本

| 版本 | 说明 | 通道化 |
|------|------|--------|
| FlexE 1.0 | 基础规范 | 支持 |
| FlexE 2.0 | 增强特性 | 支持 |
| FlexE 3.0 | 800G 支持 | 支持 |

### 5.4 FlexE 应用

**速率绑定**:
- 2x 100G → 200G
- 4x 100G → 400G
- 8x 100G → 800G

**子速率映射**:
- 将多个低速率服务映射到高速率链路
- 增强灵活性

## 6. FlexO (Flexible Optical)

### 6.1 FlexO 概述

**FlexO (Flexible Optical)** 是针对光网络的灵活接口规范。

### 6.2 FlexO vs FlexE

| 特性 | FlexE | FlexO |
|------|-------|-------|
| 层级 | MAC 层 | PHY 层 |
| 介质 | 铜缆/光纤 | 光纤 |
| 应用 | 以太网互联 | 光网络互联 |
| 绑定 | 电气绑定 | 光层绑定 |

### 6.3 FlexO 应用

**DWDM 系统**:
- 多波长 FlexO 绑定
- 可变速率
- 灵活栅格

## 8. OIF CEI-Linear (线性电气接口)

### 8.1 CEI-Linear 概述

**CEI-Linear** 是 OIF 针对 Linear Drive (非重定时) 光模块新定义的电气接口规范。

### 8.2 Linear vs Retimed

| 特性 | CEI-112G (Retimed) | CEI-112G-Linear |
|------|---------------------|------------------|
| 重定时 | 是 | 否 |
| CDR | 模块内含CDR | 主机侧CDR |
| 功耗 | 高 (~15pJ/b) | 低 (~5pJ/b) |
| 延迟 | ~100ns | ~0 |
| 眼图 | CDR恢复后 | 线性放大后 |

### 8.3 CEI-Linear 合规测试点

```
主机(TX)              光模块              主机(RX)
   │                    │                    │
   │    TP1 (电气)     │    TP2 (光)       │
   ▼                    ▼                    ▼
┌──────────────────────────────────────────────┐
│  CEI-Linear TX spec   │   Optical spec    │
└──────────────────────────────────────────────┘
   ▲                    ▲                    ▲
   │    TP4 (电气)     │    TP3 (光)       │
   │                    │                    │
```

| 测试点 | 位置 | 说明 |
|--------|------|------|
| TP1 | 主机ASIC输出 | 发射电气接口 |
| TP2 | 模块输入(光) | 光发射机输入 |
| TP3 | 模块输出(光) | 光接收机输出 |
| TP4 | 模块输出(电气) | 接收电气接口 |

### 8.4 CEI-Linear 目标应用

| 应用 | 距离 | 功耗目标 |
|------|------|---------|
| 前面板可插拔 | <500m | <8W/800G |
| CPO/NPO | <2m | <5W/800G |
| Die-to-Die | <50mm | <2W/800G |
