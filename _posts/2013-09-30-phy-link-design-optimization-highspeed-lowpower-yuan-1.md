## 2. 研究背景与问题定义

### 2.1 研究背景

**数据中心发展现状：**
- 全球数据流量持续快速增长，据Cisco报告，到2023年全球互联网用户将达53亿(占66%人口)
- 数据中心数量快速增长，预计2021年全球超大规模数据中心达628个
- 数据中心用电量约占全球用电量的1%，若按需扩展，2030年可能达到8%
- 需要开发高速、低功耗、低成本的数据中心设备

**SerDes技术需求：**
- SERDES(Serializer-Deserializer)是高数据速率通信的首选方案
- 可实现更少电缆、低功耗、低接地隔离器需求、低连接器成本
- 线缆减少有利于服务器间气流流通，降低冷却需求

### 2.2 技术挑战

**长距离(Long-Reach)背板通道特性：**
- 通道长度：约1米
- 28 GHz奈奎斯特频率下衰减：约30 dB(不含封装损耗)
- PAM-4调制方案成为主流，可实现双倍数据速率

**OIF CEI-112G应用空间：**
| 通道类型 | 28GHz衰减 | 均衡要求 |
|---------|----------|---------|
| XSR(超短距) | 6-10 dB | 轻量均衡 |
| VSR(短距) | 12-16 dB | FEC + Tx FIR + Rx CTLE |
| MR(中距) | 20 dB | FEC + 多抽头均衡 |
| LR(长距) | 28-30 dB | FEC + 多抽头均衡 |

### 2.3 核心问题定义

本论文聚焦于**112 Gbps PAM-4长距离背板接收机的低功耗模拟前端(AFE)设计**，具体问题包括：

1. **功耗问题**：传统CML架构的CTLE功耗较高，需要创新架构降低功耗

2. **线性度问题**：PAM-4电压电平间距更紧，对线性度要求更高

3. **带宽问题**：30 dB衰减@28 GHz的通道需要有效的均衡方案

4. **PVT变化问题**：需要可调谐性以跟踪工艺、电压、温度变化

## 3. 核心技术方案

### 3.1 系统架构

**接收机模拟前端架构：**

```
输入信号 → 前端端接网络 → CTLE(含HF/MF/Buffer级) → 输出缓冲 → 后端无源网络
                ↓
           CMFB回路(共模反馈)
```

**CTLE三级架构：**
1. **高频提升级(HF-CTLE)**：高频增益提升，补偿通道衰减
2. **中频增益级(MF-CTLE)**：改善中频响应，减少码间干扰(ISI)
3. **缓冲级(Buffer)**：驱动后级电路

### 3.2 关键电路模块

#### 3.2.1 前端端接网络(Front-End Termination Network)

**设计目标**：实现理想50 Ω端接，同时提供一定被动均衡

**电路实现**：
- 采用电感Lₜ与端接电阻串联
- 在输入端形成高通滤波器特性
- 提升高频增益，补偿通道高频衰减

**传输函数**：
```
H(s) = (R_L + sL_t) / (R_L + sL_t + R_s)
```

#### 3.2.2 CMOS三态反相器作为放大器单元

**创新核心**：利用CMOS反相器替代传统CML架构

**优势**：
- 高跨导(gm)提供低功耗均衡方案
- 可配置为可调电阻(二极管连接负载)
- 可配置为可调电感(主动电感负载)
- 三态反相器支持功率可扩展性

**基本配置**：
```
反相器作为放大器：
- NMOS和PMOS作为差分对工作
- 提供高gm/功耗比

反相器作为可调电阻：
- 二极管连接配置
- 通过偏置调节等效电阻

反相器作为可调电感：
- 主动电感实现
- 用于高频提升
```

#### 3.2.3 高频提升级(HF-CTLE)

**设计目的**：补偿通道的高频衰减

**实现方式**：
- 利用主动电感负载提升高频增益
- 可调谐参数：Trim_hf
- 实现高频 boost

**电路拓扑**：
```
输入 → 反相器放大器(主动电感负载) → 输出
         ↑
    高频提升控制(Trim_hf)
```

#### 3.2.4 中频增益级(MF-CTLE)

**设计目的**：
- 改善中频响应
- 减少后码ISI(Main-cursor与post-cursor比率)
- 提高眼图开口

**设计发现**：
- 无MF-CTLE：垂直眼高26 mVpp，水平眼开57% UI
- 有MF-CTLE：垂直眼高60 mVpp，水平眼开57% UI
- MF-CTLE使眼高增加约131%

#### 3.2.5 共模反馈回路(CMFB)

**设计目的**：提高CTLE的共模抑制比(CMRR)

**优势**：
- 相比自偏置二极管连接反相器负载，功耗更低
- 提供低功耗偏置方案

**CMRR分析**：
| 级 | CMRR |
|----|------|
| HF-CTLE | ~11 dB |
| MF-CTLE | ~13 dB |
| Buffer | ~10 dB |
| 整体 | 满足要求 |

### 3.3 工艺与设计参数

**工艺**：16 nm FinFET

**关键参数**：
| 参数 | 数值 |
|------|------|
| 电源电压 | 0.8 V |
| 数据速率 | 112 Gbps PAM-4 |
| 奈奎斯特频率 | 28 GHz |
| 通道衰减 | 30 dB @ 28 GHz |
| CTLE均衡量 | 17 dB |
| 功耗 | 10 mW |

## 4. 关键设计与创新点

### 4.1 CMOS反相器替代CML架构

**创新点1：基于CMOS反相器的低功耗CTLE**

传统方案使用CML(电流模式逻辑)架构，而本论文创新性地使用CMOS反相器作为放大器：

**CML架构问题**：
- 需要尾电流源，功耗固定
- gm/功耗比低
- 带宽受限

**CMOS反相器优势**：
- 无尾电流源损耗
- 高跨导效率
- 支持功率可扩展
- 与FinFET工艺兼容

**对比分析**：
| 指标 | CML方案 | CMOS反相器方案 |
|------|---------|---------------|
| 功耗效率 | 低 | 高 |
| 线性度 | 较好 | 需CMFB改善 |
| 可调谐性 | 有限 | 多维度可调 |
| PVT跟踪 | 困难 | 支持 |

### 4.2 前端被动均衡网络

**创新点2：前端端接网络的被动均衡**

在传统50 Ω端接基础上增加电感Lₜ：
- 实现输入端高频提升
- 减少对主动均衡的依赖
- 降低功耗

**频率响应优化**：
- 低频：电阻端接，无boost
- 高频：电感提供高通特性
- 提升整体均衡效率

### 4.3 多级CTLE架构优化

**创新点3：HF+MF+Buffer三级架构**

**各阶段功能**：
1. **HF-CTLE**：高频补偿，通道衰减补偿
2. **MF-CTLE**：中频增益，改善ISI
3. **Buffer**：驱动能力，增强负载驱动

**眼图改善效果**：
| 配置 | 垂直眼高 | 改善幅度 |
|------|---------|---------|
| 无MF-CTLE | 26 mVpp | 基准 |
| 有MF-CTLE | 60 mVpp | +131% |

### 4.4 可调谐性设计

**创新点4：多维度可调谐 knobs**

| 调谐参数 | 功能 |
|---------|------|
| Trim_hf | 高频boost控制 |
| Trim_bias_hf | 高频偏置控制 |
| Trim_res_hf | 高频电阻调节 |
| Trim_dc | 直流增益控制 |
| Trim_mf | 中频增益控制 |
| Trim_buf1 | 缓冲级控制 |

**PVT跟踪能力**：
- 支持工艺角(SS/TT/FF)跟踪
- 支持温度变化补偿
- 支持电压变化调整

## 6. 技术优势与局限性

### 6.1 技术优势

1. **超低功耗**：仅10 mW，远低于传统CML方案的20-100 mW

2. **单级CTLE均衡**：使用CTLE作为唯一主动均衡手段，简化系统复杂度

3. **FinFET工艺兼容**：利用16 nm FinFET的高跨导特性

4. **可扩展性**：三态反相器支持功率模式切换

5. **PVT鲁棒性**：多维度调谐支持全工艺角覆盖

6. **长距离支持**：30 dB衰减通道验证有效

### 6.2 局限性

1. **仅RX均衡**：未讨论TX侧FFE优化

2. **单一通道类型**：主要针对背板通道，其他通道类型未验证

3. **FEC依赖**：30 dB通道可能需要FEC辅助达到目标BER

4. **线性度限制**：PAM-4线性度要求严格，需进一步验证

5. **封装效应**：未考虑封装对通道的影响

## 9. 可借鉴的设计思路/参数

### 9.1 核心设计公式

**前端端接网络传输函数**：
```
H(s) = (R_L + sL_t) / (R_L + sL_t + R_s)
```

**CTLE幅度响应(简化)**：
```
|H(jω)| ≈ gm × Z_L(ω) / (1 + gm × Z_L(ω))
```

**主动电感实现**：
```
L_eff = C_gs × (gm/gm')² / gds²
```

**带宽与上升时间关系**：
```
BW ≈ 0.35 / Trise
```

### 9.2 关键设计参数

**前端设计参数**：
| 参数 | 推荐值 |
|------|--------|
| 端接电阻 | 50 Ω |
| 电感L_t | 优化值(看仿真) |
| 输入电容 | 最小化 |

**CTLE设计参数**：
| 参数 | 推荐值 |
|------|--------|
| DC增益 | 0-3 dB |
| 高频boost | 10-20 dB |
| 带宽 | >28 GHz |
| CMFB环路带宽 | <信号带宽 |

**偏置参数**：
| 参数 | 典型值 |
|------|--------|
| VDD | 0.8 V |
| gm | 最大化(功耗允许) |
| 负载电阻 | 中等(带宽/增益权衡) |

### 9.3 设计方法论

**低功耗CTLE设计流程**：
```
1. 确定通道特性(衰减量、带宽需求)
2. 规划均衡架构(HF/MF/Buf级配比)
3. 选择放大器拓扑(CMOS反相器优势)
4. 设计可调谐 knobs(多维度)
5. 验证PVT角覆盖
6. 优化功耗效率
```

**眼图优化策略**：
```
目标：max(垂直眼高 × 水平眼宽)

1. 增加HF boost → 改善高频衰减
2. 优化MF gain → 减少post-cursor ISI
3. 控制CMFB → 稳定共模
4. 调谐Trims → PVT补偿
```

### 9.4 关键电路模块参考

**CMOS三态反相器单元**：
- 可配置为放大/电阻/电感模式
