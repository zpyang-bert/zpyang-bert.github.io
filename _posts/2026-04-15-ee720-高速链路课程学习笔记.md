## 知识体系结构

### 模块一：基础理论 (L1-L4)
```
intro → channels → tdr/s-param → channel pulse model
```
**核心概念**：
- 高速串行通信基础
- 信道损耗与带宽
- TDR 测量原理
- S参数定义

### 模块二：TX/RX 电路 (L5-L8)
```
termination → tx driver → rx circuits → eq(tx/rx)
```
**核心概念**：
- 阻抗匹配与端接
- TX driver 设计
- RX 前端电路
- 预加重/去加重 (TX EQ)
- CTLE/DFE (RX EQ)

### 模块三：信号完整性 (L9-L10)
```
noise sources → jitter
```
**核心概念**：
- 噪声分类与预算
- 抖动分析与测量
- TJ/BJ/DJ 分析

### 模块四：时钟架构 (L11-L14)
```
plls → cdrs → fwd clock deskew → clk distribution
```
**核心概念**：
- PLL 基本原理
- CDR 架构
- 前向时钟技术
- 时钟分配网络

### 模块五：光通信 (L15 + Extra)
```
optical io → ultra-high-speed wireline
```
**核心概念**：
- 光 I/O 接口
- 超高速电链路

## 重点讲义详解

### Lecture 1: Introduction
- 高速链路 vs 并行总线
- 速率与距离的关系
- 串行通信的优势

### Lecture 2: Channels
- 信道频率响应
- 损耗类型（skin effect, dielectric）
- 阻抗不连续性

### Lecture 3: TDR & S-Parameters
- 时域反射测量
- S11, S21 参数含义
- 信道带宽定义

### Lecture 5: Termination & TX Driver
- 阻抗匹配策略
- TX driver 架构
- 眼图质量优化

### Lecture 7-8: Equalization
- TX EQ: FFE, pre-emphasis
- RX EQ: CTLE, DFE, AGC
- 自适应均衡算法

### Lecture 10: Jitter
- 抖动分类
- 随机抖动 vs 确定性抖动
- 眼图模板测试

### Lecture 11-12: Clocking & CDR
- PLL 带宽与稳定性
- CDR 架构选择
- 时钟恢复环路

### Lecture 15: Optical I/O
- 光模块接口
- SerDes 与光模块互联

## 学习建议

1. **先 L1-L4**：建立信道基础概念
2. **再 L5-L8**：理解 TX/RX 电路与均衡
3. **后 L9-L14**：深入信号完整性与时钟
4. **最后 L15**：扩展到光通信
