## 一、论文核心贡献

### 1.1 技术突破

本文展示了**世界首款GaAs-on-TFLN异构集成平台**，在1064nm波段实现了全集成光发射机。

**核心成就**：
| 指标 | 数值 | 意义 |
|------|------|------|
| 波长 | **1064nm** | 空心光纤应用 |
| 调制带宽 | **>100GHz** | TFLN class |
| NRZ速率 | **100 Gb/s** | 直接调制 |
| PAM4速率 | **160 Gb/s** | 高阶调制 |
| SMSR | **>50 dB** | 优异边模抑制 |
| 功率 | **>10 mW** | 芯片输出 |
| 调谐范围 | **>20 nm** | Vernier环实现 |

### 1.2 1064nm波段的战略价值

**为什么选择1064nm？**

| 传统波段 | 特点 | 1064nm优势 |
|----------|------|-----------|
| 850nm | VCSEL主导 | 无源光器件成熟 |
| 1310nm | 零色散 | 低延迟空心光纤 |
| 1550nm | 最小衰减 | 铒镱共掺放大 |

**空心光纤 (HCF) 驱动**：
```
HCF优势：
- 延迟降低 ~30%（以空气为主）
- 非线性效应极低
- 波长选择更灵活

HCF损耗特性：
- 660nm: 低损耗窗口
- 850nm: 低损耗窗口
- 1064nm: 低损耗窗口 ✓

1064nm + HCF = 极低延迟互联
```

**单片集成价值**：
```
传统方案：

## 三、GaAs激光器设计

### 3.1 DFB激光器性能

**1064nm GaAs DFB**：
| 参数 | 数值 | 说明 |
|------|------|------|
| 波长 | 1064nm | Yb光纤增益峰 |
| SMSR | >50dB | 单模工作 |
| 阈值电流 | 27mA | 适中 |
| 输出功率 | >10mW | 耦合进TFLN |

**DFB光栅设计**：
```
光栅周期：
Λ = λ/(2×n_eff)
  = 1064nm/(2×3.4)
  ≈ 156nm

 grating耦合系数：
κ ≈ 50-100 cm⁻¹ (典型)
```

### 3.2 Vernier可调谐激光器

**双环Vernier原理**：
```
两个环：
- 主环 (Main Ring): FSR₁
- 从环 (Vernier Ring): FSR₂

FSR差 = FSR₁ - FSR₂

有效FSR增大：
FSR_eff = FSR₁ × FSR₂ / |FSR₁ - FSR₂|

实现 >20nm 调谐范围
```

**连续可调谐**：
```
调谐机制：
- 热光效应
- 电流注入

温度调谐系数：
dn/dT ≈ 2×10⁻⁴/K

环周长 ~ 1mm
Δλ/ΔT ≈ 0.1nm/K

20nm调谐需要 ~200K温度变化
→ 多个热调谐电极
```

### 3.3 半导体光放大器 (SOA)

**SOA功能**：
```
1. 输出功率放大
   - 小信号增益: 15-20 dB
   - 饱和输出: >15 dBm

2. 门控功能
   - 快速开关
   - 功率控制

3. 集成优势
   - 与DFB工艺兼容
   - 共享外延结构
```

## 四、TFLN调制器设计

### 4.1 薄膜铌酸锂优势

**TFLN vs 体材料LiNbO₃**：
| 特性 | TFLN | 体材料 | 优势 |
|------|------|--------|------|
| 厚度 | 0.5-1 μm | 3-5mm | 微波匹配 |
| Vπ·L | ~2-3V·mm | ~4V·mm | 低驱动 |
| 带宽 | >100GHz | ~40GHz | 高速 |
| 成本 | 高 | 低 | 体材料 |
| 耦合 | 挑战 | 成熟 | 体材料 |

### 4.2 MZM横截面

**TFLN波导结构**：
```
        SiO₂ 包层 (BPSG)
    ═══════════════════════
    │    LiNbO₃ 薄膜    │  ← 0.5-1 μm
    │   (x-cut, y-propagation)
    │  ┌──────────────┐  │
    │  │  电极        │  │  ← CPW电极
    │  │  (TWE)      │  │
    │  └──────────────┘  │
    └─────────────────────┘
         SiO₂ 掩埋层
            Si 衬底
```

### 4.3 高频性能

**S参数测量**：
| 参数 | 数值 | 说明 |
|------|------|------|
| S₁₁ | < -10dB @ 100GHz | 阻抗匹配 |
| S₂₁ (EO) | > -3dB @ 100GHz | 3dB带宽>110GHz |

**带宽受限因素**：
```
1. 电极损耗
   - 趋肤效应
   - 介质损耗

2. 速度失配
   - 光微同速

3. 模式匹配
   - 微波/光波模式

```

**Nested Antiresonant HCF**：
```
结构：
- 悬浮内核

性能对比：
