## 一、论文核心贡献

### 1.1 技术突破

本文展示了**TFLN EOM与驱动EIC的晶圆级异构集成**，实现了封装后>100GHz带宽的CPO引擎。

**核心成就**：
| 指标 | 数值 | 意义 |
|------|------|------|
| EO带宽(封装前) | **110 GHz** | 极限带宽 |
| EO带宽(封装后) | **>100 GHz** | 封装保持率>90% |
| Vπ | **4 V** | CMOS兼容驱动 |
| 插入损耗 | **<4.2 dB** | 低损耗 |
| S₁₁ | **<-10 dB @ 100GHz** | 良好匹配 |
| 时间偏差 | **0.8 fs @ 1s** | 飞秒级精度 |
| MDEV | **1.2×10⁻¹⁵ @ 1s** | 频率稳定度 |

### 1.2 CPO的战略价值

**共封装光学 vs 可插拔**：

| 特性 | CPO | 可插拔 |
|------|-----|--------|
| 距离 | <2m | >2m |
| 功耗 | <5 pJ/bit | ~15 pJ/bit |
| 延迟 | <1ns | ~10ns |
| 密度 | 极高 | 中 |
| 散热 | 共享 | 独立 |
| 可维护性 | 低 | 高 |
| 成本(小批量) | 高 | 低 |

**封装挑战**：
```
传统封装问题：
- Wire-bond → 电感限制 ~30GHz
- 芯片级封装 → 寄生效应
- 光耦合 → 损耗增加

晶圆级方案优势：
- 短互连 → <0.5 dB @ 100GHz
- 高密度 → 支持>200GHz
- 批量生产 → 成本降低
```

## 二、晶圆级异构集成工艺

### 2.1 工艺流程

**6步晶圆级异构集成**：
```
Step 1: TFLN EOM芯片准备
         - 薄膜LiNbO₃波导
         - CPW电极
         ↓
Step 2: EIC (驱动IC)芯片准备
         - SiGe/CMOS驱动放大器
         - 50Ω匹配网络
         ↓
Step 3: 首次Seed层沉积
         - Cr/Cu (15nm/150nm)
         ↓
Step 4: 芯片贴装与对准
         - Die-to-wafer键合
         - 对准精度 <1μm
         ↓
Step 5: 聚合物填充
         - 底部填充
         - 平面化
         ↓
Step 6: 第二次Seed层 + 电镀
         - 12μm厚光刻胶
         - 3μm Cu + 1μm Au电镀
         ↓
         → 芯片级互连 + 扇出重布线
```

### 2.2 互连结构

**电镀铜柱互连**：
```
互连参数：
- 铜柱高度: ~12 μm
- 铜柱直径: ~10 μm
- 节距: ~50 μm
- 电阻: <10 mΩ/柱
- 电感: <0.1 nH

对比Wire-bond：
- Wire-bond: 0.3-0.5 nH/mm
- 芯片柱: <0.1 nH total

→ 带宽提升3-5×
```

### 2.3 扇出重布线层 (RDL)

**重布线功能**：
```
从芯片阵列 → 晶圆级互联

RDL层：
- 介电: BCB或PI
- 金属: Cu/Au
- 线条宽度: 5-10 μm
- 间距: >5 μm
- 阻抗: 50Ω 匹配
```

### 4.2 94 GHz高频调制

**为什么选择94 GHz？**
```
1. TFLN带宽足够
   - EOM 3dB > 100 GHz

2. 降低噪声影响
   - 高频调制 → SNR提升
   - 抗环境干扰

3. 频率稳定度改善
   - 相位噪声归一化
```

### 4.3 飞秒级精度

**时间偏差结果**：
```
1秒平均时间:
- 时间偏差 (σ): 0.8 fs
- 相当于光程 0.24 μm

10秒平均:
- σ ≈ 0.25 fs

100秒平均:
- σ ≈ 0.08 fs

→ 飞秒精度稳定性
```

** Allan偏差 / MDEV**：
```
MDEV (改良Allan偏差):

τ (平均时间)    MDEV

100 s          8×10⁻¹⁷

- 传统光学梳: 10⁻¹⁵ @ 1s
- 本方案: 1.2×10⁻¹⁵ @ 1s ✓

优势:
```

## 六、异构集成深度技术

### 6.1 芯片贴装对准

**Die-to-wafer键合**：
```
对准精度要求: <1 μm

面临挑战:
1. 热膨胀失配
   - Si: 2.6×10⁻⁶/K
   - LiNbO₃: 0.5×10⁻⁶/K
   - 温差 ΔT = 100°C
   - 失配: ~0.26 μm ✓

2. 表面平整度
   - TIR < 5 μm
   - 污染控制

3. 键合强度
   - >5 MPa 剪切强度
```

### 6.2 应力管理

**封装应力影响**：
```
应力来源:
- 热膨胀失配
- 模塑料固化收缩
- 操作机械应力

对TFLN的影响:
- Pockels效应变化
- 相位漂移
- 消光比恶化

缓解措施:
- 应力缓冲层
- 局部加强筋
- 温度补偿
```

### 6.3 热管理

**功耗分布**：
| 组件 | 功耗 | 散热方式 |
|------|------|----------|
| TFLN EOM | ~100mW | 被动散热 |
| 驱动EIC | ~500mW | 主动散热 |
| SOA/激光 | ~2W | 热沉 |
| 总计 | ~2.6W | 共享热管理 |
