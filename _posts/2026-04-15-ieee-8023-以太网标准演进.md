
## 2. IEEE 802.3 关键技术

### 2.1 PAM4 调制

**为什么从 NRZ 转向 PAM4**:

| 方面 | NRZ @ 50G | PAM4 @ 50G |
|------|-----------|------------|
| 带宽需求 | 25 GHz | 12.5 GHz |
| 编码效率 | 1 bit/sym | 2 bits/sym |
| SNR 要求 | 低 | 高 (~9dB) |
| 眼图 | 1 眼 | 3 眼 |

**PAM4 标准要求**:
- RS-FEC (Reed Solomon FEC)
- Gray 编码
- 眼图模板测试

### 2.2 RS-FEC (Reed Solomon Forward Error Correction)

**作用**: 纠错编码，降低误码率

**配置 (400G Ethernet)**:
```
RS(544,514)
- 544 符号块
- 514 数据符号
- 30 纠错符号
- 可纠正: 15 符号错误
```

**性能提升**:
- 无 FEC: BER ~ 1e-5
- 有 RS-FEC: BER ~ 1e-12
- 编码增益: ~6 dB

### 2.3 Auto-Negotiation

**功能**: 自动协商速率/双工

**协商流程**:
```
1. 检测链路伙伴
2. 交换能力 (100M/1G/10G/...)
3. 选择最高公共速率
4. 建立连接
```

**40G/100G 特殊考虑**:
- 不支持 Auto-Neg
- 使用训练序列建立链路

## 3. 802.3 标准体系 (2024-2026)

### 3.1 现行标准

| 标准 | 名称 | 说明 |
|------|------|------|
| 802.3ae | 10GbE | 光纤/铜缆 |
| 802.3ba | 40G/100G | 多通道并行 |
| 802.3bj | 100G Backplane | 背板以太网 |
| 802.3bm | 40G/100G Fiber | 多模光纤 |
| 802.3cd | 50G | 4 级 PAM4 |
| 802.3bs | 200G/400G | 8x25G / 4x50G |
| 802.3cb | 25G/50G | 对称以太网 |
| 802.3cz | 100G BASE | 多模光纤 |
| 802.3dj | 800G/1.6T | 下一代 (制定中) |
| 802.3df | 224G/lane | 224G SerDes (制定中) |

### 3.2 802.3dj (800G/1.6T Ethernet)

**目标**:
- 800 Gbps (8x100G)
- 1.6 Tbps (16x100G 或 8x200G)
- PAM4 调制

**物理层规范**:
| 规范 | 距离 | 介质 |
|------|------|------|
| 800GBASE-SR8 | 100m | MMF |
| 800GBASE-LR8 | 10km | SMF |
| 800GBASE-FR8 | 2km | SMF |
| 1.6TBASE-SR16 | 100m | MMF |
| 1.6TBASE-LR16 | 10km | SMF |

### 3.3 802.3df (224G SerDes)

**关键参数**:
- 单通道: 224 Gbps
- 调制: PAM4
- 波特率: 112 Gbaud

**应用**:
- 1.6T/3.2T Ethernet
- 支持更长距离
- 芯片到芯片互联

## 4. 以太网物理层架构

### 4.1 层次结构

```
OSI 层级:
┌─────────────────────────────────────┐
│       Application Layer             │
├─────────────────────────────────────┤
│       Presentation Layer            │
├─────────────────────────────────────┤
│         Session Layer               │
├─────────────────────────────────────┤
│         Transport Layer              │
├─────────────────────────────────────┤
│          Network Layer              │
├─────────────────────────────────────┤
│       Data Link Layer               │
│    ├── LLC (802.2)                  │
│    └── MAC (802.3)                  │
├─────────────────────────────────────┤
│       Physical Layer                │
│    ├── 802.3 (Ethernet)             │
│    └── SerDes/PHY                   │
└─────────────────────────────────────┘
```

### 4.2 以太网帧结构

```
前导码    帧开始    目的MAC    源MAC    类型    数据    FCS
(7B)     (1B)     (6B)     (6B)    (2B)   (46-1500B) (4B)
```

### 4.3 编码方式

**8b/10b 编码 (早期)**:
- 效率: 80%
- 直流平衡
- 嵌入时钟

**64b/66b 编码 (当前)**:
- 效率: 97%
- 用于 10G+
- 帧同步

**128b/132b 编码 (800G)**:
- 用于 800G/1.6T
- 更高效率

### 5.1 背板以太网 (Backplane)

**BASE-KR**:
| 参数 | 规范 |
|------|------|
| 速率 | 10G/25G/56G |
| 距离 | < 1m |
| 阻抗 | 100 Ω 差分 |

### 5.2 光模块接口

**QSFP28 (100G)**:
- 4x25G
- 功耗 < 3.5W
- 间距: MR/LR/SR

**QSFP-DD (400G/800G)**:
- 8x50G (400G)
- 8x100G (800G)
- 向后兼容 QSFP28

**OSFP (800G)**:
- 8x100G
- 更高密度
- 热管理改进

### 5.3 DAC/ACC/AOC

| DAC | 直连铜缆 (Direct Attach Copper) | < 5m |
| ACC | 有源铜缆 (Active Copper Cable) | < 10m |
| AOC | 有源光缆 (Active Optical Cable) | < 100m |

## 6. 抖动与眼图要求

### 6.1 抖动预算

**总抖动 (TJ) 分解**:
```
TJ = BJ + DJ

其中:
- BJ (随机抖动): 高斯分布, 无法消除
- DJ (确定性抖动): 可预测, 可补偿

典型抖动预算:
- TJ: < 0.3 UI
- BJ: < 0.15 UI
- DJ: < 0.15 UI
```

### 6.2 眼图模板

**NRZ 眼图模板**:
```
            眼高 > 200 mV
            眼宽 > 0.4 UI
              │
       ───────┼───────
       ───────┼───────
    ─────────┼─────────
    ─────────┼─────────
       ───────┼───────
       ───────┼───────
```

**PAM4 眼图模板**:
```
       眼高 (最上层) > 50 mV
       眼高 (中间) > 20 mV
              │
    ──────────┼───────── 11
    ──────────┼───────── 10
    ──────────┼───────── 01
    ──────────┼───────── 00
```
