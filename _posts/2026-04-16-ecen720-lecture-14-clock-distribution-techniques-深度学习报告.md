## 一、导线缩放

### 1.1 VLSI互连导线

```
技术节点缩放:
• 从N到N+1节点，所有事物理想缩放0.7x
• 密度增加2x

导线电阻变化:
• 导线电阻增加2x
• 显著增加导线RC延迟
• 补偿方法: 导线变高

电容变化:
• 电容增长较慢
• 更高导线增加侧壁电容
• 低k介质有助于减少电容
```

![](/img/serdes/fundamentals/lectures/lecture10_ee720_jitter_深度学习报告/_images/img-000.jpg)

> 🔍 深度说明：
> 【研究背景】随着工艺节点从N缩放到N+1，导线宽度和厚度通常按0.7x缩放，但高度可能增加以补偿电阻增加。铜互连中的趋肤效应和晶界散射导致导线电阻增加，RC延迟成为制约全局信号传输的关键因素。
> 【核心结论】导线电阻随节点缩放呈2倍增长，而导线电容增长较慢。变高导线是常用的补偿策略，但会导致侧壁电容增加，需要引入低k介质来缓解。
> 【工程价值】在先进工艺节点（如7nm、5nm）的时钟分布网络设计中，导线RC延迟可能超过数十FO4（Fan-out 4）门延迟，严重限制时钟到达时间。需要采用分布式缓冲器和电感峰化技术。
> 【落地注意】时钟网络的金属层选择至关重要，高层金属具有更低的电阻率和更好的厚度控制，有利于减少RC延迟。

### 1.2 导线延迟

```
1cm导线的FO4延迟:
• 全局片上导线RC延迟变成很多(100+)门延迟
• (如果用一个集中驱动器驱动)

限制:
• RC主导的片上导线 vs (R)LC主导的片外导线
```

### 1.3 导线带宽限制

```
全局片上导线带宽远差于芯片到芯片信道

原因:
• RC主导 vs (R)LC主导
• 片上导线性能差距
```

## 二、时钟分布

### 2.1 串行I/O系统时钟分布

```
两种架构:
• 嵌入式时钟系统
• 前向时钟系统

都需要:
• 裸片全局时钟分布网络
```

![](/img/serdes/fundamentals/lectures/lecture14_ee720_clk_distribution_深度学习报告/_images/img-077.jpg)

> 🔍 深度说明：
> 【研究背景】串行I/O系统需要稳定的时钟基准来同步数据采样。嵌入式时钟系统将时钟嵌入数据流进行传输，前向时钟系统则从TX单独转发时钟到RX，两种架构都需要高效的裸片时钟分布网络。
> 【核心结论】时钟分布网络必须提供低抖动的时钟信号到所有接收端，同时最小化skew。传统的H树结构适用于集中式时钟源，而网状结构更适合分布式时钟架构。
> 【工程价值】在112Gb/s PAM-4系统中，时钟分布网络的抖动性能直接影响系统整体性能，需要仔细设计缓冲器级数和布局。
> 【落地注意】时钟分布网络的功耗可能占整个TX功耗的15-20%，需要在性能和功耗之间进行权衡。

### 2.2 级联时钟缓冲器

```
缓冲器链:
• 每级有JTF和相位噪声(PN)
• 总输出抖动计算:
T_j_o = (...((T_j_i × JTF_1 + PN_1) × JTF_2 + PN_2)...) × JTF_N + PN_N

每级贡献累积
```

### 2.3 时钟缓冲器JTF

```
在时钟分布链中存在显著抖动放大
• 28Gb/s在时钟分配链中有抖动放大

设计动机:
• 促使112Gb/s系统使用¼速率时钟方案
• 14GHz时钟
• 需要正交相位间隔和占空比校正以获得均匀输出眼图
```

![](/img/serdes/fundamentals/lectures/lecture14_ee720_clk_distribution_深度学习报告/_images/img-079.jpg)

> 🔍 深度说明：
> 【研究背景】时钟缓冲器的JTF描述了缓冲器对输入抖动的传递特性。在级联结构中，每一级的JTF都会影响最终的输出抖动。早期设计中忽略了JTF的累积效应，导致实际性能与预期不符。
> 【核心结论】时钟缓冲器的低通频率响应导致高频抖动被放大，放大倍数与缓冲器的上升/下降时间相关。在28Gb/s时钟分配链中，抖动放大效应显著，需要特别关注。
> 【工程价值】112Gb/s系统采用¼速率时钟方案（14GHz）而非½速率（28GHz），是因为更高频率的时钟会经历更严重的抖动放大。选择较低频率时钟可以有效降低抖动放大效应。
> 【落地注意】时钟缓冲器的设计需要在带宽和抖动放大之间进行权衡，使用感应峰化可以扩展带宽但会增加电路复杂度。

### 2.4 14GHz正交时钟分布

```
PLL输出相位噪声被时钟分布JTF倍增

CDR滤波器(高通)应用于获得未跟踪的有效TX抖动

性能:
• LC-PLL: 208fs (3MHz CDR BW), 185fs (10MHz CDR BW)
```

## 四、CML2CMOS转换器

### 4.1 基本CML2CMOS转换器

```
差分输入级 + 高摆幅输出级

潜在问题:
• 对电源噪声敏感
• 降低低摆幅分布技术的抖动优势
• 通常需要占空比控制
```

![](/img/serdes/fundamentals/lectures/lecture11_ee720_clocking_arch_plls_深度学习报告/_images/img-080.jpg)

> 🔍 深度说明：
> 【研究背景】CML2CMOS转换器用于将低摆幅CML时钟信号转换为高摆幅CMOS信号，以便驱动后续的数字电路。在高速SerDes中，时钟分布网络通常使用CML以保持低抖动，但接收端可能需要CMOS电平。
> 【核心结论】基本CML2CMOS转换器对电源噪声敏感，因为其高增益特性会放大电源噪声。降低低摆幅分布技术的抖动优势是主要设计挑战。
> 【工程价值】选择合适的CML2CMOS转换器架构对于保持时钟质量至关重要。在设计时需要权衡噪声敏感度和占空比性能。
> 【落地注意】使用LDO为转换器供电可以有效隔离电源噪声，但会增加功耗和面积。

### 4.2 AC耦合自偏置反相器输入级

```
改进架构:
• AC耦合自偏置反相器输入级
• 跨耦合缓冲器级
• 改善占空比性能
```

![](/img/serdes/fundamentals/lectures/lecture14_ee720_clk_distribution_深度学习报告/_images/img-082.jpg)

> 🔍 深度说明：
> 【研究背景】AC耦合自偏置反相器输入级通过隔直电容消除DC偏移，自偏置机制自动设置工作点，跨耦合缓冲器提供额外的增益和带宽扩展。这种架构改进了基本CML2CMOS的占空比性能。
> 【核心结论】AC耦合消除了输入偏置问题，自偏置提供了稳定的工作点，跨耦合缓冲器改善了瞬态响应。占空比失真可以降低到1%以下。
> 【工程价值】该架构适用于对占空比要求严格的系统，如需要精确正交相位的时钟分布网络。
> 【落地注意】隔直电容的取值需要在低频截止频率和电容面积之间进行权衡，通常在100pF到1nF范围。

## 五、多相位生成

### 5.1 ILO-based多相位时钟生成

```
基于注入锁定振荡器的多相位生成:
• 注入时钟产生多个输出相位
• 粗频率调谐环路确保ILO锁定
• 细正交锁定环路最小化相位误差

输出:
• CK0, CK45, CK90, CK135, CK180, CK225, CK270, CK315
```

![](/img/serdes/fundamentals/lectures/lecture14_ee720_clk_distribution_深度学习报告/_images/img-083.jpg)

> 🔍 深度说明：
> 【研究背景】ILO利用注入锁定效应产生多个相位时钟输出。当振荡器被外部时钟注入时，输出相位锁定到注入时钟，其锁定带宽内的抖动被跟踪，之外的噪声被抑制。
> 【核心结论】ILO可产生8相或更多相位的时钟，精度取决于锁定环路的设计。粗频率调谐确保ILO工作在目标频率，细正交锁定环路通过反馈控制最小化相邻相位之间的误差。
> 【工程价值】ILO-based多相位生成适用于需要大量正交相位的系统，如相干通信和多通道数据转换器。其固有抖动过滤能力是其主要优势。
> 【落地注意】ILO的锁定带宽设计需要在抖动跟踪能力和噪声抑制之间进行平衡，通常选择目标频率的±10%作为锁定范围。
