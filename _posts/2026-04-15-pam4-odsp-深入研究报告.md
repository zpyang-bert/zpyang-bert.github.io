## 一、PAM4调制技术基础

### 1.1 为什么需要PAM4

随着高速SerDes数据率不断提升（56Gb/s, 112Gb/s, 224Gb/s），传统NRZ/PAM2面临严峻挑战：

| 问题 | NRZ面临挑战 |
|------|------------|
| 带宽压力 | 56Gb/s NRZ需要28GHz带宽 → PAM4只需14GHz |
| 信道损耗 | 高频损耗更严重 |
| 功耗 | 更高带宽意味着更高功耗 |
| 成本 | 光学组件成本高 |

**PAM4核心优势**：在相同符号率下将带宽需求减半

```
NRZ:     ■  ■  ■  ■  ■  ■  ■  ■   (2电平, 1符号=1bit)
PAM4:    ■■■■■■■■■■■■■■■   (4电平, 1符号=2bits)

频谱对比:
NRZ带宽需求:    28 GHz (56Gb/s)
PAM4带宽需求:   14 GHz (56Gb/s, 2bits/symbol)
```

### 1.2 PAM4信号特性

**4电平脉冲幅度调制**：

| 电平 | 逻辑电平 | 电压幅度（典型） |
|------|---------|----------------|
| 3 (上上) | 11 | +Fullscale |
| 2 (上下) | 10 | +1/3 Fullscale |
| 1 | 01 | -1/3 Fullscale |
| 0 (下下) | 00 | -Fullscale |

**眼图结构**：PAM4有三个叠加的眼图（上方眼、中间眼、下方眼）

```
PAM4眼图示意:
Level 3 ─────────────── 11
Level 2 ─────────────── 10
Level 1 ─────────────── 01  
Level 0 ─────────────── 00

垂直眼张开度: 仅为一个NRZ眼高的1/3
水平眼张开度: 受限于符号间干扰和噪声
```

**挑战**：
- 垂直眼高仅为NRZ的1/3 → SNR恶化约9.5dB
- 需更精确的判决阈值
- DAC/ADC分辨率要求更高

### 1.3 PAM4应用场景

| 应用场景 | 数据率 | 调制方式 |
|----------|--------|---------|
| 数据中心互连 | 56Gb/s/lane | PAM4 |
| 224G SerDes | 112Gb/s/lane | PAM4 |
| 400G/800G光模块 | 53.125GBd | PAM4 |
| 50G/100G以太网 | 26.56GBd | PAM4 |
| 超级计算机互联 | 56Gb/s/lane | PAM4 |

## 二、PAM4 oDSP系统架构

### 2.1 相干光通信系统概述

相干检测相比直接检测（DD）可实现更高频谱效率：

| 特性 | 直接检测 (DD) | 相干检测 (Coherent) |
|------|--------------|-------------------|
| 灵敏度 | 较低 | 高10-20dB |
| 频谱效率 | 低 | 高（PAM4/QPSK/QAM） |
| 线性要求 | 宽松 | 严格 |
| DSP复杂度 | 低 | 高 |
| 成本 | 低 | 高 |
| 色散补偿 | 需DCF | 数字补偿 |

### 2.2 相干光DSP链路架构

**典型相干光收发机DSP链**：

```
发射端:
┌──────┐  ┌────────┐  ┌─────────┐  ┌────────┐  ┌─────────┐  ┌─────────┐
│ FEC  │─▶│ Mapper │─▶│  DSP    │─▶│  DAC   │─▶│ 驱动    │─▶│ 光学    │
│ 编码 │  │(预均衡)│  │(星座图) │  │        │  │ 放大器  │  │ 调制器  │
└──────┘  └────────┘  └─────────┘  └────────┘  └─────────┘  └─────────┘

接收端:
┌──────┐  ┌────────┐  ┌─────────┐  ┌────────┐  ┌─────────┐  ┌─────────┐
│ 光学 │─▶│平衡    │─▶│   ADC   │─▶│  DSP   │─▶│ 载波    │─▶│ FEC     │
│ 检测 │  │相干接收│  │        │  │(均衡)  │  │ 恢复    │  │ 解码    │
└──────┘  └────────┘  └─────────┘  └─────────┘  └─────────┘  └─────────┘
```

### 2.3 关键DSP模块详解

#### 2.3.1 发射机DSP

**星座图映射 (Constellation Mapping)**
- QPSK: 4个星座点，2bit/symbol
- 16QAM: 16个星座点，4bit/symbol
- 64QAM: 64个星座点，6bit/symbol

**数字预均衡 (Pre-equalization)**
- 预补偿信道损伤
- FIR滤波器实现

#### 2.3.2 接收机DSP

**色散补偿 (Chromatic Dispersion Compensation)**
- 数字色散滤波器 (FIR-based)
- 长距离光纤必备

**时钟恢复与符号同步**
- Gardner算法
- Mueller-Muller算法

**均衡器**
- 静态均衡 (Static Equalizer)
- 自适应均衡 (Adaptive Equalizer)
  - LMS (Least Mean Squares)
  - RLS (Recursive Least Squares)

**载波恢复 (Carrier Recovery)**
- QPSK/QAM载波相位恢复
- VCO反馈控制

**偏振解复用 (Polarization Demux)**
- CMA (Constant Modulus Algorithm)
- MMA (Multi-Modulus Algorithm)

## 三、光学I/O系统

### 3.1 高速光链路系统

光学互连消除了许多信道限制：

```
典型高速光链路:
┌─────────────────────────────────────────────────────────────┐
│  TX侧:                                                        │
│  ┌──────┐  ┌──────┐  ┌──────┐  ┌────────┐  ┌───────┐      │
│  │ DSP  │─▶│ DAC  │─▶│ 驱动 │─▶│ 调制器  │─▶│ 光输出 │      │
│  └──────┘  └──────┘  └──────┘  └────────┘  └───────┘      │
│                                                              │
│  光纤通道:                                                     │
│  损耗: ~0.25dB/km (单模光纤@1550nm)                            │
│  带宽: >10THz                                                 │
│                                                              │
│  RX侧:                                                        │
│  ┌──────┐  ┌──────┐  ┌──────┐  ┌──────┐  ┌──────┐          │
│  │光检测│─▶│TIA   │─▶│ ADC  │─▶│ DSP  │─▶│ FEC  │          │
│  └──────┘  └──────┘  └──────┘  └──────┘  └──────┘          │
└─────────────────────────────────────────────────────────────┘
```

**光学互连优势**：
- 损耗极低（光纤0.25dB/km vs 电缆100dB/km@10GHz）
- 串扰极小
- 波导尺寸小
- 支持波分复用(WDM)实现高信息密度

### 3.2 波分复用 (WDM)

WDM允许在单根光纤上封装多个高速信号：

| WDM类型 | 波长间隔 | 通道数 |
|---------|---------|--------|
| CWDM | 20nm | 最多18通道 |
| DWDM | 0.8nm (~100GHz) | 80+通道 |
| UDWDM | <0.8nm | 160+通道 |

### 3.3 光学信道特性

#### 3.3.1 光纤基础

| 参数 | 典型值 |
|------|--------|
| 单模光纤损耗 | ~0.25dB/km @ 1550nm |
| 频率相关损耗 | <0.5dB/km (带宽>10THz) |
| 色散 | ~17ps/nm/km @ 1550nm |
| 非线性效应 | SPM, XPM, FWM |

#### 3.3.2 带宽限制因素

- **色散**: 脉冲展宽（长距离关键）
- **非线性**: 自相位调制(SPM)、交叉相位调制(XPM)
- **激光线宽**: 长距离(>1km)重要

### 3.4 芯片间波导实例

#### 12通道带状光纤 (Reflex Photonics)
- 12通道 @ 250μm间距
- 每通道10Gb/s → 40Gb/s/mm

#### PCB中聚合物波导
- 通道间距<100μm
- 每通道10Gb/s → 100+Gb/s/mm
- 对比：典型差分带状线间距~500μm

### 3.5 自由空间光学链路

- 光学成像系统实现芯片间光路由
- 无需物理波导连接
- 对准要求严格

### 3.6 CMOS波导技术

| 工艺 | 波导损耗 | 备注 |
|------|---------|------|
| Bulk CMOS | ~1000dB/cm | 需后处理降至10dB/cm |
| SOI | 低损耗 | 厚埋氧层良好光场约束 |
| 后端金属层 | 低损耗 | 可在金属化后加工 |

## 四、光学调制技术

### 4.1 直接调制 vs 外调制

| 方式 | 原理 | 优缺点 |
|------|------|--------|
| 直接调制 (DML) | 改变激光器注入电流 | 简单、便宜；但频率啁啾限制 |
| 外调制 (EML) | 电吸收调制器(EAM)+连续激光器 | 高速、低啁啾；但成本高 |

### 4.2 马赫-曾德调制器 (MZM)

**工作原理**：
- 干涉仪结构，两个臂分别调制
- 通过电极改变折射率（热光或电光效应）
- 输出为两臂信号的干涉

**传输函数**：
$$T = \frac{1}{2}[1 + \cos(\phi_1 - \phi_2)] = \cos^2\left(\frac{\phi_1 - \phi_2}{2}\right)$$

**驱动方式**：
- Push-pull操作：两臂反向驱动
- 半波电压$V_\pi$：产生π相移的电压

### 4.3 I/Q调制器

用于QPSK和更高级调制格式：

```
I路 ──▶ MZM_I ──┐
                   ├─▶ 输出
Q路 ──▶ MZM_Q ──┘
       ↑
    90°相移
```

### 4.4 硅光子调制器

| 类型 | 机制 | 速度 | 损耗 |
|------|------|------|------|
| 热光 | 折射率变化 | 慢(kHz) | - |
| 载流子耗尽 | PN结反向偏置 | 数十GHz | 中等 |
| 载流子注入 | PN结注入 | 快 | 低 |

**载流子耗尽型**（主流）：
- 40+GBd PAM4可行
- CMOS兼容工艺

## 五、光学接收技术

### 5.1 直接检测接收机

```
光输入 ──▶ PIN/APD ──▶ TIA ──▶ 限幅放大器 ──▶ 判决电路
```

**PIN光电二极管**：
- 无增益
- 响应度~0.8A/W @ 1550nm
- 高速应用

**APD雪崩光电二极管**：
- 内部增益~10-20x
- 更高灵敏度
- 需高压偏置

### 5.2 相干检测接收机

**平衡检测架构**：
```
信号光 ──▶ 90°混频器 ◀── 本振光(LO)
              │
              ▼
         平衡光电二极管对
              │
              ▼
         I路 + Q路 → ADC → DSP
```

**优势**：
- 灵敏度提高10-20dB
- 保留幅度和相位信息
- 支持高级调制

### 5.3 跨阻放大器 (TIA)设计

**关键参数**：

| 参数 | 目标 | 说明 |
|------|------|------|
| 带宽 | 0.7×符号率 | 均衡后优化 |
| 跨阻增益 | >70dBΩ | 71dBΩ典型 |
| 输入噪声 | <20pA/√Hz | 低噪声 |
| 线性 | 高 | PAM4需要 |
| 功耗 | <50mW | 受限 |

**带宽扩展技术**：
- Cherry-Hooper放大器
- 分布式放大器
- 可调谐反馈

### 6.1 112Gbps PAM4模拟前端

**关键模块**：

| 方面 | 电气SerDes | 光学SerDes |

## 九、PAM4 oDSP设计流程

### 9.1 系统设计步骤

1. **需求定义**
   - 数据率、距离、BER目标
   - 功耗、成本约束

2. **调制格式选择**
   - NRZ (<25Gb/s)
   - PAM4 (25-100Gb/s)
   - DP-QPSK/16QAM (相干光, >100Gb/s)

3. **信道分析**
   - 损耗预算
   - 带宽需求
   - 串扰评估

4. **均衡策略**
   - TX FFE + RX CTLE + DFE
   - 全DSP (ADC-Based)

5. **测试验证**
   - 眼图分析
   - BER测试
   - 抖动容限测试

### 9.2 光学系统额外考虑

1. **光功率预算**
   - 发射功率 (~0dBm典型)
   - 接收灵敏度
   - 链路损耗

2. **色散管理**
   - 短距离：光纤色散小
   - 长距离：需CD补偿

3. **非线性效应**
   - 高功率下显著
   - 需功率控制

## 附录：PAM4关键公式

**PAM4符号映射**：
$$b_{2k+1}, b_{2k} \rightarrow s_k \in \{+3, +1, -1, -3\}$$

**PAM4理论SNR**（BER=10⁻¹²）：
$$SNR_{PAM4} \approx 23.5dB$$
$$SNR_{NRZ} \approx 14dB$$
$$\Delta SNR \approx 9.5dB$$

**带宽效率**：
$$\eta = \frac{\log_2(M)}{T_s} [bit/s/Hz]$$

其中M=4 for PAM4, T_s=符号周期
