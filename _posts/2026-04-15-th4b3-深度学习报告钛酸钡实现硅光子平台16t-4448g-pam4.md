# Th4B.3 深度学习报告：钛酸钡实现硅光子平台1.6T (4×448G PAM4)

**原始论文**：McGill University + Lumiphase AG + Ciena — "Barium Titanate Enabling Net 1.6T (4x448 Gbps PAM4) On a Silicon Photonics Platform"
**整理日期**: 2026/04/15
**研究深度**: 深度解读（结合BTO材料科学、硅光子集成、薄膜调制器、DR4多通道）


## 一、论文核心贡献

### 1.1 技术突破

本文展示了**世界首款基于钛酸钡(BTO)的DR4芯片**，在硅光子平台上实现单片集成，达成1.6T总容量。

**核心成就**：
| 指标 | 数值 | 意义 |
|------|------|------|
| 总容量 | **1.6 Tb/s** | 4×448G |
| 单通道速率 | **448 Gbps** | PAM4 @ 225GBd |
| EO带宽(6dB) | **105 GHz** | 业界领先 |
| EO带宽(3dB) | **55 GHz** | 高效调制区 |
| 差分Vπ | **6 V** | BTO低驱动 |
| 消光比 | **>35 dB** | 高ER |
| 通道数 | **4通道** | DR4配置 |
| 温度范围 | **20-85°C** | 宽温度运行 |

### 1.2 为什么选择BTO

**BTO vs TFLN vs 其他材料**：

| 材料 | DC Pockels系数 | 带宽 | Vπ·L | CMOS兼容 |
|------|----------------|------|-------|----------|
| **BTO** | 高 | >100GHz | 低 | ✓ |
| TFLN | ~30 pm/V | >100GHz | ~3V·mm | ✓ |
| LiNbO₃ | ~30 pm/V | ~40GHz | ~4V·mm | ✗ |
| SiPh | ~0.8 pm/V | ~60GHz | 高 | ✓ |

**BTO独特优势**：
```
1. 已知最大DC Pockels系数之一
2. 低光学损耗
3. 与标准硅光子工艺兼容
4. 薄膜后仍保持高压电效应
5. Curie温度在薄膜中显著提高
```

## 二、钛酸钡 (BTO) 材料特性

### 2.1 晶体结构

**钙钛矿结构**：
```
BTO 化学式: BaTiO₃

晶体结构 (钙钛矿):
       O²⁻
      ╱   ╲
   Ba²⁺    Ti⁴⁺
      ╲   ╱
       O²⁻

四方相 (室温):
- c/a ≈ 1.01
- 自发极化: ~26 μC/cm²
- 相变温度 (Curie): 120°C (体材料)
```

**薄膜vs体材料**：
```
薄膜BTO特性:
- Curie温度: >250°C (vs 120°C体材料)
- 应力增强效应
- 晶粒尺寸效应

→ 高温稳定性显著提升
```

### 2.2 Pockels效应

**线性电光系数**：
```
r₃₃ (BTO) ≈ 30-50 pm/V (最优取向)

相位调制:
Δφ = (2π/λ) · n³·r·E·L/2

对于BTO:
- n ≈ 2.4
- r₃₃ ≈ 40 pm/V
- E = V/d

半波电压-长度积:
Vπ·L ≈ λ/(2·n³·r₃₃) ≈ 3-5 V·mm
```

### 2.3 DC Pockels效应

**低频高压电效应**：
```
BTO的DC Pockels系数：

优势：
- 直流稳定
- 无电荷积累
- 低频调制

对比传统Pockels：
- 传统LiNbO₃: DC漂移明显
- BTO: DC稳定 ~±4%
```

## 四、多通道系统

### 4.1 DR4配置

**4通道并行架构**：
```
输入:
QD DFB激光器 (20 dBm)
    ↓
1×4 分路器 (片上)
    ↓
通道1-4 (各自MZM)
    ↓
片上监控PD
    ↓
光纤输出
```

**光源选择**：
```
量子点DFB (QD DFB):
- 波长: O-band
- 功率: 20 dBm
- 线宽: 窄
- 温度稳定: 好
```

### 4.2 调制格式

**PAM4/6/8支持**：
| 格式 | 符号率 | 每通道速率 | 总容量 |
|------|--------|-----------|--------|
| PAM4 | 225 GBd | 448 Gbps | 1.6T |
| PAM6 | 225 GBd | 506 Gbps | 1.8T |
| PAM8 | 225 GBd | 540 Gbps | 1.9T |

### 4.3 封装方案

**OIF兼容CDM封装**：
```
封装配置:
- BTO裸die
- 差分RF驱动器 (up to 30 dB gain @ 100 GHz)
- OIF兼容接口
- IMDD系统适配

Wire-bond连接:
- 驱动器 → BTO差分RF接口
- 优化: 短wire-bond减少寄生
```

## 五、实验结果

### 5.1 PAM4性能

**4×448G @ 225GBd**：
```
BER vs 符号率：

MZM1: ● (最好)
MZM2: ●
MZM3: ● (最差, 1dB更高IL)
MZM4: ●

所有MZM @ 225 GBd:
- BER < 4.5×10⁻³ (HD-FEC门限) ✓
- BER < 2.2×10⁻⁴ (KP4门限) ✓
```

**眼图结果**：
```
225 GBd PAM4:
- BER: 2.7×10⁻³
- DFE均衡后
- RTO 8次平均

眼图张开清晰
```

### 5.2 PAM6/8性能

**PAM6 @ 225GBd**：
```
506 Gbps (225×log₂(6) ≈ 587 Gbps?)

BER vs 符号率：

**PAM8 @ 225GBd**：
```

符号率范围:

### 5.3 温度特性

**20°C-85°C性能**：
```
温度扫描结果：

20°C: ● (最好)
50°C: ●
85°C: ● (略差)

性能下降原因:
- 驱动器增益降低

BTO本身特性:

```

## 六、与TFLN对比

### 6.1 带宽对比

**EO带宽对比**：
| 指标 | BTO (本文) | TFLN | 对比 |
|------|-----------|------|------|
| 3dB带宽 | 55 GHz | >100 GHz | 较低 |
| 6dB带宽 | **105 GHz** | >100 GHz | 相当 |
| Vπ | 6V | 2-4V | 较高 |

**带宽定义差异**：
```
3dB vs 6dB:
- 3dB: 传统定义 (功率降一半)
- 6dB: 考虑到阻抗匹配

BTO的6dB=105GHz表示：
- 阻抗匹配后性能
- 仍有调制能力
```

### 6.2 集成优势

**硅光子平台BTO**：
```
vs TFLN:
- TFLN: 独立调制器
- BTO: 与SiN单片集成

优势：
- 监控PD单片集成
- 热调器单片集成
- 50Ω终结单片集成
- 成本更低
```

### 6.3 性能瓶颈

**当前限制**：
```
Vπ = 6V (较高)
- 需要更高功率驱动器
- 对比TFLN的2-4V

改进方向：
- BTO薄膜优化
- 电极设计优化
- 降低Vπ
```
