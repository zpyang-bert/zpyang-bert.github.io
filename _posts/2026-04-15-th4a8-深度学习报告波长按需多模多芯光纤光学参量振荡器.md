**核心成就**：

| 光源 | 带宽 | 调谐 | 功率密度 |

## 二、四波混频原理

### 2.1 四波混频 (FWM)

**FWM基本原理**：
```
三阶非线性过程：
ω₁ + ω₂ → ω₃ + ω₄ (简并FWM)

能量守恒：
ω₁ + ω₂ = ω₃ + ω₄

动量守恒 (相位匹配):
Δk = k₃ + k₄ - k₁ - k₂ = 0

频率关系：
ω₃ = ω₁ + (ω₁ - ω₂)
ω₄ = 2ω₂ - ω₁
```

**泵浦-信号-闲频**：
```
泵浦 (ωp): 强功率
信号 (ωs): 待放大
闲频 (ωi): 生成的

相位匹配条件：
Δk = 2γPp + Δk_linear = 0

其中：
- γ = 非线性系数
- Pp = 泵浦功率
- Δk_linear = 线性相位失配
```

### 2.2 多模FWM

**模式间FWM (intermodal FWM)**：
```
泵浦: LP₀₁ 模式
信号: LP₁₁ 模式

相互作用：
m₁ + m₁ → m₂ + m₃

相位匹配条件放宽：
- 不同模式有效折射率差
- 提供额外相位匹配自由度

优势：
- 宽调谐范围
- 多波长生成
```

**多模相位匹配**：
```
有效折射率差:
Δn_eff = n_eff(m₁) - n_eff(m₂)

模式色散贡献:
Δk_disp = (β₂ + β_modal)·Δω²

实现条件：
β₂·Δω² ≈ Δk_linear
```

### 2.3 参量增益

**增益带宽**：
```
G ∝ γ·Pp·L_eff

其中：
- γ = 非线性系数 (~10 /W/km for SMF)
- Pp = 泵浦功率
- L_eff = 有效长度

增益谱宽:
Δω_g ≈ 2√(γ·Pp/|β₂|)

典型值：
γ = 10 /W/km
Pp = 10 W
β₂ = -20 ps²/km
Δω_g ≈ 2 THz
```

## 三、拉曼散射协同

### 3.1 受激拉曼散射 (SRS)

**SRS基本原理**：
```
Stokes频移:
Ω_R ≈ 13.2 THz ( silica fiber)

功率转移：
dPs/dz = -αs·Ps + gR·Ps·Pp

其中：
- gR = 拉曼增益系数
- Pp = 泵浦功率
- Ps = Stokes功率
```

### 3.2 FWM + SRS协同

**本文创新点**：
```
协同机制：
1. FWM提供相位敏感增益
2. SRS提供宽谱带增益
3. 两者互补

优势：
- 效率 > 40% (vs 单独FWM < 10%)
- 覆盖更宽频带
- 降低泵浦功率要求
```

**实验验证**：
```
泵浦: 1040 nm
Stokes波段: ~1230 nm
Anti-Stokes波段: ~900 nm

SRS频移: ~6.7 THz (~70nm)
FWM频移: ~13.2 THz (~140nm)

协同效果:
- 6.7 THz处: 10%转换, ~200mW
- 13.2 THz处: 40%转换, ~800mW
```

### 4.2 双芯与四芯光纤

**双芯光纤 (DCF)**：
```
两个高折射率芯

特性：
- 强耦合
- 超模 (symmetric/antisymmetric)
- 可用于定向耦合

应用：
- 开关
- 耦合器
- 双参量振荡
```

**四芯光纤 (4CF)**：
```
四个芯排列

可能的配置：
- 方形排列
- 菱形排列
- 线性排列

优势：
- 多个参量通道
- 更高功率处理
```

### 4.3 高双折射设计

**纤芯高折射率差**：
```
Δn ≈ 0.01-0.02

应力双折射：
- Birefringence B ≈ 10⁻⁴
- 拍长 L_B ≈ λ/B ≈ 10-100 μm

偏振保持：
- 拍长短 → 偏振保持好
- 用于相干应用
```

## 六、实验结果

### 6.1 单模vs多模泵浦

**LP₀₁ vs LP₁₁泵浦对比**：
| 泵浦模式 | 单通道增益 | OPO后增益 | 效率 |
|----------|-----------|-----------|------|
| LP₀₁x | 高 | 高 | ~40% |
| LP₀₁y | 高 | 高 | ~40% |
| LP₁₁ax | 低 | 高 | ~10% |
| LP₁₁bx | 低 | 高 | ~10% |

**分析**：
```
LP₀₁泵浦：
- 已经是单模 → 单通道FWM
- 高单程增益
- 腔反馈增益小

LP₁₁泵浦：
- 多模 → 多通道FWM
- 低单程增益
- OPO提供强反馈 → 大幅提升
```

### 6.2 频移范围

**±45 THz调谐**：
```
泵浦: 1040 nm

Stokes频移:
- Δω = -45 THz → λ ≈ 1230 nm
- Δω = +45 THz → λ ≈ 900 nm

Anti-Stokes频移:
- Δω = +45 THz → λ ≈ 780 nm
- Δω = -45 THz → λ ≈ 1014 nm

覆盖范围:
~780 nm - 1230 nm (450 nm带宽)
```

### 6.3 多波长同时生成

**双频移实验**：
```
6.7 THz + 13.2 THz同时

Stokes波段:
- ~1067 nm (6.7 THz): ~10% 转换
- ~1089 nm (13.2 THz): ~40% 转换

Anti-Stokes波段:
- ~1014 nm (6.7 THz)
- ~996 nm (13.2 THz)

功率:
- 1067 nm: ~200 mW
- 1089 nm: ~800 mW
```

**光谱拼接概念**：
```
1040 nm泵浦 → 6.7 THz (FWM)

           → 19.9 THz

拼接效应：

**模式配对选择**：
```
m₁-m₃ 相互作用:
- 频移 ~6.7 THz
- 需要LP₀₁→LP₁₁泵浦

m₂-m₃ 相互作用:
- 频移 ~13.2 THz

## 十一、深度技术问题

### 11.1 阈值条件

**OPO阈值计算**：
```
阈值功率:
P_th ≈ (α·L)² / (γ·L)²

其中：
- α = 腔损耗 (dB/m)
- L = 光纤长度
- γ = 非线性系数

典型值：
α = 0.2 dB/km @ 1040 nm
L = 25 m
γ = 10 /W/km
P_th ≈ 0.5 W
```

### 11.2 偏振态控制

**PM光纤中偏振**：
```
主轴 (快轴/慢轴):
- 拍长 L_B ≈ 2-4 mm
- 偏振消光比 > 30 dB

对FOPO的影响:
- 偏振保持 → 稳定振荡
- 模式竞争 → 抑制
```

### 11.3 模式竞争

**多模式振荡稳定性**：
```
竞争机制:
- 模式竞争泵浦功率
- 模式间能量转移
- 空间 Hole burning

抑制方法:
- 模式选择滤波器
- 非对称耦合
- 强泵浦饱和
```
