## 1. 标准概述

### 1.1 标准背景

IEEE 802.3cd-2018是IEEE 802.3以太网标准的第3号修正案，于2018年12月5日获得批准，2019年2月15日正式发布。该标准在IEEE Std 802.3-2018的基础上新增了Clause 131至Clause 140以及Annex 135A至Annex 136D，定义了50 Gb/s、100 Gb/s和200 Gb/s速率的以太网MAC参数和物理层规范。

### 1.2 核心定位

802.3cd是以太网技术发展中的"承上启下"标准：
- **向上**：衔接已定义的200GbE和400GbE标准（Clause 119-120）
- **向下**：扩展25GbE和50GbE应用场景
- **创新**：首次在以太网中大规模应用PAM4调制技术

### 1.3 支持的物理层类型

| 速率 | 类型 | 介质 | 调制方式 |
|------|------|------|----------|
| 50 Gb/s | 50GBASE-SR | 多模光纤(MMF) | PAM4 |
| 50 Gb/s | 50GBASE-FR | 单模光纤(SMF) | PAM4 |
| 50 Gb/s | 50GBASE-LR | 单模光纤(SMF) | PAM4 |
| 50 Gb/s | 50GBASE-CR | 铜缆 | PAM4 |
| 50 Gb/s | 50GBASE-KR | 背板 | PAM4 |
| 100 Gb/s | 100GBASE-SR2 | 多模光纤 | PAM4 |
| 100 Gb/s | 100GBASE-KR2 | 背板 | PAM4 |
| 100 Gb/s | 100GBASE-CR2 | 铜缆 | PAM4 |
| 100 Gb/s | 100GBASE-DR | 单模光纤 | PAM4 |
| 200 Gb/s | 200GBASE-SR4 | 多模光纤 | PAM4 |
| 200 Gb/s | 200GBASE-KR4 | 背板 | PAM4 |
| 200 Gb/s | 200GBASE-CR4 | 铜缆 | PAM4 |

## 3. MAC参数规范

### 3.1 50GMII (50 Gb/s Media Independent Interface)

802.3cd定义了50GMII作为50 Gb/s速率的MAC与PHY之间的接口：

- **数据宽度**：当使能时为64位，禁用时为32位
- **时钟域**：使用50.000 GBd的TX_CLK和RX_CLK
- **信号**：
  - TXD[63:0] / RXD[63:0]：发送/接收数据
  - TX_CLK / RX_CLK：时钟信号
  - TX_EN / RX_DV：数据有效指示
  - TX_ER / RX_ER：错误指示
  - CRS_COL：载波检测/冲突
  - MDC/MDIO：管理接口

### 3.2 EEE (Energy Efficient Ethernet) 支持

50GBASE-R PHY支持IEEE 802.3az EEE节能以太网特性：
- 快速唤醒时间（Fast Wake）
- LPI (Low Power Idle) 模式管理

## 4. 50Gb/s物理层架构

### 4.1 层级结构

50Gb/s以太网采用分层架构，从MAC到物理介质：

```
MAC Layer
    |
    v
Reconciliation Sublayer (RS)
    |
    v
50GMII (50 Gb/s Media Independent Interface)
    |
    v
Physical Coding Sublayer (PCS) - 64B/66B encoding
    |
    v
Reed-Solomon FEC (RS-FEC) - Clause 134
    |
    v
Physical Medium Attachment (PMA) - Clause 135
    |
    v
Physical Medium Dependent (PMD) - Clause 136-140
    |
    v
Physical Medium
```

### 4.2 编码方案

| 子层 | 编码类型 | 说明 |
|------|----------|------|
| PCS | 64B/66B | 与100GBASE-R/200GBASE-R兼容 |
| FEC | Reed-Solomon (544,514) | RS(544,514) for 50GBASE-R |
| PMA | PAM4 | 2比特/符号 |
| PMD | NRZ/PAM4 | 根据物理类型不同 |

## 5. Reconciliation子层与50GMII接口 (Clause 132)

### 5.1 功能概述

Clause 132定义了50 Gb/s速率的Reconciliation Sublayer (RS)及其与50GMII的映射关系。

### 5.2 50GMII结构

50GMII支持两种模式：
- **64位模式**：全速50 Gb/s工作
- **32位模式**：向后兼容或降低复杂度实现

### 5.3 LPI (Low Power Idle) 支持

RS子层负责：
- LPI断言检测
- LPI指示传播
- 快速唤醒支持

## 6. 物理编码子层 PCS (Clause 133)

### 6.1 50GBASE-R PCS规范

Clause 133定义了50GBASE-R的PCS，与已有的100GBASE-R PCS (Clause 82) 保持兼容。

### 6.2 64B/66B编码

- **块结构**：将64位数据编码为66位块
- **同步头**：2位同步头用于块边界识别
- **加扰**：使用自同步扰码器

### 6.3 对齐标记 (Alignment Markers)

PCS在多车道系统中使用对齐标记进行：
- 块同步
- 车道对齐
- 偏斜补偿

### 6.4 自动协商 (Auto-Negotiation)

Clause 133包含对背板以太网自动协商功能的支持，用于：
- 能力交换
- 链路配置
- 速率匹配

## 7. Reed-Solomon前向纠错 (Clause 134)

### 7.1 RS-FEC概述

Clause 134为50GBASE-R PHY定义了Reed-Solomon前向纠错子层，提供强大的纠错能力以支持长距离传输和高可靠性应用。

### 7.2 RS(544,514)编码

| 参数 | 值 |
|------|-----|
| 总符号数 | 544 |
| 数据符号 | 514 |
| 纠错能力 | 15符号 |
| 块长度 | 66位 × 514 ≈ 33.4 kb |

### 7.3 FEC功能块

#### 发送路径
1. **PCS车道块同步**
2. **PCS对齐锁定与去偏斜**
3. **PCS车道重排序**
4. **对齐标记移除**
5. **64B/66B到256B/257B转换**
6. **对齐标记映射与插入**
7. **Reed-Solomon编码**
8. **符号分配**
9. **发送比特排序**

#### 接收路径
1. **对齐锁定与去偏斜**
2. **FEC车道重排序**
3. **Reed-Solomon解码**
4. **对齐标记移除**
5. **256B/257B到64B/66B转换**
6. **块分配**
7. **对齐标记映射与插入**
8. **接收比特排序**

### 7.4 MDIO管理参数

FEC子层提供丰富的MDIO管理参数：

| 参数 | 说明 |
|------|------|
| FEC_bypass_indication_enable | FEC旁路使能 |
| FEC_degraded_SER_enable | 降级SER监测使能 |
| FEC_corrected_cw_counter | 纠正的码字计数器 |
| FEC_uncorrected_cw_counter | 未纠正的码字计数器 |
| BIP_error_counter_i | 比特间插校验错误计数器 |
| block_lock<x> | 块锁定状态 |
| am_lock<x> | 对齐标记锁定状态 |

### 7.5 可选功能

- **FEC Error Indication Bypass**：允许错误指示旁路
- **FEC Degraded SER**：支持降级符号错误率监测

## 8. 物理介质附加子层 PMA (Clause 135)

### 8.1 PMA位置与功能

Clause 135定义了50GBASE-R和100GBASE-P的PMA子层，位于FEC与PMD之间。

### 8.2 核心功能

#### 8.2.1 逐车道时钟与数据恢复 (CDR)
- 每个输入车道独立的CDR
- 从接收信号中恢复时钟

#### 8.2.2 比特级复用/解复用
- 支持多车道的并行到串行/串行到并行转换
- 50GBASE-R：1车道（可选聚合）
- 100GBASE-P：4车道

#### 8.2.3 偏斜与偏斜变化控制
PMA管理多车道系统中的偏斜问题：

| 偏斜点 | 说明 |
|--------|------|
| SP0 | PMA输入（来自FEC/PCS） |
| SP1 | PAM4编码前 |
| SP2 | PAM4编码后 |
| SP5 | PMA输出（到PMD） |
| SP6/SP7 | 各种测试点 |

#### 8.2.4 PAM4编码
- 采用四电平脉冲幅度调制
- 每符号2比特
- Gray码映射

### 8.3 PMA服务接口

PMA提供标准的服务接口供上层(FEC)和下层(PMD)调用。

### 8.4 测试模式

PMA支持多种测试模式：

#### NRZ信号测试图案
- PRBS31：伪随机二进制序列31
- PRBS9：伪随机二进制序列9
- 方波测试模式

#### PAM4信号测试图案
- PRBS13Q：四电平PRBS
- PRBS31Q：四电平PRBS31
- SSPRQ：短串行扰码伪随机
- 四电平方波

## 9. 50GBASE-CR/100GBASE-CR2/200GBASE-CR4电气背板 (Clause 136)

### 9.1 概述

Clause 136定义了铜缆连接的PMD子层规范，支持50GBASE-CR、100GBASE-CR2和200GBASE-CR4。

### 9.2 车道配置

| 类型 | 车道数 | 每车道速率 | 编码 |
|------|--------|------------|------|
| 50GBASE-CR | 1 | 25 GBd | PAM4 |
| 100GBASE-CR2 | 2 | 25 GBd | PAM4 |
| 200GBASE-CR4 | 4 | 25 GBd | PAM4 |

### 9.3 PMD控制功能

Clause 136定义了详细的训练帧结构和控制机制，用于均衡器训练和链路建立。

#### 训练帧结构
- **帧标记**：用于帧同步
- **控制字段**：
  - 初始条件请求
  - 调制和预编码请求
  - 系数选择
  - 系数请求
- **状态字段**：
  - 接收器就绪
  - 调制和预编码状态
  - 接收帧锁定
  - 初始条件状态
  - 奇偶校验位

#### 均衡控制
- 初始条件设置请求/响应过程
- 系数更新请求/响应过程
- 支持发送和接收均衡

### 9.4 调制和预编码

Clause 136支持PAM4调制，并定义了相应的预编码方案以优化信号质量。

### 9.5 电气特性

#### 发送器特性
- 输出波形规范
- 稳态电压和线性拟合脉冲峰值
- 系数初始化和步长
- J3u抖动
- 有效回波损耗(ERL)

#### 接收器特性
- 输入幅度容限
- 干扰容忍度
- 抖动容忍度
- 信号速率范围
- 接收器ERL

### 9.6 信道特性

定义了电缆组件的：
- 特性阻抗：85Ω或100Ω
- 插入损耗
- 有效回波损耗
- 微分到共模参数
- **Channel Operating Margin (COM)**：信道运行余量

## 15. 管理参数与MDIO支持

### 15.1 MDIO/MDC管理接口

802.3cd扩展了现有的MDIO管理框架，添加了50Gb/s和100Gb/s相关管理对象。

### 15.2 PHY设备属性

新增的PHY设备属性包括：

| 属性 | 说明 |
|------|------|
| aPhyType | PHY类型 |
| aPhyTypeList | 支持的PHY类型列表 |
| aSymbolErrorDuringCarrier | 载波期间的符号错误 |

### 15.3 能力寄存器

在IEEE 802.3标准的寄存器中添加了新的能力位：
- 50GBASE-LR能力
- 50GBASE-FR能力
- 50GBASE-SR能力
- 50GBASE-CR能力
- 50GBASE-KR能力

### 15.4 FEC管理

FEC子层的MDIO映射包括：
- FEC旁路指示使能
- FEC降级SER使能
- 纠错/未纠错码字计数器
- 每车道符号错误计数器
- 对齐状态

## 16. 技术特点总结

### 16.1 PAM4调制的广泛应用

802.3cd是首个大规模采用PAM4调制技术的以太网标准：

| 优势 | 说明 |
|------|------|
| 频谱效率 | 相比NRZ提升2倍 |
| 带宽需求 | 25 GBd可实现50 Gb/s |
| 成本效益 | 降低光/电器件带宽要求 |

### 16.2 RS-FEC的必要性

Reed-Solomon前向纠错是50Gb/s以太网的关键使能技术：
- 纠错能力：15符号/544符号块
- 编码增益：约5-7 dB
- 支持长距离传输
- 降低BER要求

### 16.3 链路训练与自适应

 Clause 136/137定义的训练机制：
- 自动均衡器训练
- 系数协商
- 调制方式协商
- 确保链路兼容性

### 16.4 兼容性设计

- 50GBASE-R PCS与100GBASE-R/200GBASE-R兼容
- 共享FEC架构
- 统一的管理接口
- 分层设计便于实现

### 16.5 应用场景

| 场景 | 推荐类型 |
|------|----------|
| 数据中心叶脊互联 | 100GBASE-SR4 |
| 数据中心到园区 | 100GBASE-DR |
| 背板互联 | 50GBASE-KR/100GBASE-KR2 |
| 芯片到光模块 | LAUI-2 C2M |
| 长距离单模 | 50GBASE-LR |
