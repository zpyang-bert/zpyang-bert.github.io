## 一、论文核心贡献

### 1.1 技术突破

本文展示了**业界首次多厂商互操作800G光传输**，跨越1500km的SSMF，使用800G-OpenROADM、800-ZR和800GBASE规格。

**核心成就**：
| 指标 | 数值 | 意义 |
|------|------|------|
| 传输距离 | **1500 km** (15×100km) | 超长距 |
| 厂商数量 | **6家** | 多厂商 |
| 规格 | **3种800G** | OpenROADM/ZR/GBASE |
| ROSNR | **~23.1 dB** | 低于25.1dB要求 |
| 频谱效率 | **150 GHz槽** | 间隔 |
| WSS级联 | **8个** | ROADM透明传输 |

### 1.2 为什么多厂商互操作重要

**开放光网络趋势**：
```
传统封闭系统：
- 单一厂商独占
- 锁定效应
- 成本高

开放系统需求：
- 多厂商互通
- 降低采购成本
- 提高网络灵活性
- 促进行业竞争
```

**800G可插拔光模块**：
```
应用场景：
- 城域网络 (MAN)
- 数据中心互联 (DCI)
- 以太网客户端

驱动因素：
- AI/ML流量增长
- IP-over-WDM应用
- 成本/功耗效率
```

**OpenROADM MSA**：

关键参数：
- 符号率: ~60 GBd

**Cisco/Acacia主导**：
```
ZR特点：
- 数据中心互联
- 固定调制 (QPSK/16QAM)
- 直连应用

关键参数：
- 距离: 120km (标准)
- 扩展: 可达 500km
- 低功耗
```

```
FR4特点：
- 2×400G通道
- CWDM4波长 (1271-1331nm)
- 双工光纤

应用：
- 2km距离
- 数据中心内部
```

## 三、概率星座图整形 (PCS)

### 3.1 首次在OpenROADM应用

**概率整形 vs 几何整形**：
```
几何整形 (Geo shaping):
- 星座点位置优化
- 均匀间隔

概率整形 (Probabilistic shaping):
- 星座点概率不等
- 中心点概率高
- 接近香农极限

本文首次用于OpenROADM
```

**实现方式**：
```
分布匹配 (Distribution Matching):
- 恒定成分分布
- 软判决FEC协同
- 降低-gap (约1dB)
```

### 3.2 优势

**PCS增益**：
```
vs 均匀QAM：
- 相同平均功率
- 更接近香农极限
- 约1dB SNR改善

适用场景：
- 高OSNR城域
- 长距离传输
```

## 七、关键技术

### 7.1 RRC滤波

**根升余弦滤波**：
```
Roll-off因子: ~0.125

频谱效率:
η = Rs / (Rs × (1+α))
  = 1 / (1+α)
  = 1 / 1.125
  = 0.89

vs α=0.2:
η = 0.83

更窄滤波 → 更高效率
```

### 7.2 功率谱密度恒定

**PSD归一化**：
```
100G通道: 50 GHz谱槽
800G通道: 150 GHz谱槽 (3×)

800G功率 = 3×100G功率

效果：
- WDM通道间功率平衡
- 非线性效应均匀
```

### 7.3 EDFA配置

**单级放大**：
```
每跨段EDFA:
- 增益: 补偿跨段损耗 (~20dB)
- 噪声指数: ~4.5 dB
- 简单可靠

vs 两级EDFA:
- 更低噪声
- 更高成本
- 适于长距离
```
