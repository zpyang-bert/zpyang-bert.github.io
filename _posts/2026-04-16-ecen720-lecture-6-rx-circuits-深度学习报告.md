## 一、接收机参数体系

### 1.1 RX关键性能指标

```
接收机三大核心参数:

1. 接收机灵敏度 (RX Sensitivity)
   • 输入参考噪声
   • 电压偏移
   • 最小锁存分辨率电压

2. 时域偏移
   • 孔径时间(Aperture Time)
   • 峰峰值抖动

3. 电压域偏移
   • 输入参考偏移
   • 比较器滞回

眼图约束:
• 最小眼宽 = 孔径时间 + 峰峰值抖动
• 最小眼高 = 灵敏度 + 峰峰值电压偏移
```

### 1.2 RX灵敏度分析

```
RX灵敏度公式:
V_pp,s = 2 × v_n_rms × SNR + V_min + V_offset

其中:
• V_pp,s: 最小可检测峰峰值信号
• v_n_rms: 输入参考噪声RMS值
• SNR: 给定BER所需的信噪比
• V_min: 最小锁存分辨率电压(来自滞回/有限再生增益)
• V_offset: 输入偏移

典型数值:
• v_n_rms = 1mV
• V_min + V_offset < 6mV
• BER=10⁻¹² → SNR≈7 → V_pp,s ≈ 20mV

高阶调制影响:
• PAM-4需要3个判决电平
• 每个电平的等效噪声裕量降低
```

![](lecture6_ee720_rx_circuits_images/img-387.jpg)

> 🔍 深度说明：
> 【研究背景】RX灵敏度是决定系统能检测的最小信号的关键指标，直接影响传输距离和信道衰减预算；PAM-4由于需要3个判决电平，灵敏度分析比NRZ更复杂。
> 【核心结论】灵敏度公式表明RX灵敏度由输入参考噪声、SNR要求、最小锁存分辨率电压和输入偏移共同决定；BER=10⁻¹²时，典型灵敏度需求约20mVppd。
> 【工程价值】在设计56Gb/s+ SerDes时，需要根据信道衰减预算反推RX灵敏度需求，进而确定前端噪声指标和比较器设计要求；眼图闭合度直接由灵敏度决定。
> 【落地注意】PAM-4的三个判决电平将等效噪声裕量降低约4倍（相比NRZ），对RX灵敏度和线性度要求更严格；设计时需考虑PAM-4的阈值漂移和校正机制。

## 四、时钟比较器

### 4.1 比较器类型

```
动态比较器分类:

Strong-Arm Latch:
• 无静态功耗
• 全CMOS输出电平
• 需要级联R-S latch

CML Latch:
• 高速
• 需要级联另一个CML latch
• 静态功耗

Charge-Steering Latch:
• 单时钟相位
• 快速
• 低电压操作
```

### 4.2 StrongARM Latch工作原理

**四阶段工作周期**：
```
1. Reset阶段
   • 时钟低电平
   • 节点预充电

2. Sampling阶段
   • 时钟变高, t₀
   • M1对放电X/X'
   • M2对放电out+/-
   
3. Regeneration阶段
   • PMOS开启, t₁
   • M1在线性区,电路不再敏感于v_in
   • 交叉耦合反相器通过正反馈放大

4. Decision阶段
   • 时钟开启后t₂时刻
   • 差分输出确定
```

**再生增益**：
```
G_R = exp((t₂ - t₁)/τ_R)

其中:
τ_R = C_out/(g_m2,r + g_m3,r)

再生时间常数越小 → 再生越快
```

### 4.3 RS Latch优化

**NIKOLIC JSSC 2000优化**：
```
优化目标:
• 对称pull-up和pull-down路径
• 显著加速

评估模式(时钟高):
• 大驱动晶体管激活
• 改变输出数据
• Keeper路径禁用

保持/预充电模式(时钟低):
• 大驱动晶体管三态
• 小keeper交叉耦合反相器保持数据

性能提升:
• Strong-Arm flip-flop延迟改善近2倍
• 优于其他先进flip-flop拓扑
```

### 4.4 采样器分析

**采样函数理论**：
```
采样器建模:
v_sample = ∫ v_in(τ) · h(τ) dτ

其中h(τ)是采样函数(加权函数)

理想采样函数:
• 积分等于1
• 是delta函数
• 采样值仅是精确采样时间的函数

实际采样函数:
• 在标称采样时间附近加权输入信号
• 有一定积分窗口
```

### 4.5 孔径时间

```
定义:
孔径时间 = 采样函数峰值的80%宽度

数学定义:
0.1 = ∫_{-∞}^{t₁₀} h(τ) dτ
0.9 = ∫_{-∞}^{t₉₀} h(τ) dτ

w₈₀ = t₉₀ - t₁₀

孔径时间越小 → 采样分辨率越高
```

### 4.6 比较器ISF分析

**时钟比较器LTV模型**：
```
比较器 = 噪声非线性滤波器 + 理想采样器 + 判决器

小信号响应用ISF(Impulse Sensitivity Function):
Γ(τ) = h(t_obs, τ)

输出电压:
v_o(t_obs) = ∫ v_i(τ) · Γ(τ) dτ

判决:
D_k = sgn(v_o(t_obs + kT))
```

### 4.7 Strong-Arm vs CML比较器

**采样时间比较**：
```
Strong-Arm Latch:
• 采样时间随VDD变化
• 对电源敏感

CML Latch:
• 采样时间对VDD不敏感
• 更稳定
```

**采样增益vs带宽**：
```
CML Latch:
• 小输入对时采样增益更高
• 增加输入对直接增加输出电容

StrongARM Latch:
• 更高采样带宽
• 增加输入对 → gm增加比电容更快
```

![](lecture6_ee720_rx_circuits_images/img-010.jpg)

> 🔍 深度说明：
> 【研究背景】时钟比较器是RX的核心判决单元，其采样时间特性和再生增益直接影响RX的时序裕量和灵敏度；Strong-Arm Latch是高速SerDes最流行的比较器架构。
> 【核心结论】Strong-Arm Latch通过四阶段（Reset→Sampling→Regeneration→Decision）工作；再生增益G_R = exp((t₂-t₁)/τ_R)；与CML Latch相比，Strong-Arm采样带宽更高但采样时间对电源敏感。
> 【工程价值】比较器再生时间常数τ_R决定判决速度；设计时需权衡输入对尺寸（影响gm和电容）以优化带宽vs增益权衡；NIKOLIC JSSC 2000的RS Latch优化可实现近2倍延迟改善。
> 【落地注意】Strong-Arm的采样时间随VDD变化是PDN噪声耦合的来源；高速SerDes通常需要独立的电源域来隔离比较器电源噪声；CML Latch虽然采样时间稳定但静态功耗较高。

> 🔍 深度说明：
> 【研究背景】Charge-Steering Latch是低电压应用的有力候选，仅需单时钟相位，采样孔径小；VLSI 2013 Chiang和ISSCC 2014 Bai的工作展示了其在小孔径和低延迟方面的优势。

## 五、灵敏度与偏置校正

### 5.1 灵敏度预算

```
RX灵敏度组成:
V_pp,s = 2 × v_n_rms × SNR + V_min + V_offset*

其中:
• 高斯(无界)输入参考噪声: 来自输入放大器、比较器、终端
• SNR: 给定BER所需信噪比
  - BER=10⁻¹² → SNR≈7
• V_min: 最小锁存分辨率电压
  - 通常 <5mV
  - 来源: 滞回、有限再生增益、有界噪声源
• V_offset: 输入偏移
  - 主要来源: 电路失配(Vth失配)
  - 如果未校正，可能是最大分量
```

### 5.2 前端噪声分析

```
噪声功率谱密度:
V²_n,out(f) = |H(f)|² · V²_n,FE(f)

其中:
• H(f): 前端传递函数
• V²_n,FE(f): 前端等效输入噪声谱密度

总噪声功率:
v²_n,out = ∫ |H(f)|² · V²_n,FE(f) df

注意:
• H(f)通常快速滚降
• 精确上限不关键
• 可设为很高(无穷)
```

### 5.3 比较器噪声

```
器件噪声导致:
• 即使输入为零也会有随机判决
• 噪声方差通过拟合输出高斯CDF获得

仿真方法:
• PSS+PAC+PNOISE
• 需要后处理从边带传递函数提取ISF
```

### 5.4 亚稳态

```
亚稳态公式:
t_metastability ~ τ · ln(V_in⁻¹)

其中:
• τ: 时间常数
• V_in: 输入过驱动电压

关键点:
• 比较器评估时间随输入减小而增长
• 当输入太小时，比较器没有足够时间完全评估
• 亚稳态窗口是比较器V_min的主要来源
```

![](lecture6_ee720_rx_circuits_images/img-043.jpg)

> 🔍 深度说明：
> 【研究背景】亚稳态是高速比较器设计中的关键问题，当输入过驱动电压过小时，比较器没有足够时间完成判决，导致输出不确定；这直接影响BER和系统余量。
> 【核心结论】亚稳态时间t_metastability ~ τ·ln(V_in⁻¹)，与再生时间常数τ和输入过驱动电压相关；亚稳态窗口是V_min的主要来源之一，需要通过设计优化来最小化。
> 【工程价值】在设计比较器时，需要预留足够的评估时间（通过调整时钟占空比或降低数据率）来避免亚稳态误码；BER要求越严，对亚稳态控制要求越高。
> 【落地注意】对于56Gb/s+ SerDes，时钟占空比控制成为关键，因为评估时间直接由时钟高电平宽度决定；需在时钟产生电路中实现精确duty cycle控制（<2%误差）。

### 5.5 偏置校正范围与精度

**统计设计原则**：
```
设计目标: 覆盖±3σ变化 (99.7%良率)

示例: 输入差分晶体管 W=4μm, L=150nm

失配参数:
σ_Vt = A_Vt/√(WL) = 2.8mV·μm/√(4μm×150nm) ≈ 3.6mV
σ_β = A_β/√(WL) = 2%/μm·√(4μm×150nm) ≈ 2.6%

假设(乐观): 输入偏移仅由输入对Vth失配主导
→ 需要设计±11mV的偏移校正范围

如果要在1mV内消除:
• 需要5-bit偏移消除精度
• 1 LSB = 22mV/31 ≈ 0.65mV
```

### 5.6 电流模式偏移校正

**JSSC 2008 (Balamurugan)**：
```
方法:
• 差分电流注入输入放大器负载
• 感应输入参考偏移来抵消固有放大器偏移

优势:
• 可扩展范围进行链路裕量测试
• 所有偏移设置下总偏移电流恒定
• 输出共模电平恒定

实现:
• 输入放大器和2路交错架构的各接收机分段
• 均进行偏移校正
```

### 5.7 电容偏移校正

```
方法:
• 感知放大器内部节点电容不平衡
• 感应输入参考偏移

机制:
• 内部节点预充电
• 增加更多积分时间 → 增加偏移范围

代价:
• 附加电容增加采样孔径时间

90nm CMOS, 输入W/L=4μ/0.15μ:
• 输入短路到共模电压
• 调整设置直到观察到均匀的"1"和"0"分布
• 偏移校正设置可能对输入共模敏感
```

![](lecture6_ee720_rx_circuits_images/img-048.jpg)

> 🔍 深度说明：
> 【研究背景】电路失配（尤其是输入对Vth失配）是RX偏移的主要来源，未经校正可能成为灵敏度预算中最大的分量；偏置校正需要覆盖±3σ工艺变化以保证99.7%良率。
> 【核心结论】W=4μm, L=150nm的输入对产生约3.6mV的σ_Vt，需要±11mV的校正范围；5-bit精度实现1mV内的偏移消除；JSSC 2008的电流注入方案实现恒定总偏移电流和输出共模。
> 【工程价值】偏置校正范围和精度设计直接影响良率成本；设计时需在校正范围（需更大范围以覆盖更多工艺角）和精度（需更高精度以减小残余偏移）间权衡。
> 【落地注意】电容偏移校正方案会增大采样孔径时间，需要在偏移校正精度和时序余量间做权衡；输入共模敏感度需要在设计时充分评估；2路交错架构需要分别校正各路偏移。

## 六、解复用

### 6.1 解复用优势

```
解复用好处:
• 时钟频率相对于数据率降低
• 给比较器更多再生和预充电时间
• 需要精确相位间隔
• 但不像TX多路复用那样对duty cycle敏感

时钟频率降低:
• 1:2 demux → 时钟频率 = 数据率/2
• 1:4 demux → 时钟频率 = 数据率/4
• 1:8 demux → 时钟频率 = 数据率/8
```

### 6.2 1:4解复用RX示例

```
权衡:
• 增加解复用因子 → 更高数据率
• 代价: 输入或预放负载电容增加
• 高多路复用因子 → 对相位偏移更敏感(以度为单位)
```

### 6.3 低压串行I/O收发器

**ISSCC 2013 (Song)**：
```
架构:
• TX输出多路复用 (4:1)
• RX输入多路复用 (1:8)
• 低电压操作

65nm CMOS原型测试:
• 20cm FR-4信道
• 4.8-8Gb/s数据率

能效优化:
• 最优0.47pJ/b能效在6.4Gb/s
• 低数据率: 静态电流摊销少
• 高数据率: 更高电压用于串行化时序
```

### 6.4 1:8输入解复用

```
特点:
• 1:8输入解复用允许输入比较器低电压操作
• Injection-locked振荡器用于高效多相位时钟生成和去偏斜

优势:
• 降低比较器速度压力
• 多相时钟分配
```

![](lecture6_ee720_rx_circuits_images/img-049.jpg)

> 🔍 深度说明：
> 【研究背景】RX解复用通过降低时钟频率来减轻比较器速度压力，同时给比较器更多再生和预充电时间；与TX多路复用不同，RX解复用对duty cycle不敏感但对相位间隔精度要求高。
> 【核心结论】ISSCC 2013 Song的1:8输入解复用配合Injection-locked振荡器实现高效多相位时钟生成和去偏斜；65nm实现4.8-8Gb/s，0.47pJ/b最优能效在6.4Gb/s。
> 【工程价值】对于56Gb/s SerDes，需要1:4或1:8解复用；解复用因子越高，输入比较器速度压力越低，但相位偏移敏感度以度为单位增加；Injection-locked时钟可降低分配复杂度。
> 【落地注意】高解复用因子增加输入电容负载，需要在比较器带宽和解复用因子间权衡；多相位时钟的相位精度直接决定采样时刻准确性；Injection-locked振荡器需要设计锁定范围和相位噪声性能。
