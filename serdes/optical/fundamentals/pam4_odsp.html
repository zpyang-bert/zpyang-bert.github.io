PAM4/oDSP 技术详解¶

创建日期: 2026/04/15 更新日期: 2026/04/15 分类: SerDes / 光通信 / 调制 标签: #serdes #pam4 #odsp #coherent #modulation

1. PAM4 调制技术¶

1.1 什么是 PAM4¶

PAM4 (Pulse Amplitude Modulation 4-Level) 是一种四电平脉冲幅度调制技术，每个符号携带 2 bits 信息。

NRZ vs PAM4 对比:

NRZ (1 bit/symbol):
┌───┐   ┌─┐       ┌───┐
│   │   │ │       │   │
│   │   │ │       │   │
└───┘   └─┘       └───┘
  0       1         0

PAM4 (2 bits/symbol):
        高
  ──────┼───── 11
        │     ← 两电平之间
  ──────┼───── 10
        │     ←
  ──────┼───── 01
        │     ←
  ──────┼───── 00
        低

符号速率相同情况下:
- NRZ: 1 bit/symbol → 25 Gbaud → 25 Gbps
- PAM4: 2 bits/symbol → 25 Gbaud → 50 Gbps

1.2 PAM4 优势与挑战¶

优势: | 方面 | 说明 | |------|------| | 频谱效率 | 2x 于 NRZ | | 带宽需求 | 相同速率下带宽减半 | | 成本效益 | 降低光模块成本 | | 应用广泛 | 200G/400G/800G Ethernet |

挑战: | 挑战 | 说明 | |------|------| | SNR 要求 | 比 NRZ 高约 9dB | | 线性度 | 需要线性驱动/接收 | | 眼图 | 三个眼图，更密集 | | 均衡 | 更复杂的均衡算法 | | CDRs | 需要更精确的时钟恢复 |

1.3 PAM4 信号特性¶

眼图参数:
眼宽: ~0.3 UI (典型)
眼高: ~100 mV (典型)
过渡时间: < 30 ps

与 NRZ 对比:
NRZ 眼高: 200 mV @ 25 Gbps
PAM4 眼高: ~100 mV (最上层/最下层)
           ~50 mV (中间眼)

2. oDSP (Optical DSP) 系统架构¶

2.1 相干光通信系统¶

完整相干收发机架构:

发送端:

│              ┌───────────────────────────┘                   │
│              ▼                                                │
│         [I/Q调制器] ◀── [激光器 (LO)]                         │
│              │                                                │
│              ▼                                                │

│              │                                                │
│              ▼                                                │

接收端:

│                        │                                     │

2.2 oDSP 处理流程¶

发送端 DSP:

1. Gray 编码
   └── 将二进制转换为 Gray 码 (降低误码传播)

2. 符号映射
   └── 映射到 QPSK/16QAM/64QAM 星座点

3. 上采样
   └── 2x 过采样 (便于滤波)

4. 滤波器 (RRC)
   └── 根升余弦滤波 (脉冲成形)

5. I/Q 生成
   └── 复数信号生成

6. 偏振复用
   └── X/Y 偏振信号合并

接收端 DSP:

1. ADC 采样
   └── 高速模数转换 (64+ GSa/s)

2. 数字下变频
   └── 复数基带信号

3. 补偿算法
   ├── 载波相位恢复
   ├── 偏振解复用
   ├── 色散补偿 (CD)
   └── 均衡 (CMA/LMS)

4. 判决与解码
   └── 硬判决/软判决

5. Gray 解码
   └── 还原二进制数据

3. 关键算法¶

3.1 色散补偿 (CD Compensation)¶

色散效应: - 纤维色散 (Chromatic Dispersion, CD) - 偏振模色散 (Polarization Mode Dispersion, PMD)

补偿方法:

时域色散滤波器:
y[n] = Σ h[k] * x[n-k]

频域色散补偿:
X(f) = FFT{x[n]}
Y(f) = X(f) * exp(j*β2*L*f²/2)
y[n] = IFFT{Y(f)}

其中:
β2: 群延迟色散参数
L: 光纤长度

3.2 偏振解复用¶

盲均衡算法:

CMA (Constant Modulus Algorithm):
误差: e = R² - |y|²
更新: w = w + μ * e * y * x*

MMA (Multi-Modulus Algorithm):
用于 QAM 信号
误差: e = (R_I² - |I|²) + j(R_Q² - |Q|²)

3.3 载波恢复¶

Viterbi-Viterbi 算法 (QPSK):

φ = (x⁴) / |x⁴|

用于 4th power 载波相位恢复
复杂度低效果好

相位搜索算法:

用于高阶 QAM (16QAM, 64QAM)
遍历可能的相位值
选择使误差最小的相位

3.4 前向纠错 (FEC)¶

软判决 LDPC:

输入: ADC 采样 (软信息)
     │
     ▼
 LDPC 解码器
     │
     ├── 迭代译码 (10-20 次)
     │
     ▼
输出: 硬判决数据 + 校验

性能提升: ~2dB vs 硬判决

4. PAM4 与相干技术对比¶

4.1 强度调制直接检测 (IM/DD)¶

系统架构:

PAM4 IM/DD:

TX: PAM4 信号 → DML/EML → 光纤
RX: 光电探测器 → TIA → ADC → DSP → 数据

适用场景: - 数据中心 (DCI) - 短距离 (< 10km) - 成本敏感应用

PAM4 IM/DD 优势: - 简单架构 - 低成本 - 低功耗

4.2 相干检测¶

系统架构:

相干 PAM4:

TX: PAM4 → I/Q 调制 → 激光器 → 光纤
RX: LO 混频 → 平衡探测器 → ADC → oDSP

适用场景: - 长距离 (> 80km) - 高性能要求 - 城域/骨干网

相干优势: - 高灵敏度 - 色散容忍 - 偏振无关

4.3 对比表¶

参数 PAM4 IM/DD 相干 PAM4
距离 < 10km 10km - 1000km+
复杂度 低 高
功耗 低 高 (oDSP)
成本 低 高
灵敏度 一般 优秀
色散容忍 低 高
速率 100G/400G 400G/800G+

5. LPO (Linear Pluggable Optics)¶

5.1 什么是 LPO¶

LPO (Linear Pluggable Optics) 是一种低功耗可插拔光模块，特点： - 去掉 DSP 芯片 - 使用线性驱动/接收 - 与交换机共封装光学 (CPO) 互补

5.2 LPO vs 传统可插拔¶

传统可插拔 (Traditional Pluggable):
┌─────────────┐
│  DSP       │  ← 大功耗
│  CDR       │
│  Driver    │
│  Laser     │
│  TIA       │
└─────────────┘

LPO:
┌─────────────┐
│  Driver    │  ← 低功耗
│  Laser     │
└─────────────┘
     ↑
   Linear (无 DSP)

5.3 LPO 优缺点¶

优点: - 功耗降低 50%+ - 延迟降低 (无 DSP 处理) - 成本降低

缺点: - 距离限制 (需线性信道) - 交换机端需要 DSP - 互通性挑战

5.4 应用场景¶

  • 数据中心交换机互联
  • AI/ML GPU 互联
  • 超大规模数据中心

6. 高速 SerDes 与 PAM4¶

6.1 112G SerDes PAM4¶

关键参数: - 速率: 112 Gbps/lane - 调制: PAM4 - 波特率: 56 Gbaud - 应用: 800G/1.6T 光模块

架构:

TX:
并行数据 → Gray编码 → PAM4映射 → FFE → 驱动器 → 输出

RX:
输入 → CTLE → AGC → ADC → DFE → CDR → Gray解码 → 并行数据

6.2 224G SerDes (下一代)¶

路线图: - 2025-2026: 224G/lane - 调制: PAM4 (或更高阶) - 目标: 1.6T/3.2T 光模块

技术挑战: - 信道损耗: ~40dB - 封装密度 - 功耗管理 - 散热

7. 测试与验证¶

7.1 PAM4 眼图测试¶

测试设备: - 高速示波器 (> 50 GHz) - BERT (误码仪) - 相干检测仪

关键指标: - 眼图眼宽/眼高 - 过渡时间 - 抖动 (TJ/BJ/DJ) - 误码率 (BER)

7.2 oDSP 测试¶

测试项目: - EVM (Error Vector Magnitude) - 星座图 - 载波泄漏 - 相位噪声

EVM 计算:

EVM = sqrt(Σ|d_ref - d_act|² / Σ|d_ref|²)

典型要求:
- QPSK: < 15%
- 16QAM: < 12%
- 64QAM: < 8%
