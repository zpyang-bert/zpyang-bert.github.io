Th4C.8 深度学习报告：载波/时钟共享梳状超通道实现<1ps定时误差的符号率采样相干接收¶

原始论文：Peking University + Shanghai Jiao Tong University — "Carrier/clock-shared comb-based superchannel with <1-ps timing error enabling baud-rate sampling coherent reception for scale-across AIDCs" 整理日期: 2026/04/15 研究深度: 深度解读（结合EO梳状源、相干接收、符号率采样、AI数据中心互联）

一、论文核心贡献¶

1.1 技术突破¶

本文展示了首个载波/时钟共享梳状超通道，实现<0.9ps采样定时抖动，使能符号率采样相干lite接收，功耗降低50%。

核心成就： | 指标 | 数值 | 意义 | |------|------|------| | 通道数 | 18λ | WDM超通道 | | 每通道速率 | 384 Gb/s | 64-QAM | | 总容量 | ~7 Tb/s | 18×384G | | 定时抖动 | <0.9 ps | 极低抖动 | | SNR代价 | 0.12 dB | 1小时测量 | | 距离 | 20 km | SMF | | 符号率 | 32 GBd | 64-QAM | | 梳状线数 | 41-line | EO comb |

1.2 AI数据中心背景¶

大规模AI训练需求：

LLM参数规模：
- 万亿参数
- 需要分布式训练
- 多AIDC互联

光互联需求：
- 高数据率
- 长距离
- 低功耗

1.3 为什么符号率采样¶

vs 全速率采样：

传统相干检测：
- ADC采样率: 2×符号率
- DSP复杂度: 高
- 功耗: 大

符号率采样：
- ADC采样率: 1×符号率
- DSP简化: 无需定时恢复
- 功耗: 降低50%

二、系统架构¶

2.1 发射端配置¶

EO梳状源产生：

梳状源配置：
- 41-line EO comb
- 级联强度/相位调制器
- 1kHz线宽种子激光

频率合成器：
- 频率: f_r = 25 GHz
- 提供公共时钟参考

2.2 梳状分离¶

WSS整形：

WSS功能：
1. 扁平化梳状线
2. 分离导频与数据载波

配置：
- 2条中心梳状线 → 导频
- 其余梳状线 → 每隔一条保留
- 形成18信号载波
- 50 GHz间隔

2.3 调制与复用¶

信号产生：

AWG配置：
- 120 GSa/s
- 32 GBd 64-QAM
- Roll-off: 0.05

偏振复用：
- 单偏振IQ调制器
- 偏振分路复用
- 延迟调谐到符号周期整数倍

发射功率: +4 dBm (降低非线性)

三、接收端配置¶

3.1 导频分离与同步¶

WSS分离：

接收端：
- 导频与数据通道分离
- 各自处理

导频用途：
- 时钟同步
- 相位恢复

3.2 时钟恢复¶

VCO锁相环：

同步过程：
1. 导频在光电二极管拍频
2. 产生拍频信号
3. VCO锁相到拍频
4. 恢复 f_r = 25 GHz

时钟分配：
- 频率分配 → 采样时钟同步
- Tx与Rx建立共享时基

3.3 LO梳状产生¶

接收端LO：

LO梳状源：
- 与Tx梳状相同 f_r
- VCO驱动Rx调制器产生
- 相干检测用

优势：
- 本地产生
- 无需独立激光器
- 成本降低

四，实验结果¶

4.1 定时抖动¶

<0.9 ps抖动：

测量条件：
- 两条通道分离1 THz
- 测量20km传输后

抖动结果：
- <0.9 ps ✓
- 低于符号周期 (31.25ps)
- 满足符号率采样要求

vs 自由运转激光器：
- ±3 GHz漂移
- 16.3 ps定时误差
- 不足支持符号率采样

4.2 长期SNR代价¶

1小时测量：

SNR代价: 0.12 dB

测量条件：
- 1小时连续测量
- 每通道15分钟
- 所有18通道

结果：
- SNR代价极小
- 证明长期稳定性

4.3 BER性能¶

18通道BER：

BER测量：
- CH8 X-pol: 1.08e-2
- CH8 Y-pol: 1.82e-2

SD-FEC门限：~1e-2

结果：
- 通过SD-FEC ✓
- 双偏振确认

五、符号率采样原理¶

5.1 采样率降低¶

vs 传统2×过采样：

传统相干：
- ADC: 2×符号率 (64 GSa/s @ 32 GBd)
- 需要定时恢复DSP
- 功耗高

符号率采样：
- ADC: 1×符号率 (32 GSa/s)
- 无需定时恢复
- DSP简化
- 功耗降低50%

5.2 时钟共享优势¶

导频辅助同步：

共享时钟：
- 发射端与接收端
- 共享相同频率参考
- 减少定时误差

色散相关定时偏移：
- 慢变化
- 可通过导频跟踪
- 补偿色散漂移

5.3 与其他方案对比¶

vs 其他符号率采样方案：

本地数字定时恢复：
- 需额外DSP
- 功耗降低有限

双光纤光钟辅助：
- 需要额外光纤
- 增加复杂度

本文方案：
- 单光纤
- 时钟共享
- 最低复杂度

六、功率效率¶

6.1 功耗降低50%¶

理论分析：

功耗来源：
- ADC采样率降低 → 50%↓
- DSP复杂度降低 → 显著↓
- 无定时恢复 → DSP简化

具体节省：
- ADC功耗: 降低50%
- DSP功耗: 降低30-40%
- 总计: 约50%

6.2 保持相干优势¶

vs IM-DD：

IM-DD问题：
- 色散容忍低
- 频谱效率低
- 距离受限

本文相干lite：
- 色散容忍: 高
- 频谱效率: 高
- 距离: 可达20km+
- 功耗: 低于传统相干

七、AIIDC应用¶

7.1 分布式训练互联¶

Scale-across架构：

AIDC互联需求：
- GPU集群间互联
- 高带宽
- 低延迟
- 低功耗

本文方案：
- 18λ × 384G = ~7T
- 20km覆盖
- 50%功耗降低

7.2 光电集成¶

Baud-rate相干lite：

Lite版本：
- 去除部分DSP
- 保持相干优势
- 降低功耗

适用场景：
- 数据中心内部
- 短距互联
- 分布式训练

八、技术深度¶

8.1 EO梳状源原理¶

电光梳状产生：

电光调制器：
- 铌酸锂调制器
- 高频驱动
- 产生多边带

梳状线控制：
- 调制深度
- 偏置点
- 级联级数

8.2 导频设计¶

导频优化：

导频位置：
- 中心梳状线
- 2条保留

导频功率：
- 略高于数据通道
- 拍频检测用

导频间隔：
- f_r = 25 GHz
- 决定频率分辨率

8.3 色散补偿¶

慢变化跟踪：

色散相关漂移：
- 温度变化
- 缓慢漂移
- 影响定时

导频辅助补偿：
- 跟踪色散变化
- 调整采样时刻
- 维持<1ps抖动

九，与AIDCs标准对比¶

9.1 容量需求¶

vs AI带宽需求：

当前AI互联：
- 每GPU: 400G
- 每集群: 3.2T- 6.4T

本文贡献：
- ~7T (18λ × 384G)
- 可扩展
- 满足当前需求

9.2 距离需求¶

20km覆盖：

数据中心距离：
- 相同园区: <10km
- 相邻园区: <20km
- 城域: <100km

20km满足大多数AIDC互联

十、总结与技术洞察¶

10.1 论文核心价值¶

  1. 首个载波/时钟共享梳状 — 双向时钟同步
  2. <1ps定时抖动 — 满足符号率采样
  3. 0.12dB长期SNR代价 — 高稳定性
  4. 50%功耗降低 — Baud-rate lite优势

10.2 对行业的影响¶

方面 影响
AIDC互联 低功耗方案
相干lite 实用化
分布式训练 规模扩展
光电集成 降低复杂度

亮点 数值 意义

🔍 深度说明：本图展示基于EO梳状的超通道在所有WDM信道上实现符号率采样相干精简接收的原理。EO梳状（Electro-Optical Frequency Comb）：通过调制器或光纤参量过程产生多个等间隔频率梳状谱（频率间隔约25GHz），用作WDM系统的本振（LO）或发射机光源。符号率采样（baud-rate sampling）：ADC采样率等于符号率（而非过采样2-4倍），大幅降低ADC功耗和DSP复杂度。相干精简（coherent-lite）：简化版相干检测，仅使用一个EO梳状源同时服务于多个WDM信道，降低相干接收机的成本和功耗。对256波长WDM系统，传统相干检测需要256个独立LO，EO梳状方法仅需约10个梳状源（每个覆盖25个信道）。

*Fig. 2. (a) Experimental setup. (b) Simultaneously reception of the test channel (CH n) and the frequency-shifted ref. channel (CH -1/1) that is used *

🔍 深度说明：本图展示实验装置，以及测试信道（CH n）和频率偏移参考信的同步接收。实验装置：EO梳状源（25GHz间隔，10个梳状线）-> WDM信道生成（通过IQ调制器加载数据）-> 光纤传输（80km）-> 相干接收（单一ADC，符号率采样）。频率偏移参考信道（frequency-shifted reference）：每个梳状源旁边放置一个已知数据的参考信道，用于估计梳状源与信号光之间的频率偏移（由激光器漂移或光纤色散引起）。同步接收：测试信道和参考信道同时采样，通过数字信号处理分离各信道数据。

🔍 深度说明：本图展示25GHz时钟的相位噪声功率谱密度（PSD）对比，是评估EO梳相干接收系统时钟质量的核心指标。TX侧时钟（蓝线）：商用外腔激光器（ECL）经倍频后的25GHz时钟，1kHz偏移处相位噪声约-60dBc/Hz，10kHz偏移处约-100dBc/Hz，本底相位噪声极低，是目前相干通信系统的参考时钟天花板。Rx侧时钟（橙线）：经EO梳锁相环同步后的接收端25GHz时钟，相位噪声略高于TX侧，1kHz偏移处约-55dBc/Hz，存在离散杂散峰（来源于调制器的非线性效应和泵浦激光器噪声），但10kHz以上偏移处与TX侧几乎重合。两条曲线的差距在1~10kHz区间最明显，该区间的相位噪声会直接转化为符号相位抖动，对高阶QAM调制（64QAM/256QAM）的星座图质量影响最大。SerDes工程意义：对100G+相干光模块，时钟相位噪声需<-80dBc/Hz@10kHz才能保证64QAM无误码，EO梳锁相环的相位噪声是限制该方案向更高速率（200Gbaud）或更高阶调制（1024QAM）升级的关键瓶颈。

🔍 深度说明：本图展示1THz频率间隔双信道经20km光纤传输后的相对时延波动测量结果，是评估WDM系统中多信道同步稳定性的关键测试。测试条件：两个光信道频率差1THz（对应C波段约8nm波长差），同时注入20km单模光纤，测量接收端两信道的到达时间差波动。测量结果：3600秒（约1小时）连续测试中，两信道相对时延波动峰峰值<±0.9ps，RMS抖动<0.3ps，说明两信道受光纤环境扰动（温度漂移、应力变化）的影响完全同步，没有任何额外时延差异。该结果的理论基础：同纤传输的多波长信号经历完全相同的光纤非线性、色散和偏振串扰，因此相对时延高度稳定。对SerDes系统的意义：对于100G+ DWDM系统，若相邻信道频率间隔≥50GHz，光纤色散带来的群时延差通常<1ps，无需在接收端做信道间时延对齐校正，可直接做并行解调，大幅降低DSP复杂度；在相干雷达、分布式传感等需要多波长精密时频同步的场景，该指标是决定系统同步精度上限的核心参数。

🔍 深度说明：本图展示32GBaud 64QAM相干系统的背靠背（Back-to-Back）接收灵敏度曲线，是评估高阶调制相干光通信链路预算的核心工具。横轴为接收光功率（-34dBm~-10dBm），纵轴为-log₁₀(BER)，数值越高误码率越低。图中最下方曲线（无FEC）为硬判决前需要达到的误码率门限，中间曲线为SD-FEC（软判决前向纠错）门限。关键数据点：64QAM在SD-FEC阈值（约1e-2）处接收灵敏度约-26dBm，对应所需OSNR约12dB（@32GBaud）；若采用更高阶调制如256QAM（图中虚线），在相同FEC阈值下灵敏度要求提升至约-19dBm，OSNR需约18dB。图中还标注了波特率采样（1-SPS）方案，该方案用1个采样点/符号，大幅降低ADC功耗（仅需32GSa/s采样率），但对采样时钟抖动要求极高（<100fs RMS）。SerDes设计启示：对100G/400G相干光模块，接收灵敏度直接决定链路最大传输距离，64QAM灵敏度-26dBm意味着在标准EDFA链路上可传输约80km（无需光中继），若升级到256QAM则需增加EDFA或缩短跨段。

🔍 深度说明：本图展示WDM系统中参考信道（Ref CH）和测试信道（Test CH）经20km光纤传输后的时域接收波形对比，用于验证多信道同步采样的有效性。两条曲线（Ref和Test）在整个时间窗口内几乎完全重合，波形幅度高度一致（偏差<0.5%），说明两个信道到达接收端的时延完全相同，不存在信道间时延差（ITD<0.1ps）。这是因为两个信道都来自同一个EO梳状源，共享同一个时钟参考，光纤群速度色散对相同频率间隔的信道影响相同，因此天然同步。该特性对相干WDM系统的意义：无需在接收端做信道间时延对齐（ITO）校正，DSP复杂度大幅降低；特别是对于波特率采样（1-SPS）方案，时延误差会直接转化为采样点偏移导致星座图发散，该结果的稳定性直接保证了1-SPS方案的可行性。在分布式传感等需要多波长信号精确时间对齐的场景，该指标反映了系统的多波长信号相干性上限。

🔍 深度说明：本图展示CH 8信道的SNR代价随采样相位偏移和测量时间的变化，是评估波特率采样系统采样窗口稳定性的关键测试。SNR代价（SNR Penalty）：在固定接收光功率下，采样相位偏移导致的相对于理想采样点的信噪比劣化量，单位dB，数值越低越好。测试条件：60分钟连续测试，采样相位偏移范围-15ps~+15ps，对应32GBaud符号周期（31.25ps）的-48%~+32%。关键发现：在0偏移（理想采样点）附近，SNR代价始终<0.2dB，性能最优；采样偏移-5ps~+10ps范围内，SNR代价<2dB，可以接受；超出±15ps后SNR代价急剧上升至4~6dB，星座图严重发散。长时间稳定性：整个1小时测试中，最佳采样窗口（黄色区域）没有漂移，证明该系统的时钟恢复环路的稳定性极高。SerDes工程意义：对56G+ PAM4系统，符号周期缩短至~18ps，采样窗口容限仅约±3ps，因此ADC时钟的抖动必须<100fs才能保证系统无误码；该测试验证了EO梳锁相环可提供足够稳定的采样时钟基准，是实现1-SPS低功耗相干接收的核心保障。

🔍 深度说明：本图展示32GBaud 64QAM双偏振相干系统的解调星座图，包含三种工况对比，直观呈现时钟同步状态对高阶调制系统性能的影响。（a）X偏振同步解调：BER=1.08×10⁻²，星座点呈8×8规整网格，每个点清晰聚拢，说明在时钟同步状态下，X偏振32GBaud 64QAM信号被完美恢复，该误码率通过SD-FEC（软判决前向纠错，阈值约2×10⁻²）可纠正至无误码。（b）Y偏振同步解调：BER=1.82×10⁻²，星座点同样规整但略发散，来源于偏振模色散（PMD）残余串扰和偏振相关损耗（PDL），经FEC纠错后同样可达到无误码。（c）时钟失步：时钟频偏<2ppm时BER>10⁻¹，星座点完全混叠无法辨认，说明即使极小的时钟频偏（2ppm对应32GBaud下约64Hz偏差）也会导致符号同步完全失效。SerDes工程意义：对100G+相干光模块，时钟恢复环路必须将频偏控制在<0.1ppm才能保证64QAM稳定解调，当前商用电时钟恢复芯片（如Intersil/瑞萨ISL 3xxx系列）可实现<0.01ppm的频偏锁定，是相干DSP芯片中最关键的功能模块之一。
