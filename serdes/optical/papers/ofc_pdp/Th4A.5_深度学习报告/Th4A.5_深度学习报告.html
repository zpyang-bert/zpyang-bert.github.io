Th4A.5 深度学习报告：GaAs-on-TFLN异构集成1064nm全集成发射机¶

原始论文：Nexus Photonics + Northeastern University — "Fully Integrated 1064 nm Transmitters with Widely Tunable GaAs Lasers and >100-GHz Thin-Film LiNbO₃ Modulators" 整理日期: 2026/04/15 研究深度: 深度解读（结合异构集成、激光物理、空心光纤通信）

一、论文核心贡献¶

1.1 技术突破¶

本文展示了世界首款GaAs-on-TFLN异构集成平台，在1064nm波段实现了全集成光发射机。

核心成就： | 指标 | 数值 | 意义 | |------|------|------| | 波长 | 1064nm | 空心光纤应用 | | 调制带宽 | >100GHz | TFLN class | | NRZ速率 | 100 Gb/s | 直接调制 | | PAM4速率 | 160 Gb/s | 高阶调制 | | SMSR | >50 dB | 优异边模抑制 | | 功率 | >10 mW | 芯片输出 | | 调谐范围 | >20 nm | Vernier环实现 |

1.2 1064nm波段的战略价值¶

为什么选择1064nm？

传统波段 特点 1064nm优势
850nm VCSEL主导 无源光器件成熟
1310nm 零色散 低延迟空心光纤
1550nm 最小衰减 铒镱共掺放大

空心光纤 (HCF) 驱动：

HCF优势：
- 延迟降低 ~30%（以空气为主）
- 非线性效应极低
- 波长选择更灵活

HCF损耗特性：
- 660nm: 低损耗窗口
- 850nm: 低损耗窗口
- 1064nm: 低损耗窗口 ✓

1064nm + HCF = 极低延迟互联

二、异构集成平台¶

2.1 GaAs-on-TFLN架构¶

为什么异构集成？

需求 GaAs TFLN 异构
高效光源 ✓ ✗ ✓
低Vπ调制 ✗ ✓ ✓
高速响应 ✓ ✓ ✓
紧凑集成 一般 好 ✓

单片集成价值：

传统方案：
- 激光器 (GaAs chip)
- 调制器 (LiNbO₃ chip)
- 放大器 (GaAs chip)
- 尺寸大，耦合损耗高

本方案：
- 全部单片集成在TFLN上
- 芯片级互联
- 极低耦合损耗

2.2 晶圆级异构集成工艺¶

工艺流程：

         ↓

         ↓
3. GaAs外延层键合
         ↓
4. GaAs衬底去除
         ↓
5. GaAs器件加工
         ↓
6. 全wafer完成

键合技术：

Die-to-wafer键合：
- 表面活化键合
- 或者 adhesive bonding
- 对准精度: <1 μm

三、GaAs激光器设计¶

3.1 DFB激光器性能¶

1064nm GaAs DFB： | 参数 | 数值 | 说明 | |------|------|------| | 波长 | 1064nm | Yb光纤增益峰 | | SMSR | >50dB | 单模工作 | | 阈值电流 | 27mA | 适中 | | 输出功率 | >10mW | 耦合进TFLN |

DFB光栅设计：

光栅周期：
Λ = λ/(2×n_eff)
  = 1064nm/(2×3.4)
  ≈ 156nm

 grating耦合系数：
κ ≈ 50-100 cm⁻¹ (典型)

3.2 Vernier可调谐激光器¶

双环Vernier原理：

两个环：
- 主环 (Main Ring): FSR₁
- 从环 (Vernier Ring): FSR₂

FSR差 = FSR₁ - FSR₂

有效FSR增大：
FSR_eff = FSR₁ × FSR₂ / |FSR₁ - FSR₂|

实现 >20nm 调谐范围

连续可调谐：

调谐机制：
- 热光效应
- 电流注入

温度调谐系数：
dn/dT ≈ 2×10⁻⁴/K

环周长 ~ 1mm
Δλ/ΔT ≈ 0.1nm/K

20nm调谐需要 ~200K温度变化
→ 多个热调谐电极

3.3 半导体光放大器 (SOA)¶

SOA功能：

1. 输出功率放大
   - 小信号增益: 15-20 dB
   - 饱和输出: >15 dBm

2. 门控功能
   - 快速开关
   - 功率控制

3. 集成优势
   - 与DFB工艺兼容
   - 共享外延结构

4.3 高频性能¶

S参数测量： | 参数 | 数值 | 说明 | |------|------|------| | S₁₁ | < -10dB @ 100GHz | 阻抗匹配 | | S₂₁ (EO) | > -3dB @ 100GHz | 3dB带宽>110GHz |

带宽受限因素：

1. 电极损耗
   - 趋肤效应
   - 介质损耗

2. 速度失配
   - 光微同速
   - 群速度差

3. 模式匹配
   - 微波/光波模式

五、传输性能¶

5.1 NRZ vs PAM4对比¶

100Gb/s NRZ：

符号率: 100 GBd
带宽需求: 50 GHz
眼图SNR: 14.61 dB ✓

NRZ优势：
- 消光比高
- SNR要求低
- 驱动简单

160Gb/s PAM4：

符号率: 80 GBd
带宽需求: 40 GHz
眼图TDECQ: 0.04 dB ✓

PAM4优势：
- 频谱效率2×
- 带宽利用率高
- 仍可均衡

5.2 测试限制¶

当前系统瓶颈：

测试链路衰减 ~20dB @ 50GHz

包括：
- 任意波形发生器
- 功率放大器
- 接收端光电转换

TFLN调制器能力: >100 GHz
→ 实际可支持更高符号率

改进方向：
- 更高带宽测试设备
- 低损耗高频电缆
- 更高功率放大器

六、1μm波段生态¶

6.1 与Yb光纤放大器的协同¶

Yb-doped光纤放大器：

增益带宽: 1030-1100nm
峰值增益: 1040nm
小信号增益: >30dB

1064nm位于增益带宽内 ✓

实现：
- 全波段放大
- 高功率输出
- 低噪声系数

6.2 空心光纤传输¶

Nested Antiresonant HCF：

结构：
- 悬浮内核
- 薄壁管支撑
- 空气传输

性能对比：

损耗:
- Solid SMF @ 1310nm: 0.3 dB/km
- HCF @ 1064nm: <1 dB/km (目标)

延迟:
- Solid SMF: 5 μs/km
- HCF: ~3.5 μs/km (空气为主)

非线性:
- HCF: 极低 (空气~无Kerr)

6.3 应用场景¶

低延迟关键应用：

1. AI/ML互联
   - GPU/CPU集群
   - 延迟敏感
   - HCF优势明显

2. 金融交易
   - 微秒级延迟
   - 竞争激烈

3. 雷达/相干束成形
   - 低延迟处理
   - 分布式系统

4. 空间通信
   - 1μm波段
   - 大气传输窗口

七、异构集成深度技术¶

7.1 界面损耗控制¶

GaAs-TFLN耦合：

耦合机制：
- 倏逝波耦合
- 垂直方向

耦合效率关键：
- 界面质量
- 模式重叠
- 折射率匹配

GaAs: n ≈ 3.4
TFLN: n ≈ 2.2

→ 需要过渡波导

🔍 深度说明：本图展示TFLN马赫-曾德调制器（MZM）截面结构示意图。核心层：180nm厚铌酸锂薄膜（X-cut，TFLN工艺，r33≈30pm/V）位于900nm厚SiO₂上 cladding（低折射率隔离层，确保光场局限在LN层），底部Si基底；侧壁角度62°（由ICP干法刻蚀形成，保证波导侧壁粗糙度<10nm以降低传播损耗）；顶部Au电极（信号电极与接地电极之间间隙~3μm，形成微波共面波导结构）。电光调制原理：RF信号沿电极传播，光在LN波导中以~0.3c的速度行进，通过r33电光系数改变LN折射率（Δn = -½·r33·n³·E），实现相位调制；电极长度1~2cm，配合行波结构实现>100GHz电光调制带宽。工程价值：TFLN波导传播损耗~2dB/cm（O波段），远低于Si波导在O波段的~4dB/cm；Vπ≈3.1V（L≈1cm），比Si MZM（Vπ≈8V）低约2.6倍，直接降低驱动放大器功耗（驱动IC功耗∝V²pp·R⁻¹）；62°侧壁角设计在保证模场对称性的同时最大化电极耦合效率。

🔍 深度说明：本图展示GaAs-on-TFLN异构激光器射频调制端口的输入反射系数（S₁₁）随频率变化曲线，测试范围0~110GHz。S₁₁代表射频调制端口的阻抗匹配质量，数值越负（dB）代表反射功率越少、端口阻抗越接近50Ω系统匹配。（a）0~20GHz低频段：S₁₁在-20~-45dB之间，最低谷值-45dB，反射功率仅0.003%，匹配性能优异；（b）20~50GHz中频段：S₁₁抬升至-15~-35dB区间，峰值约-15dB（反射~3%），仍满足应用要求；（c）50~110GHz高频段：波动幅度增大，峰值约-12dB（反射~6%），但大部分频点仍低于-20dB。核心结论：该调制端口在0~110GHz超宽频段内整体满足|S₁₁|≤-10dB的阻抗匹配要求，可支持>100GBd PAM4高速调制。工程价值：射频端口阻抗匹配直接影响驱动信号反射和眼图质量，|S₁₁|<-10dB可确保驱动功率高效注入而不过热损伤器件；设计时需重点关注电极走线宽度、接地过孔分布和封装引线电感（典型值0.1~0.5nH/mm），这些因素在>50GHz频段会成为主要非理想因素。

🔍 深度说明：本图展示TFLN调制器电光幅频响应（EO S₂₁）测试结果，0~110GHz范围。（a）VNA实测（灰色散点）与OSA边带法实测（红色散点）高度吻合，互相验证了测试准确性；（b）拟合曲线（深色实线）与仿真曲线（浅色实线）在全频段内重合度极高，说明器件模型精准；（c）-3dB参考线明确标注，响应在~110GHz位置才触达3dB截止点。核心结论：该TFLN调制器3dB电光带宽>100GHz，可支持单通道>200GBd PAM4或>100GBd QAM调制，满足下一代1.6Tb/s/λ相干/强度调制需求。工程价值：EO S₂₁在0~50GHz内维持在0~-2dB平坦响应，意味着该频段内调制效率几乎不劣化，对QAM等高阶调制格式的误码率性能极为关键；行波电极设计（velocity matching）和阻抗匹配（50Ω）是实现>100GHz带宽的核心，需要在电极几何优化（宽度/间隙）和封装设计上重点投入。
