Th4B.5 深度学习报告：42.4 THz OESCL波段450 Tb/s场部署光纤传输¶

原始论文：NICT + UCL + Aston University + Nokia Bell Labs — "450 Tb/s GMI, 42.4 THz, OESCL-Band Transmission Over a Field-Deployed Fiber" 整理日期: 2026/04/15 研究深度: 深度解读（结合超宽带传输、WDM放大、非线性效应、多波段复用）

一、论文核心贡献¶

1.1 技术突破¶

本文展示了业界最宽频带SMF传输，跨越42.4 THz (1273个O/E/S/C/L波段通道)，在39km场部署光纤上实现>450 Tb/s吞吐量。

核心成就： | 指标 | 数值 | 意义 | |------|------|------| | 总带宽 | 42.4 THz | 业界最宽 | | 通道总数 | 1273个 | 大规模WDM | | 吞吐量(GMI) | >450 Tb/s | 记录 | | 吞吐量(解码) | 418 Tb/s | 实用速率 | | 光纤类型 | G.652-D | 场部署SMF | | 传输距离 | 39 km | 城域范围 | | 调制格式 | 4/16/64/256-QAM | 自适应 |

1.2 为什么超宽带¶

扩展频谱的价值：

传统WDM限制：
- C-band: ~4 THz (96通道)
- C+L-band: ~8 THz

流量增长需求：
- 每2年翻倍
- 现有光纤耗尽
- 新光纤成本极高

解决方案：超宽带 (UWB)
- O+E+S+C+L 五波段
- 利用石英光纤全部低损耗窗口

二、波段划分与光谱¶

2.1 五个波段¶

OESCL波段定义：

波段      波长范围    频率范围      带宽
─────────────────────────────────────
O-band   1260-1360nm  220-237 THz   17 THz
E-band   1360-1460nm  205-220 THz   15 THz
S-band   1460-1530nm  196-205 THz   9 THz
C-band   1530-1565nm  191-196 THz   5 THz
L-band   1565-1625nm  185-191 THz   6 THz
─────────────────────────────────────
总计:                   42.4 THz

低损耗窗口：

石英光纤损耗谱：
- 1310nm窗口 (O): 0.32 dB/km
- 1550nm窗口 (C): 0.2 dB/km
- E-band: ~0.4 dB/km (有水峰)

研究利用全部低损耗区域

2.2 通道规划¶

1273个通道分布：

等效通道计算 (以32.5 GBd为基础)：

O-band: ~400+ 通道
E-band: ~350+ 通道
S-band: ~200+ 通道
C-band: ~100+ 通道
L-band: ~150+ 通道

总带宽: 42.43 THz

三、放大方案¶

3.1 多波段放大¶

分段放大架构：

O-band: 量子点DFA (QDFA) 或 Bi-DFA
E-band: 铋DFA (BDFA)
S-band: 铥DFA (TDFA)
C-band: 掺铒EDFA
L-band: 掺铒EDFA (扩展L-band)

每段放大针对各自波段优化

各放大器特性： | 波段 | 放大器 | 增益 | 噪声 | |------|--------|------|------| | O | QDFA | >20 dB | ~5 dB | | E | BDFA | >20 dB | ~5 dB | | S | TDFA | >20 dB | ~5 dB | | C | EDFA | >30 dB | ~4 dB | | L | EDFA | >25 dB | ~5 dB |

3.2 光谱成型¶

Dummy Band生成：

ASE噪声 → 光谱成型 → Dummy Band

方法：
- 各波段ASE噪声源
- 光谱处理器(OP)成型
- 产生连续Dummy Band
- 100GHz滑动缺口(Sliding notch)

与测试波段组合：
Dummy Band @ 10 dB coupler

四、实验配置¶

4.1 发送端¶

信号产生：

调制器:
- O/E-band: 定制
- S/C/L-band: 通用

AWG驱动:
- 92 GSa/s 任意波形发生器
- 32.5 GBd 符号率
- QAM调制: 4/16/64/256

通道配置:
- 中心通道: 独立调制
- 邻居通道: 联合调制
- 光去相关后组合

4.2 光纤链路¶

39km G.652-D场部署光纤：

光纤: ITU-T G.652-D
长度: 39 km
配置: 成对 (发送+接收)

ITU-T G.652-D特性:
- 零色散波长: ~1310nm
- 损耗: 0.32 dB/km @ 1310nm
- PMD: <0.2 ps/√km

链路损耗：

@ 1310nm OTDR: ~13.6 dB
@ 1550nm OTDR: ~8-9 dB

包含:
- 光纤损耗
- 连接器损耗
- 分路器损耗

4.3 接收端¶

多接收机配置：

O/E-band:
- 相干接收
- 33 GHz 带宽
- 80 GSa/s DSO

S/C/L-band:
- 相干接收
- 110 GHz 带宽
- 256 GSa/s DSO

五、非线性效应¶

5.1 SRS (受激拉曼散射)¶

功率转移机制：

SRS特性：
- 频移: ~13.2 THz (silica)
- 长波增益，短波损耗
- 影响: 功率重分配

测量结果：
- 最大功率23 dBm时
- O/E/S-band: 损耗增加 <1.5 dB
- C/L-band: 增益增加 <1.5 dB

无额外谱预加重

5.2 FWM (四波混频)¶

FWM损害分析：

FWM条件：
- 相位匹配在零色散波长最好
- 零色散波长: 1312.4 nm

测试方法：
- O-band dummy信号
- 100 GHz缺口
- 测量FWM地板

结果：
- 零色散附近惩罚最高
- O-band功率 >16.5 dBm时 >1 dB

选择O-band功率: 17.3 dBm

5.3 预加重策略¶

功率分配：

根据以下因素调整：
1. 放大器增益曲线
2. 光纤损耗
3. SRS效应
4. FWM限制
5. 激光安全 (23 dBm限制)

目标：均衡通道OSNR

六、DSP处理¶

6.1 MIMO均衡¶

4×4 实值时域MIMO：

结构：
- 33 半符号持续时间抽头
- 4×4 MIMO
- 实值算法 (时间复用)

收敛方法：
1. 数据辅助LMS初始化
2. 判决导向(DD)切换
3. 载波恢复在MIMO环内

6.2 FEC编码¶

DVB-S2 FEC：

编码方案：
- LDPC码 (DVB-S2标准)
- 打孔增加码率粒度

目标：
- BER < 10⁻⁵ 停止
- 1%开销硬判决码去除残余错误

码率范围：
- 从1/4 到 9/10
- 自适应选择

6.3 GMI评估¶

广义互信息：

GMI vs BER：
- GMI直接衡量信息率
- 考虑调制和信道编码
- 接近香农容量

数据率估计：
Rate = GMI × 符号率 × 通道数

七、性能结果¶

7.1 吞吐量分布¶

波长相关吞吐量：

各波段吞吐量贡献：

C-band: ~50 Tb/s (高SNR)
L-band: ~45 Tb/s
S-band: ~40 Tb/s
O-band: ~35 Tb/s (FWM限制)
E-band: ~30 Tb/s (水峰损耗)

总计: >200 Tb/s (保守估计)

7.2 调制格式选择¶

自适应调制：

根据SNR选择最优格式：

高SNR (>20 dB): 256-QAM
中SNR (15-20 dB): 64-QAM
低SNR (10-15 dB): 16-QAM
极低SNR (<10 dB): QPSK/4-QAM

目标：最大化总吞吐量

7.3 容量分析¶

总容量：

总带宽: 42.4 THz
总通道: 1273
平均谱效率: ~10-12 bit/s/Hz

总容量:
- GMI: >450 Tb/s
- FEC解码后: 418 Tb/s

每通道平均: ~330 Gb/s

八、与实验室结果对比¶

8.1 场部署vs实验室¶

场部署挑战： | 因素 | 实验室光纤 | 场部署 | |------|-----------|--------| | 接续损耗 | 低 | 高 (老旧) | | 连接器 | 优化 | 较多 | | 维护 | 无 | 有 | | 非线性 | 可控 | 复杂 |

8.2 之前的记录¶

30.4 THz / 300.3 Tb/s：

之前最宽场部署:
- 30.4 THz带宽
- 300.3 Tb/s吞吐量

本文突破:
- 42.4 THz (+39%)
- 450 Tb/s (+50%)

九、技术深度¶

9.1 零色散波长¶

G.652-D色散：

零色散波长: 1312.4 nm
色散斜率: ~0.09 ps/nm²/km

对UWB影响：
- O-band: 低色散
- E-band: ~3 ps/nm/km
- S/C/L-band: 典型色散

9.2 EDFA增益竞争¶

C/L波段增益竞争：

增益带宽限制：
- 传统EDFA: C-band优先
- 扩展L-band: 增益竞争

解决方案：
- 分段放大
- 增益平坦滤波器

9.3 通道选择¶

测试通道策略：

中心通道:
- 独立调制
- 精确测量
- 可变格式

邻居通道:
