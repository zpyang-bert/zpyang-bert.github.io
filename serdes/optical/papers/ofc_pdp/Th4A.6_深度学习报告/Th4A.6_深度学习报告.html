Th4A.6 深度学习报告：TFLN晶圆级共封装光学引擎¶

原始论文：Shanghai Jiao Tong University — "TFLN-based Wafer-Level Co-Packaged Optics Engine for Ultrahigh-Bandwidth Electro-Optical Modulation" 整理日期: 2026/04/15 研究深度: 深度解读（结合异构集成、CPO封装、微波光子学、时间频率传递）

一、论文核心贡献¶

1.1 技术突破¶

本文展示了TFLN EOM与驱动EIC的晶圆级异构集成，实现了封装后>100GHz带宽的CPO引擎。

核心成就： | 指标 | 数值 | 意义 | |------|------|------| | EO带宽(封装前) | 110 GHz | 极限带宽 | | EO带宽(封装后) | >100 GHz | 封装保持率>90% | | Vπ | 4 V | CMOS兼容驱动 | | 插入损耗 | <4.2 dB | 低损耗 | | S₁₁ | <-10 dB @ 100GHz | 良好匹配 | | 时间偏差 | 0.8 fs @ 1s | 飞秒级精度 | | MDEV | 1.2×10⁻¹⁵ @ 1s | 频率稳定度 |

1.2 CPO的战略价值¶

共封装光学 vs 可插拔：

特性 CPO 可插拔
距离 <2m >2m
功耗 <5 pJ/bit ~15 pJ/bit
延迟 <1ns ~10ns
密度 极高 中
散热 共享 独立
可维护性 低 高
成本(小批量) 高 低

封装挑战：

传统封装问题：
- Wire-bond → 电感限制 ~30GHz
- 芯片级封装 → 寄生效应
- 光耦合 → 损耗增加

晶圆级方案优势：
- 短互连 → <0.5 dB @ 100GHz
- 高密度 → 支持>200GHz
- 批量生产 → 成本降低

二、晶圆级异构集成工艺¶

2.1 工艺流程¶

6步晶圆级异构集成：

Step 1: TFLN EOM芯片准备
         - 薄膜LiNbO₃波导
         - CPW电极
         ↓
Step 2: EIC (驱动IC)芯片准备
         - SiGe/CMOS驱动放大器
         - 50Ω匹配网络
         ↓
Step 3: 首次Seed层沉积
         - Cr/Cu (15nm/150nm)
         ↓
Step 4: 芯片贴装与对准
         - Die-to-wafer键合
         - 对准精度 <1μm
         ↓
Step 5: 聚合物填充
         - 底部填充
         - 平面化
         ↓
Step 6: 第二次Seed层 + 电镀
         - 12μm厚光刻胶
         - 3μm Cu + 1μm Au电镀
         ↓
         → 芯片级互连 + 扇出重布线

2.2 互连结构¶

电镀铜柱互连：

互连参数：
- 铜柱高度: ~12 μm
- 铜柱直径: ~10 μm
- 节距: ~50 μm
- 电阻: <10 mΩ/柱
- 电感: <0.1 nH

对比Wire-bond：
- Wire-bond: 0.3-0.5 nH/mm
- 芯片柱: <0.1 nH total

→ 带宽提升3-5×

2.3 扇出重布线层 (RDL)¶

重布线功能：

从芯片阵列 → 晶圆级互联

RDL层：
- 介电: BCB或PI
- 金属: Cu/Au
- 线条宽度: 5-10 μm
- 间距: >5 μm
- 阻抗: 50Ω 匹配

三、封装后性能¶

3.1 带宽保持¶

封装前后对比： | 参数 | 封装前 | 封装后 | 保持率 | |------|--------|--------|--------| | 3dB带宽 | 110 GHz | >100 GHz | 91% | | 插入损耗 | <3 dB | <4.2 dB | +1.2dB | | Vπ | 3.5 V | 4 V | +14% |

仿真预测：

仿真结果：
- 封装结构3dB带宽: >200 GHz
- 互连损耗 @ 100GHz: <0.5 dB

→ 支持未来 >200GHz EOM

3.2 阻抗匹配¶

S₁₁ (回波损耗)：

实测结果：
- S₁₁ < -10 dB @ 100 GHz
- 良好阻抗匹配

TFLN MZM参数： | 参数 | 数值 | 说明 | |------|------|------| | 半波电压 Vπ | 4 V | 封装后 | | 消光比 | >30 dB | 高ER | | 插入损耗 | <4.2 dB | 含耦合损耗 | | 偏振相关 | <0.5 dB | TE偏振 |

四、时间频率传递应用¶

4.1 实验配置¶

光纤时频传递系统：

本地站:
    TFLN CPO Engine
         │
         ├──→ 功分器
         │        │
         │   ┌───┴───┐
         │   │       │
         │  LO    AOM
         │  本振   移频
         │   │       │
         │   └─┬─────┘
         │      │

AOM移频：

AOM功能:
- 移频: ~40-80 MHz
- 隔离: >30 dB
- 消除瑞利背散射

双AOM配置:
- 本地AOM: +f
- 远端AOM: -f
- 净频移: 2f

4.2 94 GHz高频调制¶

为什么选择94 GHz？

1. TFLN带宽足够
   - EOM 3dB > 100 GHz

2. 降低噪声影响
   - 高频调制 → SNR提升
   - 抗环境干扰

3. 频率稳定度改善
   - 相位噪声归一化

4.3 飞秒级精度¶

时间偏差结果：

1秒平均时间:
- 时间偏差 (σ): 0.8 fs
- 相当于光程 0.24 μm

10秒平均:
- σ ≈ 0.25 fs

100秒平均:
- σ ≈ 0.08 fs

→ 飞秒精度稳定性

** Allan偏差 / MDEV**：

MDEV (改良Allan偏差):

τ (平均时间)    MDEV
1 s            1.2×10⁻¹⁵
10 s           3×10⁻¹⁶
100 s          8×10⁻¹⁷

对比光学频率梳:
- 传统光学梳: 10⁻¹⁵ @ 1s
- 本方案: 1.2×10⁻¹⁵ @ 1s ✓

优势:
- 体积小 (芯片级)
- 功耗低 (<10W)
- 成本低

六、异构集成深度技术¶

6.1 芯片贴装对准¶

Die-to-wafer键合：

对准精度要求: <1 μm

面临挑战:
1. 热膨胀失配
   - Si: 2.6×10⁻⁶/K
   - LiNbO₃: 0.5×10⁻⁶/K
   - 温差 ΔT = 100°C
   - 失配: ~0.26 μm ✓

2. 表面平整度
   - TIR < 5 μm
   - 污染控制

3. 键合强度
   - >5 MPa 剪切强度

6.2 应力管理¶

封装应力影响：

应力来源:
- 热膨胀失配
- 模塑料固化收缩
- 操作机械应力

对TFLN的影响:
- Pockels效应变化
- 相位漂移
- 消光比恶化

缓解措施:
- 应力缓冲层
- 局部加强筋

🔍 深度说明：本图展示基于AOM（声光调制器）的相干光时延测量定位系统原理框图，属于微波光子测距/时频传递的典型架构。左端Port1输入光信号，经可调光衰减器（ACM）调幅后进入光环行器（单向导通）→外接目标（光被反射折返）→环行器接收口输出至1×2光分路器→两路分别进入PD光电探测→ADC采样→数字信号处理链（FFT→峰值提取→相位提取→时差计算→Δt→定位）。核心测距原理：发射信号与回波信号在光电域产生干涉，通过FFT得到频谱相位差Δφ，由Δt=Δφ/(2πf)计算时延，进一步由c·Δt/2得到目标距离。AOM在这里同时起到移频（约40-80MHz）和收发隔离（>30dB）的双重作用，有效消除瑞利背散射噪声。研究背景：相干光时延测量（Coherent OTDR / Φ-OTDR）在2018-2025年间快速发展，广泛用于周界安防、油气管道和光纤传感，本方案将AOM和相干探测结合，可在10km以上光纤实现厘米级测距精度。工程价值：对于光模块的在线监测和健康诊断，相干OTDR技术可在不中断业务的前提下检测光纤链路损耗分布事件（弯曲损耗、连接器损耗等），是CPO和可插拔光模块的必备运维工具。落地注意：AOM的移频量f直接决定测距分辨率（分辨率=c/(4f)），若f=80MHz则理论分辨率~0.94m，实际系统需考虑相位噪声和ADC采样率限制；此外OTDR的动态范围（通常40-50dB）限制了可测最大距离，高阶相干检测可提升至60dB+。

🔍 深度说明：本图展示TFLN CPO引擎输出光信号的光谱分析（OSA测试），属于光通信系统出光质量的常规表征。横轴波长范围1548-1552nm（光通信C波段低损耗窗口中心），纵轴功率刻度-10dBm至-70dBm。光谱特征：3个主信号峰分别位于1549.2nm、1550.2nm、1551.2nm，峰值功率约-20dBm，波长间隔约1nm（对应约125GHz频率间隔，符合WDM波分复用系统的标准信道间隔），3个主峰功率基本一致，说明TFLN调制器的频响在C波段内较平坦。左右两侧各有1个低功率旁峰：1548.5nm处约-60dBm，1551.8nm处约-50dBm——这些旁峰可能是相邻信道的串扰（crosstalk）或EDFA自发辐射放大（ASE）噪声叠加。底部噪声底约-70dBm，为OSA和系统的本底噪声。研究背景：对于相干光通信系统，光谱的边模抑制比（SMSR）直接影响OSNR进而影响误码率，本图3个主峰功率差<1dB说明调制链路平衡性良好。工程价值：光模块出光的光谱质量直接决定了后续相干接收的OSNR预算——若SMSR<30dB，则相邻信道串扰会显著恶化误码率（尤其在256QAM高阶调制下）。落地注意：OSA的分辨率带宽（RBW）设置会影响峰值读数，通常1nm RBW适合粗略观测，0.1nm RBW可更精确评估边模抑制比；此外光谱仪的波长校准精度（通常±0.05nm）也是多波长WDM系统信道间隔测量的重要误差来源。

🔍 深度说明：本图展示与上一个AOM系统架构基本相同的光相干时延测量定位系统原理框图（可能是同一系统的不同配置版本），信号链路完全一致：ACM→光环行器→目标反射→1×2光分路器→两路PD→ADC→DSP处理链（FFT→峰值提取→相位提取→时差计算→τ），两路分别对应相干探测的I/Q同相和正交支路，最终输出时延τ值用于目标距离解算。核心差异在于：两路PD输出的I/Q信号经FFT后，不仅提取了回波信号的频率峰值信息，还分别提取两路信号的相位信息，通过I/Q相位差Δφ=φ_I-φ_Q换算得到精确时延，I/Q相干检测相比直接检测（direct detection）可大幅提升信噪比（约3-10dB增益）。研究背景：I/Q相干探测最早用于雷达和相干光通信，后来被引入光纤时频传递和分布式传感领域，可同时获得目标的距离（通过时延τ）和位移速度（通过多普勒频移），是相干OTDR/Φ-OTDR的核心工作原理。工程价值：在光模块制造和现场部署中，相干OTDR是检测光纤链路质量的唯一手段——可检测的事件类型包括：熔接点损耗（splice loss <0.1dB）、连接器回损（connector RL >45dB）、弯曲损耗（macrobend/microbend），以及故障点定位精度通常<1m。落地注意：I/Q相位解缠（phase unwrapping）是大动态范围测量的关键难点，当相位差超过2π时需要线性解缠算法；此外ADC的ENOB（有效位数，通常<10bit@GHz以上采样率）会限制相位提取精度，高速DSP处理器的实时性（通常要求<1ms处理延迟）也是工程实现的核心瓶颈。

🔍 深度说明：本图展示TFLN CPO引擎进行光纤时频传递时的时间偏差（Time Deviation, σ）随平均时间的变化曲线，双对数坐标系，横轴平均时间τ从10⁻⁴s到10²s，纵轴时间偏差从10⁻¹²s到10⁻⁹s，对应Fig. 2的第(c)子图。曲线分三段特征：（1）下降段：τ从0.0001s到1s，σ从~1ns持续下降至~5ps，呈现σ∝τ⁻¹的白相位噪声主导特征；（2）谷底段：τ在1s~10s区间，σ达到最低值约5ps，是该系统时间稳定度最优的工作点；（3）上升段：τ>10s后，σ随τ增加而升高（从~5ps回升至~0.1ns），说明长平均时间下系统受低频闪烁噪声（1/f噪声）和慢漂移影响，稳定度恶化。研究背景：本结果发表于2025年OFC PDP，是将TFLN CPO引擎应用于光纤时频传递的首次实验验证。传统光纤时频传递依赖光学频率梳（optical frequency comb）作为频率基准，体积大、功耗高；本方案利用TFLN的高频调制特性结合数字相位锁定，实现芯片级的低SWaP时频传递。工程价值：时间频率传递的精度直接影响GPS失效场景下的通信系统性能——在相干光通信中，本振激光的相位噪声会导致相干接收EVM恶化，高稳定度时频传递链路可支持更远距离（>100km）的相干光通信而无需在线补偿。落地注意：图中τ=1s处σ~5ps，对应频率稳定度约10⁻¹⁵量级（Allan偏差），已达到光学原子钟的精度水平，但长稳（>10s）的恶化趋势需要通过双路相位抵消或温控环路来抑制；实际系统中光纤色散（~17ps/nm/km）对宽带信号的时间传递精度也有显著影响。

🔍 深度说明：本图展示修正阿伦偏差（MDEV, Modified Allan Deviation）随平均时间的变化曲线，双对数坐标系，横轴平均时间τ从10⁻⁴s到10²s，纵轴MDEV从10⁻¹⁶到10⁻¹⁰，属于时频领域的标准稳定性表征参数，对应Fig. 2的第(d)子图。MDEV相比标准Allan偏差的优势在于可以区分白相位噪声和闪烁相位噪声两种不同的噪声机制：短平均时间区间（τ<1s），MDEV以-1斜率呈线性下降（MDEV∝τ⁻¹），说明此时系统受白相位噪声主导——这是所有振荡器在短积分时间内的固有行为；长平均时间区间（τ≥1s），MDEV下降斜率显著变缓并逐渐进入平台区（MDEV~10⁻¹⁵~10⁻¹⁶），说明闪烁频率噪声（1/f噪声）成为主导，此时继续延长平均时间已无法提升稳定度，接近该频率源的噪声基底。研究背景：MDEV是评估光频传递系统和原子钟性能的核心指标。传统光频梳方案的MDEV在τ=1s时约10⁻¹⁵量级，本文TFLN CPO方案在相同条件下达到MDEV~1.2×10⁻¹⁵，基本相当，但系统体积和功耗降低2-3个数量级，这对卫星载荷和野外应急通信场景意义重大。工程价值：对于相干光通信系统的本振激光，频率稳定度直接决定相位噪声性能——在400km光纤传输后，若本振稳定度不足，相位噪声会导致相干接收机的EVM恶化约3-5dB，这对高阶QAM（64QAM/256QAM）来说是不可忽视的性能损失。落地注意：MDEV测试需要长平均时间（通常τ>100s才能评估长稳），这对实验设备的稳定性和环境隔离要求极高；另外MDEV的平台区高度与光源的相位噪声功率谱密度直接相关，测试中任何1Hz以上的环境扰动（温度波动、机械振动）都会导致测量结果失真。

🔍 深度说明：本图为光纤时频传递系统的性能基准对比散点图，横轴为单跨传输距离（0-400km），纵轴为时间偏差（10⁻¹⁶s到10⁻¹¹s，对数刻度），对应Fig. 2的第(e)子图。黑色三角（文献[13/14/15]，bulky optical comb方案）：传输距离约100km，时间偏差在10⁻¹²-10⁻¹¹s量级，属于传统大型光频梳系统的典型性能。黑色方块（文献[11] OFC 2025和作者前序工作）：传输距离接近400km时，时间偏差约10⁻¹²s，长距离传输稳定度显著恶化。黑色菱形（文献[16/17]，bulky optical comb）：传输距离100-150km，时间偏差约10⁻¹³s。五角星（本文新方案）：红色五角星在50km处时间偏差低至10⁻¹⁵s；黑色五角星在400km处时间偏差维持~2×10⁻¹⁵s，相比前序工作（10⁻¹²s）和bulky optical comb方案（10⁻¹³s），稳定度提升约100倍。图上标注"Stability enhancement ~ 100 times"。研究背景：光纤时频传递的核心挑战是：随着传输距离增加，光纤色散、非线性效应（SPM/XPM）、偏振模色散（PMD）会导致传输信号的时间抖动恶化；传统方案的长稳恶化主要源于光频梳与传输光纤之间的相位噪声传递效率。工程价值：100倍稳定度提升意味着400km光纤传递后的时间偏差从~1ps降至~10fs，相当于光程精度从~0.3mm提升至~3μm——这对于远程相干光通信（>100km）的本振分发、分布式传感网络的时间同步、以及5G/6G前传系统的时钟同步具有重大实用价值。落地注意：本文稳定度提升的关键在于TFLN CPO引擎结合AOM移频双路相位抵消技术（抵消了大部分光纤色散噪声），实际工程部署中AOM的移频精度（通常<100Hz）和双路平衡度（要求>40dB）直接决定最终传递精度；另外400km传输需要考虑EDFA级数（通常每80km一个）带来的ASE噪声积累。
