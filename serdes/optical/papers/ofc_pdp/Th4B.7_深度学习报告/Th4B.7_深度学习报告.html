Th4B.7 深度学习报告：266 km超长跨距空心光纤跨洋传输¶

原始论文：Nokia Bell Labs + YOFC — "Sparsely repeated 21.7 Tb/s Net-Rate Transoceanic Transmission with 266 km Ultra-Long Spans Enabled by Low IMI and Low loss Hollow Core Fiber" 整理日期: 2026/04/15 研究深度: 深度解读（结合空心光纤、超长跨距、跨洋传输、模式干涉抑制）

一、论文核心贡献¶

1.1 技术突破¶

本文展示了业界首个266 km超长跨距空心光纤传输，实现21.7 Tb/s净速率跨洋传输，仅需不到30个中继器。

核心成就： | 指标 | 数值 | 意义 | |------|------|------| | 净速率 | 21.7 Tb/s | 跨洋距离 | | 传输距离 | 6660 km | 25×266km | | 跨距长度 | 266 km | HCF最长 | | 中继器数量 | <30个 | 稀疏中继 | | 损耗系数 | 0.098 dB/km | 极低损耗 | | 频谱效率 | 4.48 b/s/Hz | 高效率 |

1.2 空心光纤的优势¶

HCF vs SMF： | 特性 | HCF | SMF | HCF优势 | |------|-----|-----|---------| | 损耗 | 0.1 dB/km | 0.2 dB/km | ✓ 更低 | | 非线性 | 极低 | 高 | ✓ 100×降低 | | 延迟 | ~3.5 μs/km | ~5 μs/km | ✓ 30%降低 | | 色散 | 低 | ~17 ps/nm/km | ✓ 更低 | | 偏振模色散 | 极低 | 中等 | ✓ 更好 |

超长跨距意义：

传统SMF跨距: ~80-100 km
HCF跨距: ~266 km

减少中继器数量：
- 设备成本降低
- 维护成本降低
- 功耗降低

二、GTA-ST-HCF设计与工艺¶

2.1 间隙管辅助结构¶

GTA-ST-HCF架构：

       支撑管架构
           │
    ┌──────┴──────┐
    │   空气芯      │ ← 光在空气中传输
    │              │
    │  ┌────────┐  │
    │  │ 间隙管 │  │ ← 模式滤波
    │  │ Gap    │  │
    │  │ Tube   │  │
    │  └────────┘  │
    │   薄壁玻璃管  │
    └──────────────┘

双重功能：

1. 支撑管 (Support Tube):
   - 机械支撑
   - 实现超低限制损耗

2. 间隙管 (Gap Tube):
   - 模式滤波机制
   - 降低高阶模式耦合
   - 减少IMI

2.2 低IMI设计¶

模式间干涉 (IMI)：

IMI来源:
- 高阶模式(HOM)耦合
- 模式间干涉
- 信号衰落

之前HCF问题:
- IMI ~ -52 dB/km
- 限制传输距离

本文改进:
- IMI降低 6.4倍
- 显著减少信号衰落

2.3 266 km光纤制造¶

23个GTA-ST-HCF光纤盘：

制造工艺:
1. 预制棒制备
   - 毛细管堆叠
   - 间隙管插入
   - 精确几何控制

2. 拉丝
   - 直径控制
   - 涂层固化

3. 熔接
   - 23盘光纤熔接
   - 低损耗熔接点
   - <0.1 dB/熔接点

三、传输性能¶

3.1 损耗特性¶

双向OTDR测量：

测得损耗系数: 0.098 dB/km

对比:
- 传统SMF: 0.2 dB/km
- 之前HCF: 0.15-0.2 dB/km
- 本文HCF: 0.098 dB/km ✓

266 km总损耗: ~26 dB

3.2 气体线吸收 (GLA)¶

GLA吸收峰：

位置: 1530-1545 nm
原因: 空气中的气体吸收

影响:
- 某些波长高损耗
- 需避免或补偿

规避策略:
- 自适应符号率
- 避开GLA区域

3.3 C波段WDM¶

通道配置：

总通道数: ~60个
C-band范围: 191.5-195 THz

符号率设计:
- 避开GLA: ~135通道 @ 各符号率
- GLA区域: 降低符号率

符号率范围:
- 正常: 52 GBaud
- GLA区域: 30-46 GBaud

四、实验结果¶

4.1 SNR性能¶

6660 km @ 25 round-trips：

SNR vs 频率:

191.5-195 THz区域:
- SNR ~12-14 dB
- 均衡高性能

GLA第二区域 (1530-1545nm对应):
- SNR penalty: 2-3 dB
- 仍可传输

4.2 容量-距离积¶

跨洋传输记录：

本文成果:
- 容量×距离: ~145 Pbps·km
- 相比之前HCF: 2倍提升

对比SMF:
- SMF记录: 570 Pbps·km (55km跨距)
- 短跨距优势

本文特点:
- 超长跨距
- 较少中继器
- 跨洋距离

4.3 净吞吐量¶

C波段AIR (Aggregate Information Rate)： | 距离 | AIR | 净吞吐量 | 频谱效率 | |------|-----|---------|----------| | 5320 km | 25.8 Tb/s | 23.5 Tb/s | 4.82 b/s/Hz | | 6660 km | 23.4 Tb/s | 21.7 Tb/s | 4.48 b/s/Hz | | 7990 km | 19.3 Tb/s | 15.9 Tb/s | 3.3 b/s/Hz |

五、超长跨距技术¶

5.1 中继器设计¶

低损耗+高功率：

266 km跨距需求:
- 跨距损耗: ~26 dB
- 需要高功率放大

放大器配置:
- 输入功率: 34 dBm
- 输出功率: ~35-36 dBm
- 低噪声指数

中继器数量:
- 6660 km / 266 km ≈ 25个
- 实际: <30个

5.2 自适应符号率¶

GLA规避策略：

问题:
- GLA在特定波长
- 固定符号率效率低

自适应方案:
1. 频谱分析
2. GLA区域识别
3. 动态调整符号率

效果:
- 避开吸收峰
- 保持均衡SNR
- 最大化容量

5.3 IMI抑制¶

模式纯度提升：

统计结果:
- 功率标准差降低 6.4倍
- 对比IMI=-52 dB/km HCF

实现方法:
- 间隙管模式滤波
- 支撑管结构
- 精确制造

六、与SMF对比¶

6.1 容量-距离对比¶

SMF vs HCF：

SMF记录 (55km跨距):
- 570 Pbps·km
- 需要大量中继器

HCF本文 (266km跨距):
- ~145 Pbps·km
- 中继器减少 5×

权衡:
- HCF容量较低
- 但跨距长 → 设备少

6.2 跨洋系统成本¶

总拥有成本：

中继器成本:
- 每个~$100K
- SMF需要 ~75个 (5000km)
- HCF需要 <30个

光纤成本:
- HCF: ~10× SMF

系统选择:
- 距离越长，HCF优势越大
- 跨洋应用HCF经济优势

七、技术深度¶

7.1 OTDR表征¶

双向OTDR技术：

测量方法:
1. 前向OTDR
2. 后向OTDR
3. 差分计算

分辨率: 1 pm
获得:
- 损耗系数
- 熔接点位置
- GLA吸收谱

7.2 SC-LDPC编码¶

空间耦合LDPC：

编码配置:
- SC-LDPC码
- 自适应码率
- 软判决解码

目标:
- 逼近香农极限
- 降低开销

7.3 NGMI分析¶

归一化广义互信息：

NGMI vs 码率:

NGMI > 0.85 → 可解码
NGMI > 0.95 → 接近无误码

计算得到:
- 每通道最优码率
- 自适应FEC

八、未来应用¶

8.1 跨洋系统¶

海底光缆升级：

传统SMF海底:
- 跨距: 60-80 km
- 需要大量中继器
- 成本极高

HCF海底可能:
- 跨距: 200-300 km
- 减少中继器
- 降低CAPEX/OPEX

8.2 陆上长距¶

超长距地面：

应用场景:
- 沙漠
- 山脉
- 难以建站区域

减少中继站:
- 降低维护难度
- 提高可靠性

九、总结与技术洞察¶

9.1 论文核心价值¶

  1. 首个266km HCF跨距 — 突破极限
  2. 21.7 Tb/s @ 6660 km — 跨洋验证
  3. 2倍容量提升 — 相比之前HCF
  4. 稀疏中继 — <30个中继器

9.2 对行业的影响¶

方面 影响
海底光缆 新一代可能
光纤技术 推动HCF成熟
成本 长距更经济
标准 空心光纤标准

9.3 关键技术亮点¶

亮点 数值 意义
跨距 266 km 最长HCF
损耗 0.098 dB/km 极低

🔍 深度说明：本图展示266km超长跨距空心光纤（Hollow-Core Fiber, HCF）跨洋传输实验装置和GTA-ST-HCF的衰减/色散随波长变化。HCF（空心光纤）：光纤中心为空气（折射率n≈1），包层为周期性毛细管结构（光子带隙导光），与传统实芯SMF（n≈1.47）相比，HCF具有超低延迟（-30%）、超低非线性（空气非线性系数是玻璃的1/1000）和超低衰减潜力（理论上<0.1dB/km）。GTA-ST-HCF：种特殊设计的HCF，衰减约1dB/km（当前实验水平），色散约+22ps/nm/km（与SMF相反，为正常色散区）。实验装置：TLS（可调谐激光器）-> IQ调制器 -> HCF 266km -> 相干接收机 -> DSP。266km是HCF传输的世界纪录——此前HCF传输距离限制在约100km（由衰减和非线性积累限制）。

🔍 深度说明：本图为Fig.1的(b)子图——GTA-ST-HCF空心光纤的衰减-色散联合光谱，测量波长1530~1565nm（C-band），横轴nm，左轴蓝色dB/km为衰减系数，右轴橙色ps/(nm·km)为色散系数。关键数据：衰减在0.098~0.11dB/km之间（1550nm处~0.1dB/km，比普通SMF低约50%），在1530~1540nm区间有两个损耗峰；色散随波长正线性增长，1550nm处约4.6ps/(nm·km)（仅为SMF的1/4），色散斜率~0.025ps/(nm²·km)。与SMF对比：HCF衰减优势明显（0.1 vs 0.2dB/km），色散低3/4且为正常色散区（SMF为异常色散），色散斜率更低利于WDM平坦化。工程价值：HCF的超低衰减可延长无中继传输距离30~50%，超低色散减少色散补偿需求（节省 DCM部署），但当前0.1dB/km仍高于理论极限（<0.05dB/km），且熔接损耗（0.1~0.5dB/点）和耦合损耗仍是商用化主要瓶颈。

🔍 深度说明：本图为Fig.1的(c)子图——两种空心光纤(GTA-ST-HCF vs 普通HCF)接收功率概率密度分布(PDF)对比。横轴为归一化接收光功率，纵轴为概率密度。蓝色柱状（GTA-ST-HCF）：功率高度集中于1.0附近，波动范围<±2%，传输稳定性极佳，得益于其稳相结构抑制模式耦合和外界振动干扰。红色曲线（普通HCF，损耗52dB/km）：呈宽分布，功率波动>±10%，传输稳定性差。GTA-ST-HCF是改进型高性能空心光纤，专为长距离稳定传输设计，解决了普通HCF功率抖动大的难题。工程价值：该对比解释了为何GTA-ST-HCF能实现7990km可靠传输（30×266km级联）：其功率稳定性确保接收SNR波动小，避免突发误码；而普通HCF的高功率抖动会导致FEC译码突发错误，无法支撑长距传输。该图也说明HCF的"稳相结构"设计是商用化的关键技术突破点。

🔍 深度说明：本图为Fig.1的(a)子图——266km HCF跨洋传输实验装置完整链路图。发射端：TLS（可调谐激光器，输出功率10~20dBm，线宽<1kHz，C-band）→ WS（波形整形器，生成多波WDM信号）→ PS（偏振控制器）→ IQ调制器（生成QPSK/16QAM调制信号）；传输链路：266km GTA-ST-HCF（超低损耗<0.1dB/km，SMF熔接损耗<0.5dB/点），中间可能含低噪声EDFA补偿；接收端：相干接收（平衡探测器+本地振荡器）→ ADC（>80GSa/s）→ DSP（色散补偿、偏振解复用、相位恢复、LDPC译码）→ 误码测试。266km是HCF单跨世界纪录（此前约100km），得益于GTA-ST-HCF的稳相设计抑制模式耦合。工程参考：HCF与SMF的熔接损耗是链路最大挑战（0.1~0.5dB/点），实际部署需精确端面处理和熔接参数优化；中继放大器需选用低噪声EDFA（NF<4dB）以抑制ASE对超低非线性HCF的优势抵消。

🔍 深度说明：本图展示266km跨度的双向OTDR轨迹（20µs脉冲测量）。OTDR（光时域反射仪）测量：沿光纤长度方向记录回波功率，定位事件点（熔接点、连接器）。前向vs后向OTDR差异：前向测量（从发射端注入）显示266km后功率衰减约30dB；后向测量（从接收端注入）显示相同路径。差异点：两个方向测量的差异约0.5dB，由光纤的不对称性（制造公差、安装弯曲）引起。HCF的OTDR信号比SMF弱约20dB（因HCF回波散射系数低），需要更高功率的OTDR脉冲。

🔍 深度说明：本图为Fig.3的(a)子图——6660km和7990km HCF传输后SNR随频率变化曲线，横坐标192~196THz（C-band），纵坐标SNR(dB)。蓝色菱形：6660km（30×222km SMF对比链路）；红色三角：7990km（30×266km HCF）。关键数据：两个距离中低频段（192~194.5THz）SNR平稳，194.5THz以上SNR随频率升高而下降；7990km在195THz以上SNR降至~5dB（同频6660km为7~8.5dB），差距1~2dB源于额外266km的ASE噪声和非线性累积。核心规律：高频SNR衰减更显著——受光纤带宽限制和DSP均衡残余误差影响，高频信号承受更大 ISI 代价。工程参考：设计超长距系统（如7990km HCF跨洲际链路）时，高频子信道（>194.5THz）应降低符号率或减少WDM信道数以保证Q裕量；也可通过提高ADC采样率和DSP算法带宽来延展高频性能上限。

🔍 深度说明：本图为Fig.3的(b)子图在7990km传输距离下的重复测试结果——NGMI与编码率随光频率变化的另一组测量。横坐标192~196THz，左轴NGMI（0.5~1），右轴编码率，蓝色圆点NGMI、红色方块编码率。7990km数据（对比6660km）：全频段NGMI比6660km低约0.05~0.1（额外ASE噪声和微弱非线性累积），中心频段NGMI约0.68~0.72（vs 6660km的0.74~0.78），对应编码率约0.62~0.68。SNR下降幅度：7990km在高频段（>195THz）SNR降至~5dB，相比6660km同频段低约2~3dB，这与NGMI的下降幅度一致——说明NGMI的下降主要由SNR劣化引起。HCF vs SMF：同等距离下HCF非线性代价仅~1dB，而SMF约5dB，这解释了为何HCF能在30×266km级联后仍保持>0.65的NGMI（SMF在相似距离下NGMI往往<0.6）。工程参考：7990km HCF链路的高频边缘（>195THz）建议通过降低符号率（从135GBd降至52GBd）来换取约2~3dB的SNR改善，保证该频段的NGMI仍>0.65以满足FEC无误码要求。
