Th4A.7 深度学习报告：1.6T硅光子2×FR4收发器与布拉格光栅复用器¶

原始论文：Wuhan HGGenuine Optics / Genuine Optics — "1.6T (8×200Gb/s) 2×FR4 Silicon Photonic IMDD Transceiver with Monolithically Integrated Ultra-Low Crosstalk and Wideband Multiplexer" 整理日期: 2026/04/15 研究深度: 深度解读（结合硅光子学、WDM复用、IC设计、SerDes系统）

一、论文核心贡献¶

1.1 技术突破¶

本文展示了世界首款基于布拉格光栅的1.6T硅光子2×FR4收发器，在CWDM4波长上实现了突破性性能。

核心成就： | 指标 | 数值 | 意义 | |------|------|------| | 总容量 | 1.6 Tb/s | 8×200G | | 单通道速率 | 200 Gb/s | 106.25 GBd PAM4 | | 封装形式 | OSFP 2×FR4 | 标准形态 | | DSP节点 | 3nm | 先进节点 | | 消光比 | >4 dB | 满足标准 | | 线性度 RLM | >0.96 | 优异线性 | | TDECQ | <2.5 dB | 满足FR4要求 | | 预FEC BER | <1×10⁻¹³ | 极大余量 | | 接收灵敏度 | <-4 dBm @ BER=10⁻⁶ | 高灵敏度 |

1.2 布拉格光栅复用器的创新¶

对比传统方案：

MUX类型 插入损耗 平顶带宽 串扰 尺寸
布拉格光栅 0.6-0.9 dB 16-20 nm -25 dB 1.6×0.04 mm²
级联MZI 0.9-1 dB 11-14 nm -15 dB 2.2×0.4 mm²

改进倍数： - 带宽提升: +5 nm (36%提升) - 串扰改善: +10 dB (3×改善) - 尺寸缩小: 15× (从2.2×0.4到1.6×0.04)

二、2×FR4架构¶

2.1 FR4标准¶

CWDM4波长分配：

2×FR4 = 2 × 4波长

第一组 (Lane 0-3):
- CH1: 1271 nm
- CH2: 1291 nm
- CH3: 1311 nm
- CH4: 1331 nm

第二组 (Lane 4-7):
- CH5: 1271 nm
- CH6: 1291 nm
- CH7: 1311 nm
- CH8: 1331 nm

通道间隔: 20 nm

为什么FR4？

优势：
- 使用SMF (G.652)
- 适合数据中心
- 成本低于LWDM
- 波长间隔宽 → 容错性好

挑战：
- 带宽受限 (O-band)
- 非等间隔 → 设计复杂

三、布拉格光栅原理¶

3.1 布拉格反射原理¶

光栅方程：

布拉格条件：
m·λ_B = 2·n_eff·Λ

其中：
- m = 衍射级数 (通常 m=1)
- λ_B = 布拉格波长
- n_eff = 有效折射率
- Λ = 光栅周期

设计参数：
- n_eff ≈ 2.15 (Si waveguide)
- λ_B = 1291 nm
- Λ ≈ 300 nm

反射谱特性：

反射带宽 Δλ:

Δλ = λ_B × (Δn_eff / n_eff) × (N / N_eff)

其中：
- Δn_eff = 折射率调制幅度
- N = 光栅周期数
- N_eff = 有效折射率加权

典型设计：
- 反射带宽: 15-20 nm
- 旁瓣抑制: >15 dB
- 纹波: <0.5 dB

3.2 均匀布拉格光栅¶

折射率调制：

n(z) = n₀ + Δn · cos(2πz/Λ)

其中：
- n₀ = 平均折射率
- Δn = 折射率调制幅度 (~10⁻⁴ - 10⁻³)
- Λ = 光栅周期

耦合系数 κ:
κ = π · Δn / λ_B

典型值：
κ ≈ 50-200 cm⁻¹

3.3 Apodized (渐变) 光栅¶

为什么要Apodization？

均匀光栅问题：
- 高旁瓣
- 纹波大
- 频谱不平坦

Apodization方案：
n(z) = n₀ + Δn(z) · cos(2πz/Λ)

Δn(z) 分布：
- 汉明窗: Δn(z)·sin(πz/L)
- 余弦平方: Δn(z)·cos²(πz/L)
- 高斯: Δn(z)·exp(-(z/σ)²)

效果：
- 旁瓣抑制 >20 dB
- 平顶带宽最大化
- 纹波 <0.2 dB

四、芯片设计与工艺¶

4.1 芯片布局¶

2×FR4发射机布局：

尺寸: 1.6mm × 0.04mm (布拉格MUX)

组成:
- 4× CWDM波长激光器 (可选)
- 4× SiPh MZM调制器
- 1× Bragg MUX
- 4× 热调谐电阻
- 监控PD阵列

4.2 热灵敏度优化¶

Si波导的温度依赖：

热光系数:
dn/dT ≈ 1.8×10⁻⁴ /K (Si)

波长漂移:
Δλ/ΔT ≈ 0.1 nm/K

20nm通道间隔时：
允许温漂: ±5°C (无主动热调)

优化措施:
1. 材料优化
   - 包层材料选择
   - 应力补偿

2. 波导几何
   - 尺寸优化
   - 掺杂调节

3. 工艺参数微调
   - PDK精确校正

4.3 制造工艺¶

商业200mm SiPh代工：

工艺平台:
- 220nm SOI
- SiO₂包层 (BPSG)
- Ge集成探测器
- 铜金属化

线宽控制:
- 最小特征: 100-200nm
- 公差: ±5nm

成本优势:
- 成熟平台
- 批量生产
- 良率高

余量:
瀑布曲线显示 >10× 余量

宽温度范围运行：

测试温度: 15°C - 70°C

性能稳定性:
- 所有通道一致
- 无主动热调谐
- 被动热稳定设计

实现方法:
- 材料补偿
- 波导设计优化
- 工艺参数精确控制

六、CWDM4系统设计¶

6.1 CWDM vs LWDM¶

为什么选择CWDM？ | 特性 | CWDM | LWDM | |------|------|------| | 波长间隔 | 20nm | 800GHz (~6.4nm) | | 激光要求 | 低精度 | 高精度 | | 温漂容忍 | 高 | 低 | | 成本 | 低 | 中 | | MUX设计 | 简单 | 复杂 |

6.2 波长对准¶

激光波长容差：

ITU标准 CWDM: λ ± 6.5nm

MUX通道带宽 (布拉格):
- 1dB带宽: 16-20nm
- 容差余量: 10nm

激光温漂:
- Δλ/ΔT ≈ 0.1nm/°C
- 70°C范围: 7nm
- 仍在MUX带宽内 ✓

七、与传统MUX对比¶

7.1 级联MZI问题¶

传统CMZI MUX：

结构: 3级MZI级联

问题:
1. 插入损耗高 (0.9-1dB)
2. 带宽窄 (11-14nm)
3. 串扰大 (-15dB)
4. 尺寸大 (2.2×0.4mm²)

对于高速PAM4:
- 带宽收窄 → 眼图恶化
- 串扰高 → 通道间干扰

7.2 布拉格光栅优势¶

本文创新点：

1. 带宽提升
   - 16-20nm vs 11-14nm
   - 放宽激光波长精度
   - 提高温度容限

2. 串扰降低
   - -25dB vs -15dB
   - 改善通道隔离
   - 支持更高阶调制

3. 尺寸缩小
   - 15×面积减小
   - 高密度集成
   - 降低耦合损耗

11.3 眼图闭合¶

TDECQ计算：

TDECQ = 10·log₁₀(σᵣₑc²/σ₀²)

其中:
- σᵣₑc = 接收机等效噪声
- σ₀ = 理想眼图噪声

眼图恶化因素:
- 带宽限制
- 码间干扰 (ISI)
- 非线性失真
- 噪声

🔍 深度说明：本图展示8个CWDM通道在70°C高温环境下的误码率（BER）随光调制幅度（OMA）的变化曲线（Fig.3）。横轴OMA范围-7dBm到3dBm，纵轴BER从1E-2到1E-13（对数刻度），8条曲线对应8个独立通道（CH1-CH8）。两条红色参考线：垂直线OMA=-5.5dBm为典型接收灵敏度参考点，水平线BER=1E-4为IEEE 802.3标准的前向纠错（FEC）可纠正误码率门限。所有通道在OMA>-5dBm时可实现BER<1E-4（满足FEC门限），随OMA增大BER持续下降；70°C高温下8通道一致性良好，曲线基本重合，说明器件封装设计对温度的敏感性控制到位。研究背景：800G 2×FR4光模块的高温（70°C）BER测试是IEEE 802.3cu和MSA规范要求的必测项目，用于验证光模块在数据中心热环境下的工作可靠性；BER vs OMA曲线是光模块灵敏度特性的直接表征，是制定光链路功率预算的核心依据。工程价值：光模块的光功率预算（link power budget）=发射光功率-接收灵敏度，其中接收灵敏度=OMA_thr+接收器损耗；-5.5dBm@BER=1E-4的灵敏度要求，意味着在总链路损耗10dB的场景下，发射光功率需要>-5.5+10=+4.5dBm才能正常工作，这对DFB激光器的输出功率提出了明确要求（通常需要>6dBm）。落地注意：FEC门限的选择需平衡编码增益和延迟开销——KP4 FEC（纠错能力~1E-15）的净编码增益约9.5dB，但编码延迟约100ns；CR5 FEC的延迟更低（约10ns），但编码增益只有约7dB，在TDECQ>2.5dB的高性能光模块中需要仔细权衡。
