高速硅光子学模块封装关键技术研究 - 深度学习报告¶

1. 论文基本信息¶

14 figures from original paper:

图 1.1 图 1.1 光子集成芯片

🔍 深度说明：本图展示光子集成芯片（PIC）的典型结构——将激光器、调制器、探测器、波导等多功能光器件单片集成在硅或InP衬底上。硅光子PIC采用CMOS兼容工艺（8英寸/12英寸晶圆），可批量生产降低成本。图中可见AWG（阵列波导光栅）复用器、调制器、探测器等组件的单片集成。对SerDes模块封装，硅光子PIC的优势是尺寸小（相比分离器件光模块体积缩小>50%）、批量成本低（CMOSfab流片）、可与CMOS电芯片（共封装光学CPO）深度集成。挑战在于：硅是间接带隙材料无法直接发光（需混合集成III-V激光器）、硅波导损耗（~1dB/cm@1550nm）比二氧化硅光纤大、端面耦合效率低（需光栅耦合器或端面耦合器）。商用硅光PIC（如Intel 100G CWDM4）已广泛用于数据中心光模块。

图 1.2 图 1.2 信号接口传输速率

🔍 深度说明：本图展示信号接口传输速率的演进路线图——从10Gbit/s到400Gbit/s乃至1.6Tbit/s的速率提升路径。横轴是年代（2000-2030），纵轴是速率对数坐标。关键节点：2000年代10G SFP+，2010年代28G NRZ QSFP+，2020年代56G PAM4 QSFP-DD/OSFP，2025年代224G PAM4 CPO/NPO。每一代速率提升都伴随着调制格式（NRZ到PAM4到DP-16QAM）、光模块形态（SFP+到QSFP-DD到CPO）和电接口（25G NRZ到56G PAM4到112G PAM4）的全面升级。对于SerDes芯片设计，速率每翻倍意味着带宽需求翻倍（56G PAM4眼宽仅~9ps，对抖动容限<1ps）、SerDes CDR带宽需求提升（GBaud ≈ bit_rate/2 for PAM4）、PCB走线损耗控制更严（>30dB/in@28GHz需背钻）。

图 1.3 图 1.3 硅光子市场规模预测

🔍 深度说明：本图展示硅光子市场规模预测——全球硅光子市场从2010年不足$100M增长到2025年估计$5B以上，复合年增长率CAGR>30%。市场驱动力：数据中心内部光互连需求爆发（10M+服务器联网）、5G前传光模块大批量部署、量子计算/AI加速器光互连新应用。产业链主要玩家：Intel（数据中心100G CWDM4）、Cisco（收购Acacia/Lightwire）、Luxtera/Cisco、SiFotonics、慧脑光电。对SerDes行业，硅光子趋势意味着：光模块内SerDes从独立芯片向共封装（CPO）演进——SerDes电芯片靠近光芯片（<5mm距离）以减少SerDes channel loss，PCB材料从FR4升级到M6/石油焦材料（Dk=3.0, Df<0.002@14GHz），连接器从MPO/LC升级到波分复用耦合。

图 1.4 图 1.4 光互连系统的构架

🔍 深度说明：本图展示光互连系统的典型架构——从服务器CPU/GPU到光模块的信号链路：CPU/GPU芯片 -> SerDes ASIC（重定时器/Retimer）-> PCB走线 -> 光模块SerDes CDR -> 光模块Driver -> 调制器 -> 光纤。光互连距离从芯片内（<1mm，铜缆）到板级（<1m，PCB背板）到机架间（<100m，多模光纤）到数据中心间（>100km单模光纤）分层。对不同距离优化方案不同：<10m用铜缆（AOC有源光缆或铜缆DAC），<2km用多模光纤（SR4/SWDM4），<120km用单模光纤（LR/ER/ZR），>120km用相干光模块。SerDes链路预算分析：56G PAM4系统要求总链路损耗<30dB（含PCB 15dB@14GHz、连接器3dB、光纤2dB@2km），眼图margin>30%，对SerDes TX pre-emphasis和RX CTLE/DSP要求极高。

图 1.5 图 1.5 光互连系统的典型构架：（a）直接调制结构；（b）间接调制结构

🔍 深度说明：本图对比光互连系统的两种典型架构：(a)直接调制结构——激光器直接由电流调制（典型用于VCSEL，速率10-25Gbit/s，成本低但波长固定且啁啾大）；(b)间接调制结构——连续波（CW）激光器+外部调制器（LiNbO3或SiGe或InP马赫-曾德调制器），可实现>100Gbit/s高速调制，波长可调谐但成本较高。直接调制VCSEL优势：成本低（<$1/通道）、尺寸小、易于二维阵列集成；缺点：波长稳定性差（温漂~0.07nm/°C）、啁啾大（限制传输距离300m@10G）、调制速率受限（28G已接近极限）。间接调制硅光子调制器（载流子耗尽型MZM）优势：CMOS兼容可集成、调制速率100Gbaud、支持DP-QPSK/16QAM相干检测；缺点：Vpi较高（~4V for carrier-depletion Si MZM）、损耗较大（~3dB/cm）。

图 1.6 图 1.6 光模块：(a)边沿可插拔光模块；(b) 嵌入式光模块；(c)电芯片与光芯片集成模块

🔍 深度说明：本图展示三种光模块形态：(a)边沿可插拔光模块——QSFP+/QSFP28/QSFP-DD/OSFP，模块插拔在面板前面板，可热插拔，维护方便但SerDes channel从主板到前面板走线长（>20cm PCB），高频损耗大；(b)嵌入式光模块（Embedded Optical Module，EOM）——光引擎嵌入主板，不插拔，高度集成化，走线短（<5cm）但维护困难；(c)电芯片与光芯片集成模块——共封装光学（Co-Packaged Optics, CPO），光引擎与Switch ASIC/SerDes电芯片同封装（<5mm距离），走线极短，带宽密度最高但散热挑战极大（光芯片热密度>100W/cm²）。CPO是未来趋势（2025年后51.2T Switch将采用CPO），对SerDes要求：不再需要长距离SerDes channel，功耗可降低>30%（省去CTLE/DFE重定时器），但光模块与电芯片的联合设计/测试/热协同设计复杂度极高。

图 1.6 图 1.6 光模块：(a)边沿可插拔光模块；(b) 嵌入式光模块；(c)电芯片与光芯片集成模块

🔍 深度说明：本图展示近封装光学（NPO，Near Package Optics）结构，属于CPO与传统可插拔光模块之间的过渡方案。绿色PCB为系统主基板，灰色芯片为交换ASIC（可能是Broadcom Tomahawk4或类似51.2T交换芯片），光引擎（右侧小模组）紧邻ASIC贴装，光接口从PCB侧边阵列引出。相较于前面板可插拔（SerDes走线25~65cm），NPO方案SerDes走线缩短至2.5~12.5cm，高频损耗降低70%以上，可支持112G/224G PAM4而无需Retimer重定时器，功耗比可插拔方案低15~30%。同时保留了光模块的独立可维修性——光引擎故障可单独更换，无需更换整个交换ASIC，这对CPO而言是不可能的。当前NPO主要采用柔性板（Flex）实现光引擎和PCB之间的高频电连接，柔性板走线需控制100Ω差分阻抗一致性，避免阻抗失配引入的码间干扰。对SerDes设计而言，NPO方案下SerDes仍集成在交换ASIC内，但通道数可从64路扩展到128路以上，总带宽可达25.6T~51.2T，适配下一代高密度交换需求。

2.1 研究背景¶

随着大数据、云计算、第五代移动通信(5G)以及人工智能等应用的快速发展，全球通讯数据量急剧增长。据Gartner预测，到2024年全球数据总量将达到123 ZB。数据流量需求导致更多大型数据中心建设，数据中心之间长距离高速传输链路需求迫切。

电互连面临的瓶颈： - 随着CMOS器件特征尺寸缩小，芯片间互连延时和功耗不断增加 - 电互连的传输带宽与互连截面大小成正比，与长度平方成反比 - 预计到2022年芯片的I/O功耗为250 fJ/bit - 集成密度限制导致焊盘和引脚数趋于饱和

光互连的优势： - 宽频带、抗电磁干扰、强保密性 - 低传输损耗、低功耗、低时延 - 可轻松通过波分复用达到100+ Gbps传输速率

2.2 硅光子技术发展现状¶

  • 国际进展：Intel 2010年实现50 Gbps硅基光传输芯片，2013年提高至100 Gbps；IBM投资20亿美金开展光子集成技术研究

  3. 高速光调制器设计问题：如何设计支持NRZ 25 Gbps及PAM-4 50 Gbps的硅基电光调制器

3.2 封装结构设计¶

收发一体封装方案特点：

  1. 光学部分：
  2. 光纤、光芯片(调制器/PD)、光栅放置在金属座上
  3. 避免PCB热应力导致光纤漂移

  4. 光学器件高度约750 μm，与PCB板高度匹配

  5. 电学部分：

  6. 采用有机材料基板(成熟工艺、低成本、良好高频特性)
  7. 电芯片(Driver/TIA/CDR)采用Flip-Chip封装
  8. CDR采用BGA封装通过焊球与有机基板互连

  9. 无源器件通过SMT贴装

  10. 散热设计：

  11. 光学与电学共用铜基(Copper base)基座
  12. 有机基板上设计热孔(Thermal vias)

  13. 涂布热界面材料(TIM)

  14. 组装工艺：

  15. 先组装电学部分
  16. 光芯片和电学部分共同安装在铜基上
  17. 最后进行光纤耦合(保证光耦合精度)

3.3 链路仿真系统¶

仿真平台：Lumerical Interconnect + ANSYS HFSS

链路模型关键公式：

  1. 调制器输出光功率： Pout = ½Pin[1 + cos(φ₀ + L·Δφ)]e^(-αL)

  2. 光调制幅度(OMA)： OMA = |I₁ - I₀| = Pin·e^(-αL)·|sin(L·Δφ)|

  3. 最优有源区长度： Lopt = (1/Δφ)·arctan(Δφ/α)

  4. 消光比(ER)： ER = 10·log₁₀[(1+sin(L·Δφ))/(1-sin(L·Δφ))]

  5. 自由频谱范围(FSR)： FSR = λ²/(nog·ΔL)

  6. 互连线3dB带宽经验公式： BW = 0.35/Trise

  7. 上升时间关系： Trise²,out = Trise²,in + Trise²,0

4.1 光电协同设计方法¶

创新点1：光电协同优化设计

  • 通过链路误码率仿真，确定调制器最优有源区长度
  • 仿真分析激光器光功率、探测器响应度、驱动信号峰峰值对链路性能的影响
  • 从系统层面验证封装结构能否满足指标要求

关键发现： - 调制器有源区长度在2300 μm附近时OMA取极大值 - 有源区长度在2500 μm附近时系统误码率最低(6E-12) - 有源区长度超过2800 μm后消光比开始减小

4.2 低成本封装结构¶

创新点2：简化封装工艺

  • 采用高速PCB与光芯片引线键合封装
  • 电芯片采用Flip-Chip封装形式
  • 光纤与光芯片采用垂直耦合
  • 总功率仅3.6 W，平均每比特功耗<20 pJ

封装结构电学性能： - 传输线3dB带宽：23.61 GHz (2.5cm标准测试线) - 40 GHz时最长传输线损耗约1.4 dB - 键合线(550 μm)插入损耗约-1.3 dB - 键合线(800 μm)插入损耗约-1.8 dB

4.3 新型PAM-4调制器¶

创新点3：非对称式四脉冲幅度调制器

在传统马赫-曾德尔调制器基础上： - 减小一个调制臂的有源区长度 - 用普通波导替代 - 具有结构简单、调制方式简单、带宽高、插入损耗低、光调制幅度大等优点

4.4 调制器设计优化¶

PN结结构设计优化： - 折射率优化 - 光吸收系数优化 - 本征带宽优化

行波电极设计： - 特征阻抗匹配(50 Ω) - 带宽优化 - 损耗控制

6. 技术优势与局限性¶

  1. 光电协同设计方法：通过系统级仿真实现了光电器件参数的协同优化，避免了单一器件优化导致的系统性能瓶颈

  2. 低成本封装方案：

  3. 采用有机材料基板，工艺成熟、成本低廉
  4. 简化组装工艺，提高生产效率

  5. 散热设计合理，热管理性能良好

  7. 误码率<1E-12，满足商用标准

  11. 支持NRZ 25 Gbps和PAM-4 50 Gbps
  12. 新型非对称式PAM-4调制器结构简单

6.2 局限性¶

  1. 光栅耦合效率：
  2. 发送端单光栅耦合损耗约-2 dB(可接受)
  3. 接收端单光栅耦合损耗约-5 dB(质量欠佳)

  4. 收发端光栅距离较近导致接收端耦合受影响

  5. 封装结构对性能的影响：

  6. 封装结构使误码率从5.9E-27恶化到2.537E-16
  7. 信号上升沿从12.5 ps退化到18 ps

  8. 信号抖动从0.8 ps增加到1.2 ps

  9. 调制器消光比限制：

  10. 1500 μm调制器消光比5-17 dB

  11. 需进一步优化以满足更高要求

  12. 传输距离限制：

  13. 当前设计主要针对1000-2000 m短距传输
  14. 更长距离需要光放大器等辅助器件

9.1 设计公式汇总¶

调制器核心公式：

  1. 光功率输出公式： Pout = ½Pin[1 + cos(φ₀ + L·Δφ)]e^(-αL)

  2. OMA最优化公式： Lopt = (1/Δφ)·arctan(Δφ/α)

  3. 消光比公式： ER = 10·log₁₀[(1+sin(L·Δφ))/(1-sin(L·Δφ))]

  4. FSR公式： FSR = λ²/(nog·ΔL)

  5. 等离子色散效应(1550nm)： Δn = -[8.8×10⁻²²ΔNe + 8.5×10⁻¹⁸(ΔNh)⁰·⁸] Δα = 8.5×10⁻¹⁸ΔNe + 6.0×10⁻¹⁸ΔNh

  6. 带宽经验公式： BW = 0.35/Trise

  7. 上升时间合成公式： Trise²,out = Trise²,in + Trise²,0

9.3 设计方法论借鉴¶

  1. 光电协同设计流程： 系统指标定义 → 链路建模 → 参数扫描仿真 → 器件选型 → 封装设计 → 系统验证 → 迭代优化

  2. 调制器优化策略：

  3. OMA与消光比需平衡考虑
  4. 有源区长度存在最优值，非越长越好

  5. PN结结构与行波电极需协同设计

  6. 封装设计原则：

  7. 阻抗连续性控制在10%以内
  8. 高速信号键合线尽量短

  9. 热设计与机械稳定性综合考虑

  10. 仿真验证方法：

  11. Lumerical Interconnect进行光电链路仿真
  12. ANSYS HFSS进行电磁仿真
  13. TDR和VNA进行实测验证

9.4 参考文献中的关键结论¶

  • 传输线本征上升时间应小于输入信号上升沿的50%，以保证输出波形上升时间仅比输入大10%
  • 互连线的3dB带宽应至少是输入信号有效带宽的2倍
  • 光栅耦合损耗约2-5 dB，需优化以提高效率
  • Driver输出信号经过封装结构后上升沿/下降沿会变缓，抖动增加

总结¶

本论文系统研究了高速硅光子模块封装关键技术，提出了光电协同设计方法，设计了低成本、低功耗的4×25 Gbps光收发模块封装方案，完成了硅基马赫-曾德尔调制器的设计与仿真。实测结果表明，在1 km传输距离下误码率可达1E-12以下，系统总功耗仅3.6 W，平均比特功耗<20 pJ，为数据中心等高速光互连应用提供了有价值的参考方案。
