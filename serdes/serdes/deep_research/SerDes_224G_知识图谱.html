SerDes 224G 知识图谱¶

整理日期: 2026/04/23 知识基础: 14 个深度研究报告 + 路线图 9 个任务全部完成 用途: 全局视野建立、快速参数查询、技术决策索引

一、核心参数全局表¶

1.1 速率演进时间线¶

2019  56G NRZ/PAM4   (IEEE 802.3cd)   — 模拟 CDR+DFE 成熟
2020  112G PAM4       (IEEE 802.3ck)   — ADC-Based 过渡期
2024  224G PAM4       (IEEE 802.3dj D2.4) — 工程落地关键期
2027+ 448G PAM4(?)    (预标准研究)       — 探索中

关键分界点:
  56G: 纯模拟架构仍可工作
  112G: 强混合架构(ADC+数字DFE)成为必须
  224G: 全数字均衡+强ADC是唯一可行路径

1.2 标准参数速查¶

标准 调制 波特率 UI Nyquist 信道损耗 FEC 主要形态
802.3cd 100G PAM4 26.56 GBd 37.65 ps 13.3 GHz 35 dB RS(544,514) KP4 KR4/CR4
802.3ck 200G PAM4 53.125 GBd 18.82 ps 26.6 GHz ~25 dB RS(544,514) KP4 KR4/CR4
802.3dj 200G/lane PAM4 106.25 GBd 9.41 ps 53.1 GHz ~20 dB RS(544,514) KP4 KR1/CR1
OIF CEI-224G PAM4 112 GBd 8.93 ps 56 GHz < 10 dB (XSR) 可选 XSR/VSR

1.3 ADC 选型矩阵¶

架构 速率 ENOB 功耗效率 SerDes 适用性
Flash > 100 GS/s 4-6 bit 极差 实验室概念验证
SAR 1-10 GS/s 6-10 bit 最佳 56G 及以下
Pipeline 1-5 GS/s 8-14 bit 中等 中分辩率中速
TI-SAR 10-100 GS/s 6-8 bit 良好 112G/224G 主流

224G 推荐: 8-16× TI-SAR, ENOB ≥ 6 bit, 采样率 64-128 GS/s

1.4 均衡器架构对比¶

架构 延迟 功耗 线性度 ISI 处理 DFE 错误传播 224G 适用性
CTLE 极低 低 优秀 中等 无 必须前置
DFE (1-tap) 1 UI 低 优秀 强 有(短) 基础配置
DFE (N-tap) N UI 中 优秀 极强 有(长) 标准 RX
MLSE/SMLSE 可变 高 优秀 最强 无 推荐(224G)
DNN (AI) 可变 高 良好 极强 无 未来探索

224G 架构选择: CTLE (前置) + 4-tap DFE + SMLSE (次级)

1.5 CDR 架构对比¶

架构 类型 抖动性能 功耗 适用速率 224G
Hogge 线性 中 中 ≤ 28G ✗

Mueller-Muller 非线性 中 低 ≤ 28G △
PLL (Ring VCO) 线性 中 中 56-112G △
PLL (LC VCO) 线性 低 高 56-224G ✓

224G 推荐: LC-PLL + Bang-bang PD 混合架构

二、技术决策树¶

2.1 架构决策流程¶

速率 ≤ 56G?
├── NRZ → 纯模拟 CDR + CTLE + 1-tap DFE
└── PAM4 → 模拟 CDR + CTLE + N-tap DFE
    │
速率 56-112G?
├── 短距(XSR/VSR) → 轻量 ADC (ENOB≥5) + 数字 DFE
└── 中长距(MR/LR) → ADC-Based (ENOB≥6) + 全数字均衡
    │
速率 ≥ 224G?
├── ADC: TI-SAR (ENOB≥6.5, 8-16×, 采样率>100GS/s)
├── 均衡: CTLE + 4-tap DFE + SMLSE
├── CDR: LC-PLL + Bang-bang
└── FEC: KP4 (必须)

2.2 信道-均衡器联合决策¶

信道损耗 ≤ 10dB @ Nyquist (XSR)
  → 轻量 CTLE + 1-tap DFE 足够
  → ADC ENOB 5bit 即可

信道损耗 10-20dB @ Nyquist (VSR/MR)
  → 强 CTLE + 4-tap DFE
  → ADC ENOB 5.5-6bit

信道损耗 20-30dB @ Nyquist (LR/背板)
  → CTLE + 4-tap DFE + MLSE/SMLSE
  → ADC ENOB 6.5bit 必须
  → 必须 KP4 FEC

> 30dB → 系统级联 + 中继器，或改变拓扑

三、224G Gap 风险图¶

3.1 四大核心 Gap¶

Gap 风险等级 根本原因 缓解路径
ADC: ENOB 不足 P0 功耗-精度-速率三元悖论 强 DSP 均衡补偿，或降额使用
ADC: 输入带宽不足 P0 CMOS 采样开关 fT 限制 SiGe/BiCMOS 工艺，或sub-sampling
DFE: 关键路径延迟 P1 12-18ps vs UI 9.41ps speculative DFE，feed-forward DFE
CDR: 环路带宽 P2 56GHz Nyquist 抖动传递 双环 CDR，LC VCO + 分数分频

3.2 PAM4 眼图与 ENOB 关系¶

PAM4 三个判决门限:
  Level 0 → Level 1: 判决门限 Vth1 = -0.67 × Vref
  Level 1 → Level 2: 判决门限 Vth2 = +0.67 × Vref

眼高 ≈ 2 × Vref / 3 (理想)
实际眼高 = 理论眼高 - 噪声 RMS - DFE 残留 ISI

要求 ADC ENOB ≥ 6.5bit 的推导:
  量化噪声 = Vref / (2^(N+1))
  眼高 ≈ 2 × Vref / 3
  要求: 量化噪声 < 眼高 / 10 (约 1dB 代价)
  → N ≥ 6.5bit ✓

四、工程 Checklist¶

4.1 TX 设计指标¶

□ 波特率: 106.25 GBd (224G)
□ 调制: PAM4
□ TX FFE: 4-tap (1 pre + 1 main + 2 post)
□ 输出摆幅: 符合 802.3dj 眼图模板
□ RJ: < 0.1 UI RMS (目标)
□ DJ: < 0.15 UI (目标)
□ TDECQ: < 3.4 dB (DR8-2)
□ 杂散抑制: > 40 dBc
□ 电源噪声: < 1 mV RMS (注入噪声要求)

4.2 RX 设计指标¶

□ ADC: TI-SAR, ≥ 6.5bit ENOB, ≥ 100GS/s
□ CTLE: ≥ 15 dB 增益 @ Nyquist
□ DFE: 4-tap, speculative 实现
□ MLSE (可选): SMLSE 4 状态软判决
□ CDR: 双环架构, 抖动容限 > 0.4 UI @ 10MHz
□ 灵敏度: < 50 mVpp (PAM4 @ BER=1e-4)
□ COM: > 3 dB (802.3dj 要求)

4.3 系统级指标¶

□ FEC: RS(544,514) KP4 前向纠错
□ 容许信道损耗: ≥ 25 dB @ 26.6 GHz
□ 光模块形态: DR1/FR1/LR1 或 CPO
□ 热设计: TDP 20-25W, θJA < 10°C/W (配散热器)
□ 可靠性: 1000h HTOL @ 125°C, 等效 15 年寿命
□ 生产测试: BERT 224Gb/s, 示波器 > 50GHz
