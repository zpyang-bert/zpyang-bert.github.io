摘要¶

互联网流量的爆发式增长，由多媒体、物联网和云服务驱动，持续推动着高速电互连的带宽扩展和I/O的 aggressive 扩展。最近的标准要求收发器在NRZ（非归零）调制下工作于25~28Gb/s，或在PAM-4（脉冲幅度调制-4）调制下工作于50~56Gb/s。信道损伤会显著降低工作于如此高速率的收发器的信号完整性。为确保链路可靠性，均衡对于补偿信道损伤是必要的。此外，为以更低功耗实现目标BER（误码率），在最近的收发器设计中已采用了精确且节能的均衡技术。

在接收器设计中，连续时间线性均衡器（CTLE）凭借其低功耗优势展现出其优势。本博士工作专注于模拟均衡技术的研究，并提出了三种分别针对25Gb/s NRZ、56Gb/s PAM-4和112Gb/s PAM-4的不同设计。首先，提出了一种基于横向架构的新型CTLE。得益于横向架构，它表现出高精度补偿符号间干扰（ISI）的特性，并具有适应可变速率和信道配置的灵活性。该CTLE在28nm FD SOI工艺中实现，在5Gb/s至25Gb/s的速率下对20dB损耗信道进行了测量。核心功耗为17mW（1V电源），在BER=10^-12时的水平眼图张开度大于50%，与先前报道的针对类似数据速率和信道损耗的均衡器相比表现优异。

最近，为满足更高的带宽需求，每通道工作于50~56Gb/s的收发器已被提出。由于用于25-28Gb/s NRZ的信道损伤通过增加带宽显著限制了串行链路数据速率，因此提出了双倍带宽效率的PAM-4调制来提高数据速率。本文介绍了一种工作高达64Gb/s的全模拟PAM-4接收器。接收器均衡依赖于灵活的CTLE，可以独立地在低频、中频和高频进行最佳自适应，提供非常精确的信道传递函数反演。CTLE满足CEI-56G-VSR标准的性能要求，无需DFE（判决反馈均衡器）实现。该测试芯片在28nm FD SOI工艺中实现，在最大速率下接收器从1V电源消耗180mA，仅相当于2.8mW/Gb/s。

正在研究用于100Gb/s信号传输的电接口以满足带宽需求的持续增长。在更新的以太网标准IEEE 802.3ck中，规定了一通道100Gb/s接口。本文介绍了一种在7nm FinFet工艺中设计的112Gb/s PAM-4模拟前端。目标是与基于ADC的接收器合并，以利用7nm FinFet工艺中的高性能数字信号处理（DSP）。然而，晶体管行为的显著变化、缩放的电源电压和非常不同的布局规则给模拟电路设计带来了挑战。本文介绍了7nm FinFet工艺中关于模拟电路线性度和带宽的设计考虑。仿真结果证明，模拟前端可以成功恢复通过15dB Synectic信道传输的112Gb/s PAM-4序列。

在服务器内部，如图1.5所示，有多个ASIC通过印制电路板（PCB）和背板处理、传输和接收数据。服务器内部有三种不同的高速通信接口：（1）芯片到芯片端口侧接口，连接到在发射器和接收器两侧都有重定时器的端口侧模块；（2）芯片到模块（PCB板）直连接口，直接连接到端口侧模块，其电输入或输出接口不包括重定时器；（3）芯片到背板接口，在机箱中驱动聚合高速信号通过有能力的背板或中板[4]。

2.1.1 NRZ调制¶

图2.1 NRZ信号

Vo：幅度 Tb：比特持续时间，也称为单位间隔（UI） 1/Tb：比特率

图2.2 NRZ序列x(t)

如图2.2所示，NRZ序列x(t)可以表示为：x(t) = Σk bk p(t-kTb)，其中bk=±Vo/2，p(t)是矩形脉冲函数。因此，信号x(t)是时移矩形脉冲（p(t)）与比特（bk）乘积的和。

x(t) = p(t) * Σk bk · δ(t-kTb)（*是卷积）

定义： y(t) = Σk bk · δ(t-kTb) k

x(t)的功率谱密度（PSD）可以表示为： Sxx(f) = |P(f)|^2 Syy(f)

其中P(f)是p(t)的傅里叶变换： P(f) = Tb [sin(πfTb)/(πfTb)]

如果我们假设"1"和"0"具有相等概率，y(t)的PSD为： Syy(f) = bk^2/Tb

x(t)的PSD为： Sxx(f) = 4Vo^2 |P(f)|^2/Tb

Sxx(f) = (Vo^2/4) Tb [sin(πfTb)/(πfTb)]^2

2.1.2 PAM-4调制¶

随着数据速率已达到56Gb/s或更高，背板信道的频率依赖性损伤变得有问题[7]。因此，如图2.4所示的多电平脉冲幅度调制PAM-4，与NRZ相比具有两倍的频谱效率，已被提出。一个PAM-4符号在一个单位间隔（UI）内包含2比特，这与PAM-4信号频谱占据PAM-2信号带宽的一半相关（如图2.5）。因此，PAM-4信号的奈奎斯特频率是NRZ信号的一半。

图2.4 PAM-4信号

与具有相同奈奎斯特频率的NRZ信号相比，PAM-4信号使数据密度加倍，在信号电平方面实现更高分辨率。在相同采样率系统中，与相同数据速率的NRZ信号相比，PAM-4信号在更宽的频率上分散总噪声功率，因此带宽内的噪声功率降低。然而，考虑到最大信号幅度受电源电压约束，PAM4信号的电平之间的间隔是NRZ信号的1/3。因此，PAM-4承受约20log10(1/3) = 9.5dB的SNR损失。

2.2 信号质量¶

2.2.1 符号间干扰（ISI）¶

符号间干扰（ISI）是一种数据相关的干扰形式，其中一个符号干扰先前和后续符号。ISI通常由通过有限带宽的色散信道传输的多个脉冲引起。脉冲超出其分配时间间隔的扩展导致其与相邻脉冲干扰（如图2.7所示）。系统中ISI的存在会降低信噪比，从而在接收器输出的判决器件引入误差。因此，在发送和接收滤波器的设计中，目标是最小化ISI的影响，从而以尽可能小的误差率将数字数据传送到目的地。

图2.7 ISI示例

在通信中，奈奎斯特准则定义了通信信道传输和接收无ISI信号的最小带宽。奈奎斯特理想信道定义为：

P(f) = -1/2W, -W < f < W 0, |f| ≥ W

p(t) = sinc(2Wt)

W = 1/2T = BR/2

这是一个使用"sinc"脉冲的数据序列（如图2.8），比特率为BR（比特/秒）。W称为奈奎斯特带宽，等于BR/2。然而，"sinc"脉冲不是因果的，只能在实践中近似。有几种近似方法，如"升余弦"脉冲或"方"脉冲。BR/2的奈奎斯特频率为近似"sinc"实现的最小带宽提供了良好参考。

图2.8 奈奎斯特理想信道

2.2.2 抖动¶

抖动定义为信号边沿相对于其在时间中理想位置的变化（如图2.9所示）。在串行通信系统中，抖动会影响定时裕度和同步。通常有两类抖动：随机抖动和确定性抖动。随机抖动（RJ）由器件噪声引起。它是无界的，假定服从高斯分布。通常以均方根（rms）值指定。确定性抖动（DJ）是有界的，具有明确定义的最小和最大范围。它通常以峰峰值（pk-pk）表示。确定性抖动可进一步分类为：周期性抖动（PJ）；数据相关抖动（DDJ）；和边界非相关抖动（BUJ）。PJ可定义为测量和标称周期之间的时间差。PJ由时钟或其他可调制传输边沿的周期性源引起。

DDJ是与数据流中比特序列相关的抖动。DDJ通常由信道非理想性（ISI）和占空比失真（上升和下降时间的差异）引起。BUJ通常由相邻链路的串扰引起。BUJ有界是因为有限的耦合强度，并且是非相关的，因为它与相邻"攻击"信道相关但与"受害"信道不相关。由RJ和DJ组成的总体抖动可组织成如图2.10所示的抖动树。

2.2.3 信号质量测量¶

数据眼图用于表征高速信号源并检查信号完整性。眼图是通过将数字波形的所有对应于各个比特的部分折叠成一个短间隔构成的（如图2.11所示）。

图2.11 NRZ数据眼图

眼图直观地评估有线通信系统中的带宽、衰减、抖动、上升/下降时间变化和噪声裕度。垂直和水平眼图张开度可用于量化信号质量。我们考虑眼图的两种条件，有和无非加性高斯噪声和随机抖动。在第一种情况下，通常发生在电路级设计，眼图张开度可以直接找到（如图2.12所示）。垂直眼图张开度在采样瞬间（在眼中心）测量，可归一化为全眼高度（不包括过冲或下冲）。水平眼图张开度在阈值电平测量，可归一化为比特间隔。垂直眼图闭合由符号间干扰（ISI）[6]决定，水平眼图闭合由确定性抖动（包括数据相关抖动和占空比失真）决定。

图2.12 眼图眼张开度

在第二种情况下，考虑到带有随机噪声的信号，这在实践中发生并在测量中观察到，我们应该用统计信息定义眼图张开度。我们可以在眼图中定义具有相同误码率（BER）的等高线。BER可由信噪比（SNR）[6]确定。SNR可从峰峰值幅度Vo和噪声均方根电压σ找到：SNR = Vo/2σ，BER = Q(Vo/2σ)。（Q(x)是误差函数，可在[6]中找到）。因此，我们可以在眼图中定义恒定BER等高线。通过扫描采样瞬间和判决阈值，在眼图中可以找到相同BER点。如图2.13所示，这些相同BER点构成恒定BER等高线。如果我们在给定BER的等高线内做判决，得到的BER小于等高线。在有噪声和随机抖动的系统中，我们使用眼图裕度而不是眼图张开度来评估信号质量。对于给定BER，更大的眼图裕度意味着判决阈值和采样瞬间更大的设计裕度。

图2.13 眼图中BER等高线

2.3 信道特性¶

不同应用需要不同类型的信道。信道特性强烈依赖于其类型。背板信道（如图2.15所示）用于板对板连接，是本论文的目标框架。背板信道是包含用于扩展板连接（插槽）的印制电路板，允许所有连接的板之间通信。

图2.15 背板信道[60]

如图2.15所示，背板走线可视为PCB传输线，用于将信号从发送器移动到接收板。PCB传输线由两个导体组成：信号走线和通常为地平面的返回路径。两者之间的体积由PCB介电材料组成。PCB中通常有两种基本类型的信号传输线互连：微带线和带状线。

图2.16 微带线

图2.16显示了微带线模型。它包括用于信号的单个导电走线和提供信号返回路径的导电地平面。它们由称为基底的介电层分离。因为走线没有屏蔽，所以容易受到与相邻信号线的串扰影响。

图2.17 带状线

图2.17显示了带状线。与微带线相反，信号的走线位于PCB的内侧。走线在每侧由PCB介电层隔离，然后在顶部和底部放置用于信号返回路径的导电地平面。由于这种屏蔽结构，带状线对串扰具有更强的鲁棒性，常用于背板。

背板信道中有几种损伤会在每秒几吉比特以上的数据速率降低信号完整性。背板信道中的频率依赖性损伤、反射和串扰将在以下章节讨论。

2.3.1 频率依赖性损伤¶

为了说明背板信道的频率依赖性色散和衰减，使用分布元件模型来表示PCB传输线。如图2.18所示，分布元件模型用于通过无限级联的RLGC段表示信道，其中R、L、G和C分别是单位/长度值（分别为Ω/m、H/m、S/m和F/m）。

图2.18 传输线分布元件模型

传输线方程： ∂V(x,t)/∂x = -RI(x,t) - L∂I(x,t)/∂t

∂I(x,t)/∂x = -GV(x,t) - C∂V(x,t)/∂t

时谐形式： d²V(x)/dx² = γ² V(x)

d²I(x)/dx² = γ² I(x)

时谐传输线方程的解： V(x) = Vf0 e^(-γx) + Vr0 e^(-γx)

I(x) = If0 e^(-γx) + Ir0 e^(-γx)

其中Vf0、Vr0分别是前向和反射波的电压，If0、Ir0分别是前向和反射波的电流，γ是传播常数： γ = α + jβ = √(R+jωL)(G+jωC)

其他有意义的参数是传输线特性阻抗：Zo = V(x)/I(x) = √(R+jωL)/(G+jωC)和信号相速度：v = ω/β

有损传输线引入的第一个损伤是由于信号相速度的频率依赖性。后者可以近似为v ≈ (1/√LC)[1 + (1/8)(R/ωL)² + (1/8)(G/ωC)²]^(-1)。频率依赖性导致信号在沿线传播时形状色散。

对于低损耗条件，我们可以假设R/ωL、G/ωC <<1。传播常数可简化为γ = αR + αD + jβ，其中αR ≈ R/2Zo是电阻损耗，αD ≈ GZo/2是介质损耗。

电阻和介质损耗引起衰减。此外，当R和G是频率依赖的时，衰减也是频率依赖的。以下段落提供了更详细的分析。

图2.19 矩形导体的趋肤效应

PCB传输线中的导电层遭受趋肤效应，即交流电流密度J在导体中从表面值JS按深度d指数下降，如下[8]：

J(d) ≈ JS e^(-d/δ)

其中δ称为趋肤深度。矩形导体如图2.19所示。趋肤深度δ是电流密度相对于导体表面下降e^(-1)的厚度：

δ = √(ρ/πμf)

其中ρ和μ是导体的电阻率和磁导率。一个重要的参数是临界频率fs，定义为趋肤深度等于导体高度一半的频率：

fs = ρ/πμ(h/2)²

再看图2.19，在低频ffs时，R可近似为R = 2wδ = RDC √(fs/f)。在该区域，传输线传播常数中的电阻损耗可表示为：

αR = R/(2Zo) √(f/fs) = (RDC/2Zo) √(f/fs)

介质损耗αD是由于交流电场导致介质原子旋转并以热量形式吸收信号能量引起的。这种损耗与沿线路传输的信号频率线性正比，由损耗角正切tan(δD)量化，定义为：tan(δD) = G/ωC。传输线传播常数中的介质损耗可表示为：

αD = πf tan(δD) √LC

损耗角正切越低，介质损耗越低。表1.1显示了一些典型的PCB介质材料及其各自的tan(δ)。

表2.1 介质材料[8]

为了设计电子均衡器以补偿PCB线路缺陷，引入了一种结合电阻和介质损耗的总损耗曲线的闭合形式表达式[61]：

Loss(f) = exp[-ks L(1+j)√f - kd Lf]

其中L是线路长度，ks和kd分别是与趋肤损耗和介质损耗相关的系数。从上述方程可知，趋肤效应引起的电阻损耗随信号频率的平方根增加，而介质损耗随频率线性增加。因此，在低频时，电阻损耗占主导地位，而随着频率增加，介质损耗变得显著，趋肤效应可忽略。为了深入了解，图2.20描述了一个典型的信道传递函数，其中损耗分为两个贡献。

2.3.3 串扰¶

串扰是信道各组件之间耦合的不期望信号的量度，可以在时域和频域中测量。串扰发生在多通道串行链路中，由于连接器和PCB走线的隔离不良[11]。图2.27显示了两种类型的串扰：远端串扰（FEXT）和近端串扰（NEXT）。FEXT是指传播到电缆不同端的正向波，NEXT是指反向波，传播到与期望信号起始相同的端。值得注意的是，NEXT更关键，因为干扰信号不会被信道衰减。

图2.27 串扰

高速串行链路收发器使用差分信号来抑制共模干扰源。然而，在高数据速率下，串扰被部分抵消，因为实际差分对具有将共模干扰转换为差分模式的 asymmetries。

为了准确描述无源互连中串扰的限制，引入了一个新参数称为"插入损耗（无源信道频率响应）对串扰比"（ICR）。它是插入损耗与总串扰比（dB）的比值。单个NEXT干扰源（PSNEXT）的功率和可定义为：PSNEXT(f) = -10log(Σn 10^(-NEXTn(f)/10))，其中NEXTn(f)是近端干扰源n的串扰损耗（dB）。单个FEXT干扰源（PSFEXT）的功率和可定义为：PSFEXT(f) = -10log(Σn 10^(-FEXTn(f)/10))，其中FEXTn(f)是远端干扰源n的串扰损耗（dB）。单个NEXT和FEXT干扰源（PSXT）的功率和可定义为：PSXT = -10log(10^(-PSNEXT(f)/10) + 10^(-PSFEXT(f)/10))。图2.28显示了来自IEEE 802.3ap[12]的不同数据速率的ICR要求。

第3章 模拟均衡技术¶

摘要¶

连续时间线性均衡器是一种具有高频增益提升传递函数的峰化滤波器，可有效补偿信道的频率衰减和色散。与判决反馈均衡器（DFE）相比，连续时间均衡器需要更低的功耗和更小的芯片面积，因此它是低功耗高速串行接收器的有吸引力解决方案。本章介绍了CTLE的演进。为了提供精确的均衡，CTLE从单级演变为多级，并引入低频均衡器来消除由趋肤效应引起的残余ISI。作为传统CTLE的替代方案，提出了将信号分成多个路径的分支路径均衡器。由于分支路径结构，CTLE可以独立调谐直流增益和零点频率。这样可以更灵活地塑造传递函数。宽带技术有助于扩展CTLE的工作频率。本章介绍了在数十Gb/s收发器中广泛实现的电感峰化和负电容技术。

3.1 引言¶

为了补偿第2章中介绍的信道损伤，均衡是有线通信收发器中的关键部分。基于混合信号均衡的有线通信收发器框图如图3.1所示。通常，均衡器有三个主要类别：前馈均衡器（FFE）、连续时间线性均衡器（CTLE）和判决反馈均衡器（DFE）。FFE通常实现为有限脉冲响应（FIR）滤波器。它在发射器中实现，用于对发送到信道的信号引入预加重。如果嵌入在发射器中，信号本质上是数字的，FIR滤波器以时钟触发触发器作为延迟元件的混合信号方式实现。纯模拟FIR滤波器，没有任何时钟，也可以在接收器中实现，但设计连续时间延迟元件具有挑战性，需要大面积和功耗[16]。DFE是接收器中的非线性均衡器，在判决器件（ slicer ）之后有一个反馈FIR滤波器，从接收信号中恢复数字信息。DFE非常强大，可以在不引入噪声放大的情况下消除位于接收符号峰值远侧（后光标）的信号能量。但DFE需要相对较大的功耗和硬件复杂性，因此短距离链路中的几个接收器为节省功耗和面积仅使用CTLE实现[17]。传统上，CTLE由差分对实现，通过并联电阻-电容阻抗退化，引入零极点对以提升高频增益并补偿信道的高频损耗。然而最先进的CTLE从这种简单实现向更复杂的结构演进，例如分支路径方法[18]。CTLE始终存在并在塑造接收器传递函数中发挥主要作用。本章更深入地分析了CTLE特性，包括频域和时域。此外，为满足日益增长的数据速率要求，需要电路在越来越高的频率工作，本章还介绍了在最先进CTLE中实现的宽带电路技术。通常，仅使用CTLE的接收器能够补偿奈奎斯特频率高达20dB[25][28]的信道损耗。

3.2 CTLE工作原理与演进¶

最先进的有线通信收发器必须拥抱并指定更快的数据速率以满足通信基础设施日益增长的带宽需求。为了实现良好的品质因数（FOM），收发器采用CTLE作为低成本、低功耗选项。这推动了CTLE的持续演进，以支持更高速度和改善均衡性能。

传统CTLE由简单的差分对实现，具有电阻性和电容性Rs-Cs退化（如图3.2(a)所示），引入了传递函数中的零极点对（绘制在图3.2(b)）。传递函数可表示为：

Vout(s)/Vin(s) = (gmRL)(1+s/ωz)/(1+s/ωp1)(1+s/ωp2)

其中ωz = 1/RsCs，ωp1 = 1/RsCs，ωp2 = 1/RLCL。通过调谐Rs和Cs，可以移动频率中的零极点对。此外，直流增益或等效的高频提升也由Rs改变。如果接收器输出端的期望信号幅度固定，则需要可变增益放大器（VGA）来补偿CTLE的直流增益变化。

分支路径均衡器将信号分成多个路径，可以作为退化差分对的替代方案[21]。图3.10显示了一个简单的分支路径，其中一个路径包括高通滤波器以放大高频分量，另一个路径是全通滤波器或低通滤波器以匹配第一路径的时间延迟。不是调谐退化差分对中的Rs和Cs，而是使用可变增益放大器（VGA）A0和B0的增益来改变直流增益和该均衡方案中的零点位置。该架构提供了独立改变直流增益和零点位置的选项。此外，通常用跨导实现的VGA的分辨率可以比通过修整Rs和Cs实现的更精细。因此，使用该结构可以实现更精确的传递函数塑形，针对信道特性的倒数。高级分支路径均衡器在图3.11中以分段方式塑造传递函数。模拟均衡器由三个不同路径组成，以创建期望的频率响应。两个带通滤波器后面跟着可变增益放大器（VGA），允许在f1和f2的独立增益控制。可以在f1和f2频率分别调整增益，放松自适应引擎以获得最佳调谐。VGA增益A0为低频数据提供路径。带通滤波器由具有RLC负载的差分对实现，包括允许调谐中心频率的变容二极管。VGA使用由开关控制的电阻退化的差分对实现。分支路径均衡器支持零点位置的独立调谐，这提供了一种在多零点系统中灵活塑造传递函数的方法[22]。

3.3 宽带技术¶

随着模拟均衡器的工作频率增加，宽带技术对于扩展增益级的带宽是必要的。有多种扩展带宽的方法，本章将介绍电感峰化和负电容抵消技术。

3.3.1 电感峰化¶

随着单片电感的出现，电感峰化技术已在集成电路中变得可行。想法是允许限制带宽的电容与电感共振，从而提高速度。当然，共振必须以最小的峰化和过冲发生，以提供对随机数据的良好行为响应。

并联峰化

并联峰化是一种带宽扩展技术，其中与负载电阻串联的电感连接到输出电容的旁边。图3.13显示了这项技术的简化原理图。将晶体管视为小信号依赖电流源，增益就是晶体管跨导gm和被动负载阻抗Z的乘积。因为晶体管跨导在频率上近似恒定，所以这里只考虑阻抗。对于图3.13b所示的并联峰化网络：

Z(s) = (RL+sLd)/(1+sRLC) / (Lfixed+sLdCfixed)

电感L引入一个零点，使阻抗随频率增加而上升，抵消电容的下降阻抗，从而扩展3dB带宽。带宽增加的一个等效解释是下降时间减少。也就是说，电感拒绝电流流过电阻支路，因此响应输入阶跃，更多电流最初对输出电容充电，减少下降时间。

定义参数m = RL²Ctotal/Ld，ω = 1/√(LtotalCtotal)。并联峰化网络可表示为：

Z(s) = (1+s/mω0)/(1+s/mω0 + s²/mω0²)

图3.14绘制了不同m值下的归一化电压增益。m=1.41给出最大带宽扩展，m=2.41给出最平坦响应，m=3.1给出最平坦相位的响应。

串联峰化

串联峰化可添加到并联峰化结构以实现串联-并联峰化。额外的串联峰化将系统转换为二阶滤波器，进一步改善带宽扩展。此外，电压增益的滚降比单独并联电感峰化更陡。作为缺点，串联峰化放大了放大器的延时。

3.3.2 负电容¶

为了减少给定节点上电容限制带宽的影响，可以向该节点添加呈现负电容的有源网络。差分信号路径特别适合这种方法。

负电容可以用具有纯电容退化的交叉耦合差分对实现。图3.16中晶体管漏极处的阻抗为：

1/Z = sC1 + (1 + sCgs/gm)/(Cgs/C1+2)(1/gm)

在远低于晶体管ωT的角频率（ω<<ωT=gm/Cgs）处，分子中的第二项可忽略不计，得到的阻抗由等于-C1的负电容与等于-(Cgs/C1+2)(1/gm)的负电阻串联组成。

图3.17显示了一种典型布置，使用该技术来（部分）抵消节点X和Y处的电容。为完全抵消，需要2C1 = Cx = Cy。然而，如果由于失配2C1 > Cx，则交叉耦合对可能变成弛豫振荡器。可以证明振荡条件由gmRL ≥ (Cx-C1)/2[6]给出。电路可能仍然表现出显著振铃，因此即使没有振荡也会引入ISI。因此，交叉耦合对通常仅用于Cx和Cy的部分抵消，特别是如果需要小的ISI。

3.4 均衡器自适应概述¶

为了针对变化的信道、工艺-电压-温度（PVT）和不同数据速率优化均衡，必须实现自适应循环来调谐均衡器。图3.18[21]说明了均衡器自适应的一般概念。在自适应循环中，通过测量均衡器性能质量生成误差信号，然后将调谐信号发生器将误差信号转换为调谐信号以调整均衡器的系数。

几种基于不同误差信号计算方法的自适应算法已在先进模拟均衡器中实现。在论文[25]中，通过测量低频和高频分量的功率谱密度实现频谱平衡技术。在论文[63]中，自适应算法中的误差信号通过比较低频和高频平均幅度计算。在论文[64]中，自适应循环中包含眼图监视器以记录输出波形的数据边沿分布，算法目标是最小化数据边沿的标准偏差。

另一种流行算法是误差信号通过比较恢复信号与参考信号的均方误差（MSE）来计算的算法，在最近发表的均衡器中广泛实现[21]。图3.19显示了MSE算法的一般概念。如果有N个均衡器变量p = [p1 p2 ... pN]^T，成本函数JMSE(p)定义为：JMSE(p) = E[(y-d)²] = E[e²]。由于测量信号d和y的统计信息来计算平均平方误差是不实际的，主要实现是最小均方（LMS），它使用瞬时值而不是期望值。成本函数JLMS(p)定义为：JLMS(p) = (y-d)² = e²。对于状态信号不相关的均衡器（如横向滤波器），成本函数曲面是保证单峰的（如图3.20[30]）。因此，梯度下降搜索LMS算法可应用于此类均衡器。均衡器的变量更新为p(k+1) = p(k) - μ·∇JLMS(p)，其中μ是决定收敛速度的常数。

第4章 28nm CMOS工艺灵活横向连续时间线性均衡器工作速率高达25Gb/s¶

摘要¶

本章介绍了一种灵活的横向连续时间线性均衡器。背板串行链路在25Gb/s及以上的收发器需要高精度和灵活性以匹配信道响应的高均衡器。为了满足高精度要求，本章提出了一种具有横向架构的非常灵活的连续时间线性均衡器（CTLE）。在第3章中，展示了在分支路径均衡器中独立调谐增益和零点的概念。类似于这种方法，所提出的均衡器具有可变直流增益和两个可独立调谐的零点。横向架构使其与梯度下降算法（如LMS）兼容，允许最佳自适应增益和零点频率位置，提高均衡精度。该CTLE在28nm CMOS工艺中实现，在5Gb/s至25Gb/s的速率下对20dB损耗信道进行了测量。核心功耗为17mW（1V电源），在BER=10^-12时的水平眼图张开度大于50% UI，与先前报道的针对类似数据速率和信道损耗的均衡器相比表现优异。

4.1 引言¶

正如第1章介绍的，由多媒体和云服务驱动的互联网流量的爆发式增长，持续推动着高速电互连的带宽扩展和I/O的 aggressive 扩展。确保链路可靠性和低功耗是芯片上I/O密度增加背景下的关键设计目标。最近的标准已被引入以支持高达25-28 Gb/s的NRZ信号调制，并在先进CMOS工艺节点中构建的收发器已得到证明[20, 23]。在这种速度下，长度为几十厘米的典型背板在奈奎斯特频率引入超过30dB的信道损耗，因此由于大的符号间干扰（ISI），信道均衡的改进是强制性的。均衡器必须灵活，能够在不同信道上以可变速度工作，并持续自适应以对抗导致缓慢变化的信道传递函数的环境漂移（温度和湿度）。接收器端的常见做法是将简单的连续时间线性均衡器（CTLE）与判决反馈均衡器（DFE）结合使用。CTLE已从简单的高通滤波器演变为具有传递函数中多个零极点对的更复杂拓扑，以实现更准确的信道损耗补偿[20]。考虑到典型信道传递函数的形状，衰减特性的斜率随频率增加，范围在10dB/dec到30 dB/dec之间。为了获得准确的信道反演，需要独立调谐CTLE的零点。模拟调谐技术已在最近的自适应均衡器中提出[24-28]。然而，这些技术无法独立控制均衡器的多个零点，因此不适合分别优化多个CTLE零点的频率。

图4.1 横向CTLE框图及模块H(s)原理图（单端信号用于框图便于阅读）

本工作描述了一个具有如图4.1所示的横向架构的CTLE，具有可变增益和多个零点的传递函数。横向配置允许CTLE使用梯度下降算法（如LMS）进行自适应，从而能够最佳调谐增益和零点频率位置以进行精细均衡。该均衡器已在[29]中发表。本章第4.2节描述了CTLE架构，第4.3节介绍了电路设计考虑，第4.4节显示了测量结果和比较。从实验结果来看，在28nm CMOS中实现的均衡器，在不同长度背板上以5Gb/s至25Gb/s的可变数据速率恢复了约20dB损耗。在BER=10^-12时的水平眼图张开度大于等于50%，核心功耗为17mW（1V电源）。实验结果与先前报道的针对类似数据速率和信道损耗的均衡器相比表现优异。

4.2 均衡器架构与设计¶

图4.1中均衡器的横向路径由具有传递函数H(s)的电容退化差分对组成。跨导器连接在输入、中间和输出抽头处，输出电流在共享电阻负载中求和以提供输出信号d̂(t)。每个差分对理想地引入原点中的一个零点，并假设极点足够高，H(s)≈τs，其中τ=RDC。因此，均衡器的近似传递函数为：

Heq(s) = c0 + c1τs + c2(τs)² = c0(1 - sτz1)(1 - sτz2) (4.1)

其中 τz1,2 = τ(-c1 ± √(c1² - 4c0c2))/2c0 (4.2)

公式4.1证明了均衡器具有可变直流增益和具有时间常数τz1和τz2的两个零点。从公式（4.2），零点可以通过改变跨导增益（c0..c2）或时间常数τ=RDC来移动频率。

图4.2 均衡器与20dB损耗背板信道级联时25Gb/s下的MSE曲面

由于横向架构，CTLE响应是控制系数（c0..c2）的线性组合。结果，考虑到均方误差，MSMSE = E[(d̂(n) - d(n))²]（其中d̂(n)是理想输出信号，d(n)是均衡器输出信号），作为自适应的性能标准，MSE曲面是一个抛物面[30]，具有规则形状和全局最小值。为了深入了解，图4.2显示了与c1和c2（假设c0=1）相比的MSE曲面，使用Matlab在25Gb/s数据速率下模拟，考虑了20dB损耗背板信道模型。对于1V0pk的眼图幅度，最小MSE为-26dB（最小MSE点的眼图作为插图报告在图4.2中）。考虑各种背板信道模型获得了类似结果。多亏了图4.2中曲面的规则形状，可以使用梯度下降算法实现自适应，通过使用接收器中常用的眼图监视器电路进行MSE估计来实现收敛到最佳系数集。横向均衡器架构还允许通过添加比较器（在该芯片中未实现）来实现快速收敛符号LMS算法，以感测横向路径中间节点处的均衡器状态信号[30]。Matlab模拟证实了使用两个位于不同频率的零点来改善均衡的重要性。实际上，对于单个零点（c2=0）或两个相同频率的零点（c1=2√c0c2），最小MSE从-26dB分别上升到-14dB和-16dB。

最后，值得注意的是，公式（4.1）给出的相同传递函数可以通过级联两个具有分流RC源极退化的差分对来实现，这是第3章讨论的CTLE的常见电路拓扑[24, 25]。直流增益和零点位置可以通过调谐退化电阻器和电容器来控制，但均衡器响应与控制参数不是线性的。结果，MSE曲面是不规则的，可能有局部最小值（如图4.3所示），使得梯度下降的自适应更加困难，不能确保收敛到最佳结果。此外，无法访问均衡器的内部状态信号排除了简单实现LMS算法。

均衡器设计针对28nm CMOS工艺，最大数据速率为25Gb/s。图4.1中的两个电容退化级被设计为近似理想传递函数H(s)=τs。忽略电感峰化的效果，传递函数可表示为：

其中极点为 ωp1 = gm1,2/C ωp2 = 1/CLRD

4.5 结论¶

在博士课程的第一部分开发了一种具有横向架构的连续时间线性均衡器。均衡器传递函数具有可变直流增益和两个零点，而横向架构简化了自适应，允许分别最佳调谐增益和零点频率位置，以提高均衡精度。在28nm CMOS测试芯片上的测量证明了在5至25Gb/s数据速率下跨20dB损耗信道的>50%水平眼图张开度，核心功耗为17mW。

第5章 28nm CMOS FDSOI工艺64Gb/s收发器PAM-4模拟前端¶

摘要¶

介绍了一种在28nm CMOS FDSOI中用于短距离电互连、工作高达64Gb/s的收发器PAM-4模拟前端。模拟前端包括可变增益放大器（VGA）和具有低、中、高频信道损耗补偿的灵活CTLE。VGA结合了两种不同拓扑，提供高线性度和平坦频率响应。接收器均衡仅基于灵活CTLE，证明通过可在低、中、高频独立最佳自适应的传递函数实现非常精确的信道反演。CTLE满足CEI-56G-VSR标准的性能要求，无需DFE（判决反馈均衡器）实现。因此，数据和边沿采样路径中的比较器定时约束可以通过使用跟踪保持级来放松，节省功耗。完整收发器（TX、RX和时钟生成）在PAM-4中从16Gb/s工作到64Gb/s，在NRZ中从8Gb/s到32Gb/s。在64Gb/s时，跨16.8dB损耗信道的TX到RX链路达到10^-12最小BER，在BER=10^-6时达到0.19UI水平眼图张开度，功耗为5.02mW/Gb/s。

5.1 引言¶

物联网、多媒体点播、云存储和云计算等数字密集型服务的持续增长，推动电信基础设施和数据中心持续升级，以支持指数级网络流量增长。解决更高通信速度需求的新一代电互连标准，在从每通道28Gb/s到56Gb/s及以上的迁移路径中引入了PAM-4（112Gb/s项目正在进行中）[33]。

图5.1 56Gb/s链路应用空间[33]

业界正在努力开发完整的56Gb/s PAM-4收发器[7, 34-37]，112Gb/s的构建模块正在研究中[38][39]。与NRZ相比，PAM-4中的每个符号携带两倍的信息，因此理论上将频谱占用限制在50%。更有效地利用可用链路带宽，配合降低的时钟频率和CMOS技术的持续演进，应该能够提高链路速度，同时限制整体系统成本和归一化到比特率的功耗。但是，与NRZ相比，PAM-4收发器的设计带来了许多新的挑战和权衡。PAM-4的固有1/3眼图幅度导致SNR损失，具有有限上升和下降时间的非相邻电平之间的转换减少了水平眼图张开度[40]。因此，PAM-4发射器需要提供最大信号摆幅和非常宽的等效带宽[41-44]。此外， transceiver构建块的线性度对避免四个PAM电平的失真和保持信号完整性极为关键。均衡也变得更加苛刻。事实上，多电平信号由于通道损耗和反射而增加敏感性，因为最小的转换（即相邻电平之间）受到由3倍大的峰峰值转换[45]产生的符号间干扰（ISI）的损害。这对应于与NRZ相比 ISI 影响增加3倍，因此在符号检测之前需要更精细的信道均衡。最后，收发器必须符合遗留组件，支持宽范围的数据速率和降低速度的NRZ信号，同时保持功率效率。

图5.1描述了56Gb/s电接口设想中的应用空间。考虑了非常不同的场景，从与同一封装内安装的芯片之间的超短链路（可忽略的互连损耗和反射），到长距离链路，其中电接口必须应对长达1米长的信道（背板或电缆）和由多个封装和连接器产生的严重反射。长距离链路的恶劣操作条件正在推动向基于ADC的接收器迁移，其中复杂灵活的均衡和符号检测由数字信号处理（DSP）执行[46-48]。在16nm FinFet工艺中实现的56Gb/s PAM-4接收器，在14GHz奈奎斯特频率下证明了在30-35dB信道损耗下工作，归一化功耗（不包括DSP功耗）在4.4-6.6mW/Gb/s范围内[7][36][37]。考虑到DSP功耗，[37]中报告的总RX功耗高于8mW/Gb/s。为了提高减少信道损耗和反射的链路的能源效率（例如，用于芯片到芯片或芯片到模块互连的中等或超短距离链路（图5.1）），提出了功率可扩展的基于ADC的接收器[36][37]。但对于此类应用，模拟PAM-4接收器可能提供更高的功率和面积节省[41][49]。模拟接收器效率的一个主要限制因素是判决反馈均衡器（DFE）的实现。与NRZ相比，PAM-4需要硬件三倍化和提高分辨率，在低功耗下满足关键DFE反馈定时也随着先进CMOS工艺节点而增加挑战[41]。

本章介绍了一种用于工作高达64Gb/s的收发器的PAM-4模拟前端。模拟前端的CTLE具有在低、中、高频分别最佳自适应的传递函数，允许以余量满足CEI-56G-VSR场景的性能要求。由于不需要DFE，放松的定时约束允许使用跟踪保持级实现数据、边沿和眼图监视器检测，大大降低了功率要求。在64Gb/s时，完整接收器需要180mW（来自单1V电源），仅相当于2.8mW/Gb/s。完整收发器在PAM-4中从16Gb/s工作到64Gb/s，在NRZ中从8Gb/s工作到32Gb/s。在64Gb/s时，跨16.8dB损耗信道的TX到RX链路证明了10^-12最小BER，在BER=10^-6时为0.19UI水平眼图张开度，功耗为5.02mW/Gb/s（包括RX、TX和时钟生成）。

5.3.2 连续时间线性均衡器¶

图5.7 连续时间线性均衡器电路原理图(a)及反馈均衡器调谐(b)

所提出的CTLE由三个独立控制的级组成，以通过在低、中和高频分别灵活塑造整体传递函数来精确匹配信道响应的倒数。

电路原理图绘制在图5.7(a)中。RC退化差分对在传递函数中引入一个零点，通过可编程退化电容CS在频率上移动。其目的是通过引入高达12dB的峰化来补偿信道在~1-10GHz频率范围的介质损耗。第一级中的前馈路径，由RC网络与跨导级串联（R2-C2和gm2）组成，在低频添加温和的~1.5-2dB峰化，零极点对可以通过调谐R2从0.2GHz移动到1GHz。该级在低频细化CTLE传递函数，其中趋肤效应损耗在信道频率响应中产生温和的滚降[19][20]。R2和Cs都使用迭代算法调谐，利用眼图监视器执行测量以同时最大化垂直和水平眼图张开度。迭代算法是基于扫频R2和Cs的所有组合的启发式算法。

CTLE的最后一级使用反馈拓扑实现，引入额外的高频提升（高达6dB）以在奈奎斯特频率附近精细恢复信道特性的陡峭滚降。忽略并联峰化电感L3，电路分析得到从gm3输入到CTLE输出的以下传递函数：

HHF(ω) = (gm3R3)(1+jω/ωf)/[GLOOP(1+jω/ω1)(1+jω/ω2)] (5.1)

其中GLOOP=gm5R3gm4RF是静态环路增益，ωf=1/RFCF。公式（5.1）中的两个极点ω1,2由下式给出：

ω1,2 = (ωf + ωp ± √(ωf² + ωp² - 2ωfωp(1+2GLOOP)))/2 (5.2)

其中ωp=1/R3CP是gm3输出节点处的寄生极点角频率。为确保宽裕的稳定性，并最小化由过度群延迟变化引起的失真，该级的设计是使环路增益足够低，使得从（5.2），ω1,2为实数。在这种情况下，式（5.2）对GLOOP的依赖可以线性近似为：

ω1 ≈ ωf(1 + (ωp/(ωp-ωf))GLOOP) (5.3)

ω2 ≈ ωp(1 + (ωf/(ωp-ωf))GLOOP)

从（5.1）和（5.3），该级的峰化，即max(|HHF(ω)|/|HHF(ω->0)|)，通过降低低频增益和将两个极点推向更高频率由GLOOP控制。假设ωp足够高，使得ω2 >> ω1，max(|HHF(ω)|/|HHF(ω->0)|) = 1+GLOOP。反馈拓扑的主要优点是HHF(ω)中零点的位置是恒定的。结果，可以通过GLOOP在高频率实现CTLE传递函数的非常选择性控制，对低和中频的影响可以忽略，大大简化了CTLE自适应。反馈均衡器的调谐绘制在图5.7(b)中。导致输出眼图幅度变化的中频增益变化由CTLE之后的VGA2补偿。此外，反馈拓扑允许HHF(ω)在正常RX操作期间使用LMS算法连续自适应，通过获取CTLE输出信号作为梯度信息并使用眼图监视器进行误差切片[30]。

通过考虑56Gb/s短距离链路[52]的公开可用参考信道进行仿真，证明了所提CTLE的均衡精度（如图5.8所示）。仿真遵循在RX上实现的自适应序列。首先，通过编程退化电容Cs找到中频处的最佳CTLE响应。得到的传递函数与信道响应的倒数一起报告在图5.8a中。相应最佳配置的眼图显示在图5.8b中。然后，通过作用于R2在低频调谐传递函数。相应的CTLE传递函数报告在图5.8c中，优化后的眼图显示在图5.8d中。最后，高频提升被自适应（图5.8e），在奈奎斯特频率附近精细反演信道特性。得到的眼图报告在图5.8f中。完成CTLE自适应后，水平和垂直眼图张开度分别为0.42UI和39mV（超过200mVpp眼图幅度）。CEI-56G-PAM4标准要求10^-6的原始BER，对应于高斯分布约9.5σ。图5.8中眼图中心的框表示由模拟前端噪声（σn≈2.4mVrms）和时钟生成电路（σJ≈290fs）引入的随机抖动引起的采样不确定性。通过积分仿真中噪声的功率谱密度（时钟生成电路中的相位噪声）来估计噪声和随机抖动的RMS值。CTLE自适应后的眼图张开度（图5.8f）满足目标BER，有足够的余量留给其他收发器损伤。

第6章 7nm FinFet工艺112Gb/s接收器PAM-4模拟前端¶

摘要¶

本章介绍了一种在7nm FinFet中用于工作高达112Gb/s的PAM-4接收器的模拟前端。它包括一个可变增益放大器（VGA）和一个具有低、中和高频信道损耗补偿的灵活CTLE，然后是一个缓冲器。特别注意设计高线性度电路，这是使用0.9V低电源电压保持PAM-4信号完整性的关键方面。在CTLE中利用T-coil峰化和电容中和将工作频率扩展到28GHz以上。模拟前端可与基于ADC的接收器后端合并用于短距离链路或长距离链路。仿真使用参考信道（Synectic信道）在奈奎斯特频率具有15dB损耗。结果证明了该模拟前端能够成功支持112Gb/s PAM信号。

6.1 引言¶

正如第1章介绍的，IP流量的持续增长推动了支持更高速度的串行链路收发器的开发。最近，在IEEE 802.3ck以太网标准中规定了一通道100Gb/s和四通道400Gb/s物理层，并且正在研究112Gb/s串行链路收发器，以双56Gb/s收发器的数据速率并提高功率效率[53]。数字电路极大地受益于16nm或7nm FinFet工艺中的晶体管缩放[54]，这样速度提高而功耗降低同时芯片面积缩小。结合模拟电路和数字系统的收发器开发也可以利用数字电路的更高性能。然而，在先前节点（如28nm FD SOI）中开发的传统模拟电路，由于晶体管行为的显著变化、新的复杂布局规则和建模技术，不能保证在16nm/7nm FinFet工艺中正常工作[55]。FinFet工艺中的模拟设计挑战也源于较低的工作电压、通孔和fin后布局的严重影响以及金属线允许电流密度的显著降低。

因此，有必要在FinFet工艺中开发和优化新的模拟电路。应该在模拟电路中引入更复杂的宽带技术，以满足不断提高速度的带宽需求。此外，缩放的电源电压也损害了电路的线性度，因此应仔细考虑以避免大信号失真。本工作中的PAM-4模拟前端设计为与基于ADC的接收器合并。三个主要设计目标是：（1）在奈奎斯特频率提供约10dB提升以放松ADC的有效位数（ENOB）[56]，（2）驱动ADC所呈现的大输入电容负载，以及（3）保持输出信号幅度为ADC的全量程范围（FSR）。完整接收器的目标是恢复通过长距离信道传输的112Gb/s PAM-4信号。ADC目前正在开发中，撰写时无法模拟完整收发器。本章提供的仿真仅包括模拟前端，测试平台由PRBS31最坏情况信号发生器、在奈奎斯特频率具有15d损耗的Synectic信道[61]以及后面跟着建模ADC负载的模拟前端组成。

图6.1 完整ADC接收器架构

完整系统架构如图6.1所示。接收器的模拟部分包括T-coil输入网络、模拟前端和缓冲器。T-Coil补偿来自焊盘、ESD保护器件和VGA输入级的寄生电容。包含VGA、CTLE和缓冲器的模拟前端提供低功耗均衡并设置输入摆幅以匹配ADC的FSR。7b 56-GS/s ADC是32路时间交织（TI），将差分模拟输入转换为数字信号。然后ADC输出被重新定时并发送到包括FFE和DFE的DSP。模拟前端架构和电路设计分别在第6.2节和第6.3节介绍，仿真结果在第6.4节总结，最后，第6.5节介绍结论和未来工作。

介绍了一种在7nm FinFet中设计的PAM-4模拟前端，用于与基于ADC的接收器合并，工作高达112Gb/s。架构中使用简单的VGA电路拓扑来限制大信号失真。在CTLE中实现了多种宽带技术以扩展峰化频率。仿真结果证明了模拟前端能够在112Gb/s数据速率下工作，通过在奈奎斯特频率具有15dB损耗的Synectic信道传输的PRBS 31最坏情况序列。

第1章介绍了串行链路通信和高速串行链路收发器的背景。介绍了由全球IP流量持续增长推动的串行链路通信的带宽需求。讨论了开发带有CTLE的低功耗接收器的动机。

在第4章中，介绍了一种具有横向架构的非常灵活的CTLE。均衡器传递函数具有可变直流增益和两个零点，而横向架构简化了自适应，允许分别最佳调谐增益和零点频率位置，以提高均衡精度。测试芯片在28nm CMOS FD-SOI中实现。测量结果证明了在20dB损耗信道上以5至25Gb/s数据速率的>50%水平眼图张开度和17mW核心功耗。

在第5章中，介绍了一种在28nm FDSOI-CMOS中支持工作高达64Gb/s的PAM-4模拟前端以及完整接收器。合并的VGA拓扑在线性度和减少的不期望频率峰化方面表现出良好性能。提出了一种新的CTLE电路拓扑，具有高灵活性和精度来匹配信道响应。CTLE满足CEI-56G-VSR链路的均衡要求，允许实现大部分模拟RX，无需DFE，从而节省大量功耗。测量测试在16.8dB损耗信道的TX到RX链路上进行。在64Gb/s时，完整收发器达到10^-6 BER，0.19UI定时裕度，完整接收器仅需2.8mW/Gb/s。

在第6章中，介绍了一种用于在7nm FinFet中工作高达112Gb/s的基于ADC的接收器的PAM-4模拟前端。架构中使用简单的VGA电路拓扑来限制大信号失真。实现了多种宽带技术以扩展峰化频率。仿真结果证明了模拟前端能够在112Gb/s数据速率下工作，通过在奈奎斯特频率具有15dB损耗的Synectic信道传输的PRBS 31最坏情况序列。

7.2 未来工作¶

对更高速度串行链路的需求是不可阻挡的。快速增长的云服务正在推动更高速度数据中心的开发[14]。用于下一代800Gb以太网的每通道>100Gb/s串行链路目前正在开发中。对于7nm FinFet中的112Gb/s PAM-4模拟前端，后布局提取后的性能优化是最近的目标。由于许多布局相关效应影响带宽和线性度，预期原理图和布局之间需要进行一些迭代以调整前端的最终响应。

串行链路收发器的物理层正在向先进FinFet工艺节点迁移。基于ADC接收器的收发器已在长距离信道（>30dB损耗）中显示出强大的性能[57]。然而，ADC仍然是一个功耗大的组件。对于短距离链路（<20dB损耗），全模拟解决方案也可以节省功率。在16nm/7nm FinFet工艺中，模拟设计受到缩放电源电压、Fin的寄生和布局相关效应的约束。为了实现紧凑的全模拟接收器，应该开发一些新的电路拓扑。例如，用短沟道长度晶体管堆叠替换长沟道长度晶体管以制作电流尾可以节省面积并减少寄生电阻[55]。

第3章中介绍的具有横向架构的非常灵活的CTLE尚未应用于56Gb/s或112Gb/s PAM-4接收器。原因是高分辨率VGA的线性度对PAM-4系统的影响比对NRZ系统更大。为了结合这种CTLE架构，探索保持VGA高线性度的新技术是值得的。
