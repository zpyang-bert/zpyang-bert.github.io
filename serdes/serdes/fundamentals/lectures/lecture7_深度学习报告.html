ECEN720 Lecture 7: Equalization Introduction & TX FIR Equalization — 深度学习报告¶

课程: ECEN720: High-Speed Links Circuits and Systems 授课人: Sam Palermo, Texas A&M University 整理日期: 2026/04/16 研究深度: 深度解读（结合均衡理论、TX FIR、RX CTLE/DFE、MMSE系数选择、电路拓扑）

一、均衡理论基础¶

1.1 为什么需要均衡¶

信道损伤:
• 高频衰减(趋肤效应、介质损耗)
• 频率选择性好
• 导致符号间干扰(ISI)

均衡目标:
┌──────────────────────────────────────────────┐
│ 均衡 = 信道的"逆"                             │
│                                              │
│ • 展平频率响应至Nyquist频率                    │
│ • 消除时域ISI                                │
│ • 恢复眼图张开度                              │
└──────────────────────────────────────────────┘

均衡器类型:
• TX FIR: 发射机预加重
• RX CTLE: 连续时间线性均衡
• RX DFE: 判决反馈均衡

1.2 均衡器类型对比¶

┌──────────────┬────────────┬────────────┬────────────┐
│ 类型          │ 位置       │ 优点        │ 缺点        │
├──────────────┼────────────┼────────────┼────────────┤
│ TX FIR       │ 发射机     │ 不放大噪声   │ 衰减低频   │
│              │            │ 5-6bit精度  │ 需反馈调优  │
├──────────────┼────────────┼────────────┼────────────┤
│ RX FIR       │ 接收机     │ 可放大高频   │ 放大噪声   │
│              │            │ 无需反馈信道  │ 模拟延迟难  │
├──────────────┼────────────┼────────────┼────────────┤
│ RX CTLE      │ 接收机     │ 低功耗面积   │ 1阶补偿    │
│              │            │ 可消除pre/post│ PVT敏感   │
├──────────────┼────────────┼────────────┼────────────┤
│ RX DFE       │ 接收机     │ 不放大噪声   │ 无法消除pre│
│              │            │ 自适应调优   │ 关键时序路径│
└──────────────┴────────────┴────────────┴────────────┘

🔍 深度说明： 【研究背景】均衡是高速SerDes系统设计中克服信道高频衰减和ISI的核心技术；不同均衡器类型（TX FIR、RX CTLE、DFE）各有优缺点，适用于不同信道条件和设计约束。 【核心结论】TX FIR不放大噪声但衰减低频（de-emphasis），精度可达5-6bit；CTLE低功耗但仅一阶补偿且PVT敏感；DFE不放大噪声且可自适应，但无法消除pre-cursor ISI且有关键时序路径。 【工程价值】高损耗信道（>20dB）通常需要TX EQ + RX EQ组合；短距离背板以TX EQ为主；设计时需根据信道特性和功耗预算选择均衡策略。 【落地注意】PAM-4系统中各判决电平的均衡策略可能不同，需要独立优化；TX FIR的预加重精度影响眼图质量，4-6bit DAC精度是商业实现的典型范围。

二、TX FIR均衡¶

2.1 TX FIR原理¶

时域FIR滤波器:
W(z) = w₀ + w₁z⁻¹ + w₂z⁻² + ...

其中:
• w₀: pre-cursor tap (补偿前标ISI)
• w₁: cursor tap (主信号)
• w₂: post-cursor tap (补偿后标ISI)

输出信号:
V_out[n] = Σ w_k · D[n-k]

其中D[n]为输入数据

设计原理:
• 预畸变发射脉冲
• 反送信道失真
• 代价: 衰减发射信号(de-emphasis)

2.2 TX FIR时域分析¶

10Gb/s示例：

W(z) = -0.131 + 0.595z⁻¹ - 0.274z⁻²

抽头系数:
w_pre  = -0.131
w_main =  0.595
w_post = -0.274

低频响应(所有抽头同符号):
[... 1 1 1 ...] ⊗ [-0.131 0.595 -0.274]
= [... 0.190 0.190 0.190 ...]
→ -14.4dB衰减

Nyquist频率响应(交替符号):
[... -1 1 -1 ...] ⊗ [-0.131 0.595 -0.274]
= [... 1 -1 1 ...]
→ 0dB (无衰减)

结论:
• DC衰减 -14.4dB
• Nyquist频率保持0dB
• 等效14.4dB频率峰化

2.3 TX FIR频域分析¶

频率响应推导：

令 z = e^(j2πfT_s)

W(f) = -0.131 + 0.595e^(-j2πfT_s) - 0.274e^(-j4πfT_s)

直流分量 (f=0):
z = 1
W(0) = -0.131 + 0.595 - 0.274 = 0.190
|W(0)| = -14.4dB

Nyquist分量 (f=1/(2T_s)):
z = -1
W(f_N) = -0.131 - 0.595 - 0.274 = -1.0
|W(f_N)| = 0dB

🔍 深度说明： 【研究背景】TX FIR通过预畸变发射脉冲来补偿信道频率选择性和ISI；时域和频域分析揭示了de-emphasis的本质——低频衰减而高频保持，从而实现等效频率峰化。 【核心结论】10Gb/s示例中，W(z) = -0.131 + 0.595z⁻¹ - 0.274z⁻²实现14.4dB的DC衰减和0dB的Nyquist响应；pre-cursor为负值用于补偿前标反射，post-cursor为负值用于补偿长尾ISI。 【工程价值】14.4dB的de-emphasis是高速SerDes典型配置；设计时需根据信道响应选择抽头系数；MMSE算法可以自动计算最优抽头值。 【落地注意】DC衰减意味着发射功率降低，需要更高驱动器摆幅来补偿；PAM-4的三个电平可能需要不同的de-emphasis配置；TX FIR抽头的有限精度（4-6bit）可能限制均衡效果。

2.4 MMSE系数选择¶

最小均方误差算法：

系统模型:
• y = H · W (信道脉冲响应与FIR抽头卷积)
• y_des = 期望输出(无ISI)

误差矩阵:
E = Y - Y_des = H·W - Y_des

误差范数:
||E||² = WᵀHᵀH·W - 2Y_desᵀH·W + Y_desᵀY_des

对W求导并设为零:
d(||E||²)/dW = 2HᵀH·W - 2HᵀY_des = 0

最优抽头:
W_ls = (HᵀH)⁻¹ · Hᵀ · Y_des

孤立脉冲均衡示例：

信道脉冲响应H:
• 5个前标采样
• 10个后标采样
• 3列对应3个均衡抽头

期望输出Y_des:
• cursor位置 = 1
• 其他位置 = 0

求解最优W:
W_ls = H⁺ · Y_des (伪逆)

归一化:
W_norm[n] = W_ls[n] / Σ|W_ls[i]|
→ 确保总功率不变

2.5 TX FIR抽头分辨率¶

有限精度影响：

理想抽头 (Refined Server Channel, 10Gb/s):
W_ls = [-0.818, 3.725, -1.718]

归一化后:
W_ls_norm = [-0.131, 0.595, -0.274]

实际TX DAC分辨率: 4-6 bits

量化影响:
• 映射这些均衡系数可能影响性能
• 需评估量化误差对眼图的影响

🔍 深度说明： 【研究背景】MMSE（最小均方误差）算法是计算TX FIR最优抽头的标准方法，通过伪逆运算找到使误差范数最小的抽头系数；信道脉冲响应的精确测量是算法有效的前提。 【核心结论】最优抽头W_ls = (HᵀH)⁻¹·Hᵀ·Y_des通过伪逆计算；归一化保证总功率不变；4-6bit DAC精度可能限制均衡效果，需评估量化误差影响。 【工程价值】MMSE系数选择是TX EQ设计的关键步骤；不同信道需要不同的抽头系数；设计时需要考虑抽头分辨率对量化误差的敏感性。 【落地注意】信道脉冲响应测量需要专用TDR设备；归一化后抽头系数可能不满足实际DAC的离散电平；设计时需考虑corner工艺下的均衡性能余量。

三、TX FIR电路架构¶

3.1 直接FIR架构¶

电流模式实现：

输出电压:
V_out[0] = (I_-1·D_1 + I_0·D_0 + I_1·D_-1 + I_2·D_-2) · (R_TERM/2)

其中:
• I_k: 各抽头电流权重
• D_k: 数据抽头(+1或-1)

4-tap FIR示例:
• Pre-cursor:  1/4 (25%)
• Cursor:      1x   (100%)
• 1st Post:   1/2  (50%)
• 2nd Post:   1/4  (25%)

3.2 分段DAC架构¶

优势：

分段DAC特点:
• 最小尺寸输出晶体管处理峰值电流
• 最低输出电容
• 最灵活均衡

代价:
• 最高功耗和复杂度
• 需要映射表(RAM)

设计权衡:
• 直接FIR: 较低功耗和复杂度
         但输出电容较高
• 分段DAC: 最低电容和最高灵活性
         但功耗和复杂度最高

3.3 电压模式TX FIR¶

电压模式挑战：

FIR在电压模式驱动器中更难实现:
原因: 串联阻抗

解决方案1: 输出分压器+接地分流
• 实现不同电压电平
• 维持阻抗控制
• 缺点: 预驱动器逻辑复杂

解决方案2: 增加通道分流路径
• 恒定电流从VREF电源抽取
• 缺点: 预驱动器逻辑更复杂

3.4 混合电压模式驱动¶

Song TCAS2 2012：

架构特点:
• 主驱动抽头: 电压模式
  → 给定电压摆幅下电流降低
• 预加重均衡抽头: 并行电流模式
  → 高分辨率预强调

优势:
• 结合两种驱动器优点
• 低功耗电压模式主抽头
• 灵活电流模式均衡

注意:
• 有一定动态电流变化
• 但比原始VM TX FIR #1小

3.5 阻抗调制均衡¶

阻抗调制原理：

信令功率随de-emphasis变化:
• 转换位(transition bit): 50Ω阻抗
• 连续位(run-length > 1): 更高阻抗

实现:
• 分段实现
• 阻抗随数据模式动态变化
• 功率优化

3.6 均衡效能对比¶

不同信道表现：

2-tap EQ残余ISI对比:
• 反射显著信道(中trace背板): >20%额外残余ISI
• 阻抗受控信道(单板CPW): 几乎相同性能

结论:
• 对于有反射的信道，TX EQ效果更显著
• 对于阻抗受控信道，增益递减

🔍 深度说明： 【研究背景】TX FIR电路实现需要考虑功耗、复杂度、输出电容等多个因素；直接FIR和分段DAC是两种主要架构，各有权衡；电压模式TX FIR因串联阻抗问题实现更复杂。 【核心结论】直接FIR功耗低但输出电容高；分段DAC最低电容和最高灵活性但功耗和复杂度最高；混合电压模式驱动（Song TCAS2 2012）结合两者优点——电压模式主抽头低功耗，电流模式预加重高分辨率。 【工程价值】对于56Gb/s+ SerDes，分段DAC架构因其低输出电容优势成为主流选择；阻抗调制均衡通过动态调整终端阻抗优化功率效率。 【落地注意】分段DAC需要RAM存储映射表，增加设计复杂度；预加重均衡抽头的时序对齐要求严格；混合驱动器的动态电流变化可能影响电源噪声。

四、RX均衡技术¶

4.1 RX FIR均衡¶

优点:
• 有足够动态范围时，可放大高频内容
  (而非衰减低频)
• 可消除pre-cursor和超过滤波器范围的ISI
• 滤波器抽头系数可自适应调优，无需反馈信道

缺点:
• 放大噪声和串扰
• 模拟延迟实现困难
• 抽头精度限制

4.2 RX CTLE¶

架构:
• 差分输入: V+_in, V-_in
• 差分输出: V+_out, V-_out

传递函数:
H(s) ≈ g_m·R / (1 + s/ω_p)

其中:
• g_m: 跨导
• R: 负载电阻
• ω_p: 极点频率

优点:
• 低功耗低面积实现增益和均衡
• 可消除pre-cursor和长尾ISI
• 一阶补偿

缺点:
• 通常限于一阶补偿
• 放大噪声和串扰
• PVT敏感
• 可能难调

4.3 RX DFE¶

判决反馈均衡器结构:
┌────────────────────────────────────────────┐
│ 输入 → 前向滤波 → 判决 → 反馈滤波 → 减法器   │
└────────────────────────────────────────────┘

递归滤波:
y[n] = Σ w_k · x[n-k] - Σ h_k · y_hat[n-k]
        (前向)          (反馈)

其中:
• x: 输入信号
• w_k: 前向抽头
• h_k: 反馈抽头
• y_hat: 已判决输出

优点:
• 不放大噪声和串扰
• 抽头系数可自适应调优
• 无需反馈信道

缺点:
• 无法消除pre-cursor ISI
• 关键反馈时序路径
• ISI消除时序复杂化CDR相位检测

🔍 深度说明： 【研究背景】RX DFE通过已判决的输出反馈来消除残余ISI，是高速SerDes RX的核心技术；反馈结构不放大噪声是其主要优势，但无法消除pre-cursor ISI是固有局限。 【核心结论】DFE递归滤波公式y[n] = Σw_k·x[n-k] - Σh_k·y_hat[n-k]表明判决输出直接影响当前时刻均衡；关键时序路径（从判决到反馈）限制了最高数据率；pre-cursor ISI因在判决前发生而无法通过反馈消除。 【工程价值】对于高损耗信道，DFE通常与CTLE联合使用——CTLE补偿大范围衰减，DFE消除残余ISI；DFE抽头系数的自适应调优可以跟踪信道变化。 【落地注意】DFE关键时序路径的建立和保持时间要求严格，需要在 architectural 层面优化；判决错误会通过反馈路径传播，导致错误传播；需要设计误差检测和校正机制。

4.4 DFE架构与时序¶

Tap延时结构：

单抽头DFE时序:
┌──────────────────────────────────────────────┐
│ D_in → [DFF] → Σ → Decision → [DFF] ──┐    │
│                      ↑                   │    │
│                      └──── h₁ ←──────────┘    │
└──────────────────────────────────────────────┘

关键路径:
• Data path: D_in → DFF → Σ → Decision → DFF (output)
• Feedback path: Decision → DFF → Σ (h₁ contribution)

时序约束:
• T_UI - t_DFF - t_Sigma - t_Decision > t_Setup
• 反馈路径必须在下一UI前完成

多抽头DFE：

级联结构:
• Tap 1: h₁延迟1UI
• Tap 2: h₂延迟2UI
• 级联结构减少关键路径

代价:
• 延迟链增加面积和功耗
• 长尾ISI需要更多抽头

🔍 深度说明： 【研究背景】DFE的反馈时序路径是高速实现的瓶颈，单抽头DFE需要在每个UI内完成判决和反馈两个操作；多抽头DFE通过级联结构分担延迟但增加复杂度。 【核心结论】单抽头DFE的关键路径约束为T_UI - t_DFF - t_Sigma - t_Decision > t_Setup；56Gb/s时UI约17.9ps，时序非常紧张；多抽头结构通过延长反馈路径来缓解关键路径压力。 【工程价值】对于56Gb/s+ NRZ/PAM-4，DFE架构选择直接影响时序收敛难度；unrolled DFE结构（并行比较器替代串行反馈）在高速下更常用但面积功耗更大。 【落地注意】DFE时序收敛需要在逻辑综合和布局布线阶段仔细优化；反馈路径的时钟树插入延迟需要精确控制；工艺波动可能影响不同corner下的时序余量。

五、均衡效能总结¶

5.1 均衡效能观察¶

实验观察:
1. 2-tap TX EQ提供最初的大性能提升
2. 仅TX EQ时，2-tap和4-tap差异不大
3. RX均衡，特别是DFE，允许进一步性能提升

关键警告:
• 由于关键时序路径，快速DFE难以构建
• 需要仔细时序约束分析

5.2 均衡策略选择¶

信道特性 → 均衡策略:

高损耗信道:
• 需要TX EQ + RX EQ组合
• CTLE补偿大范围衰减
• DFE消除残余ISI

有反射信道:
• TX EQ预畸变更有效
• 需要更长的pre-cursor EQ

阻抗受控信道:
• 短抽头TX EQ可能足够
• DFE增益递减
