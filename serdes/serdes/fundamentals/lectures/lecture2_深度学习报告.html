ECEN720 Lecture 2: Channel Components, Wires & Transmission Lines — 深度学习报告¶

课程: ECEN720: High-Speed Links Circuits and Systems 授课人: Sam Palermo, Texas A&M University 整理日期: 2026/04/16 研究深度: 深度解读（结合传输线理论、RLGC模型、阻抗匹配、信道损耗机理）

一、信道组件体系¶

1.1 高速链路信道全貌¶

封装SerDes系统结构：

┌──────────────────────────────────────────────────────────────────┐
│                     高速电气链路信道完整路径                         │
├──────────────────────────────────────────────────────────────────┤
│  Tx IC → 封装 → 板级传输线 → 连接器 → 背板 → 连接器 → 板级传输线 → 封装 → Rx IC │
│   Pkg    Trace    Connector   Backplane   Connector    Trace     Pkg    │
└──────────────────────────────────────────────────────────────────┘

各组件对信号完整性的影响：
• IC封装: 键合线电感(~0.7-1nH/mm)、封装走线电容
• PCB走线: 特性阻抗(Z₀=50Ω)、插入损耗
• 连接器: 阻抗连续性、串扰(16-32对/10mm)
• 背板: 多层结构(30+层)、过孔stub
• 过孔: 阻抗不连续、stub效应

1.2 IC封装类型与演进¶

封装类型 引脚数 特点 适用场景
SOP (Small Outline) 8-56 成本低 低速外设
QFP (Quad Flat Package) 64-304 四面引脚 微控制器
PBGA (Plastic BGA) 256-420 球栅阵列 消费电子
EBGA (Enhanced BGA) 352-896 增强散热 高性能
FC-BGA (Flip-Chip BGA) 1089-2116 倒装芯片、最低阻抗 SerDes前端

🔍 深度说明： 【研究背景】该图展示了从SOP到FC-BGA的封装演进路径。倒装芯片(Flip-Chip)封装消除了传统wirebond的长键合线，显著降低寄生电感，是GHz级SerDes的首选。 【核心结论】FC-BGA封装的键合线电感可降至~0.1nH（vs wirebond的~1nH），封装走线电容也可降低50%以上，这对10Gb/s以上信号传输至关重要。 【工程价值】封装选择直接影响SerDes性能：PBGA适用于低成本产品，FC-BGA适用于高端交换机、AI芯片等对信号完整性要求严苛的场景。 【落地注意】FC-BGA封装需要 substrate routing design，当前国内先进封装产能有限；封装选型时需综合考虑性能、成本、供应链稳定性。

封装键合方式对比：

标准Wirebond封装:
• 键合线电感: ~1nH/mm
• 封装走线电容: ~20fF/mm
• 寄生效应显著

Flip-Chip封装:
• 芯片直接通过焊球连接封装基板
• 无长键合线，寄生电感大幅降低
• 散热效率更高
• 适合GHz级高速I/O

封装模型参数:
┌────────────────┬────────────────┐
│ 参数           │ 数值范围        │
├────────────────┼────────────────┤
│ 键合线电感L     │ 0.7-1nH/mm     │
│ 封装走线电感L   │ ~1nH/mm        │
│ 互感K          │ 存在耦合       │
│ 走线电容C      │ 20fF/mm        │
│ 层间耦合电容C   │ 40fF/mm        │
└────────────────┴────────────────┘

1.3 PCB叠层与走线¶

PCB堆叠结构：

典型4-8层PCB叠层:
┌─────────────────┐
│ Signal Layer 1  │ ← 外层信号
├─────────────────┤
│ GND Plane       │
├─────────────────┤
│ Power Plane     │
├─────────────────┤
│ Signal Layer 2  │ ← 内层信号(带状线)
├─────────────────┤
│ GND/Power Plane │
├─────────────────┤
│ Signal Layer 3  │ ← 外层信号(微带线)
└─────────────────┘

背板可达30+层

铜厚规格:
• 信号层: "0.5oz" (17.5μm)
• 电源层: "1oz" (35μm)

微带线 vs 带状线：

微带线(Microstrip):
• 位于PCB外层，表面走线
• 一侧介质(RoG4)，一侧空气
• 优点: 布线灵活、耦合到参考平面
• 缺点: 对外界干扰敏感、串扰较大
• 特性阻抗公式: Z₀ ≈ (60/√εr)·ln(4h/w)

带状线(Stripline):
• 夹在两个平行地平面之间
• 介质均匀包围
• 优点: 隔离性好、抗串扰
• 缺点: 需要多层PCB
• 特性阻抗公式: Z₀ ≈ (60/√εr)·ln(2b/w)

1.4 连接器与过孔¶

连接器设计挑战：

连接器密度: 16-32对差分/10mm

关键设计要求:
• 维持差分阻抗连续性(Z₀=50Ω差分)
• 串扰控制
• 插损控制
**Via Stub效应(背板设计关键)**：

![](lecture2_ee720_channels_images/img-029.jpg)

> 🔍 深度说明：
> 【研究背景】Via stub是背板设计中的关键问题。未背钻的过孔会形成等效谐振腔，在特定频率造成深达-40dB的谐振null，严重恶化信道性能。
> 【核心结论】Legacy背板默认使用直通过孔，stub长度可达数百mil，在10-20GHz造成显著插损；背钻(backdrill)技术可以消除stub，保留信号完整性。
> 【工程价值】背板设计时必须考虑via stub效应；对于长背板（>20英寸），需要背钻过孔并优化via antipad设计，以将谐振频率推至30GHz以上。
> 【落地注意】背钻工艺会增加制造成本（约$0.1/pin），但在高密度交换机等高性能场景中是必须的；选型时需评估成本vs性能权衡。

## 二、传输线理论基础

### 2.1 导线模型

**集总参数模型**：

R: 导线电阻 (Ω/m) L: 导线电感 (H/m) C: 对地电容 (F/m) G: 介质电导 (S/m)

不同导线的模型适用性: ┌──────────────────┬────────┬────────┬────────┬────────┐ │ 导线类型 │ R │ L │ C │ f_c │ ├──────────────────┼────────┼────────┼────────┼────────┤ │ AWG24 双绞线 │ 0.08Ω/m│ 400nH/m│ 40pF/m │ 32kHz │ │ PCB 走线 │ 5Ω/m │ 300nH/m│ 100pF/m│ 2.7MHz │ │ 芯片最小宽度M6 │ 40kΩ/m │ 4µH/m │ 300pF/m│ 1.6GHz │ │ (0.18µm CMOS) │ │ │ │ │ └──────────────────┴────────┴────────┴────────┴────────┘

f_c = R/(2πL) — 低于此频率用RC模型，高于此频率用LC/RLGC模型


### 2.2 RLGC传输线方程

**时域电报方程(偏导数形式)**：

![](lecture2_ee720_channels_images/img-056.ppm)

> 🔍 深度说明：
> 【研究背景】该图展示了传输线理论的电报方程，是分析高速信道的数学基础。方程描述了电压和电流在传输线上的传播特性，是所有SI（信号完整性）分析的理论起点。
> 【核心结论】传播常数γ = α + jβ = √[(R+jωL)(G+jωC)]，其中α为衰减常数、β为相位常数；特性阻抗Z₀ = √[(R+jωL)/(G+jωC)]。
> 【工程价值】在设计PCB传输线时，需要确保Z₀ = 50Ω（单端）或100Ω（差分）以实现阻抗匹配；R、L、C、G的参数决定了信道的频率响应特性。
> 【落地注意】实际PCB材料手册会给出损耗角正切tanδ和介电常数εr，可据此计算L和C；仿真工具（HFSS、ADS）可更精确地提取这些参数。

**频域(正弦稳态)传输线方程**：

dV(x)/dx = -(R + jωL)·I(x) .... (3) dI(x)/dx = -(G + jωC)·V(x) .... (4)

对(3)求导并代入(4): d²V(x)/dx² = γ²·V(x) .... (5) 其中 γ = α + jβ = √[(R + jωL)(G + jωC)]

γ: 传播常数 • α: 衰减常数 (Np/m) — 表征信号衰减 • β: 相位常数 (rad/m) — 表征相位旋转


### 2.3 传播常数与波速

**通解形式**：

V(x) = V_f·e^(-γx) + V_r·e^(+γx) = 前向行波 + 后向行波(反射)

I(x) = I_f·e^(-γx) + I_r·e^(+γx)

相位速度 v_p = ω/β = 1/√(LC) (无损情况)


**无损LC传输线(特例)**：

当 R=0, G=0: γ = jβ = jω√(LC) α = 0 (无衰减!) β = ω√(LC)

特性阻抗 Z₀ = √(L/C) (纯实数)

波速 v_p = 1/√(LC) = ω/β 信号无畸变传播，阻抗不随频率变化


### 2.4 低损耗LRC传输线(实际情况)

**近似条件**: R/ωL << 1 且 G/ωC << 1

γ ≈ jω√(LC)·[1 - j(RC + GL)/(2ωLC)] = α_R + α_D + jβ

其中: α_R ≈ R/(2Z₀) — 导体损耗(趋肤效应) α_D ≈ G·Z₀/2 — 介质损耗 β ≈ ω√(LC)·[1 + (R/ωL)²/8 + (G/ωC)²/8]

特性阻抗: Z₀ ≈ √(L/C) · [1 + j(R/(2ωL) - G/(2ωC))]


**两种损耗机制对比**：

┌─────────────────────────────────────────────────────┐ │ 导体损耗 (趋肤效应) │ │ α_R ∝ √f (与频率平方根成正比) │ │ 在高频时显著 │ ├─────────────────────────────────────────────────────┤ │ 介质损耗 (介质吸收) │ │ α_D ∝ f (与频率一次成正比) │ │ 在更高频时主导 │ └─────────────────────────────────────────────────────┘

## 三、频率依赖损耗机理

### 3.1 趋肤效应(Skin Effect)

**物理机理**：

![](lecture2_ee720_channels_images/img-018.jpg)

> 🔍 深度说明：
> 【研究背景】趋肤效应图清晰展示了高频电流在导体表面的聚集现象。皮肤深度δ随频率升高而减小，导致有效导电截面积减少，等效串联电阻增加。
> 【核心结论】在1GHz时铜的皮肤深度仅~2.1μm，这意味着20μm厚的走线在高频时只有表面2μm参与导电，电阻显著增加。
> 【工程价值】趋肤效应引起的导体损耗α_R ∝ √f，在10-20GHz频段是主要损耗来源；高速SerDes信道设计必须考虑这一效应。
> 【落地注意】降低趋肤效应损耗的方法包括：使用宽走线（增加周长）、厚铜（18-35μm）、选择趋肤深度更大的导体材料（如金）。

**矩形导体临界频率**：

f_s = 皮肤深度等于导体半高度时的频率

δ = h/2 → f_s = 4ρ / (πμ·h²) = ρ / (πμ·(h/2)²)

其中 h 为导体高度(或半径)

电阻频率特性: R(f) = R_DC · √(f_s/f) 当 f < f_s R(f) = R_DC · √(f/f_s) 当 f > f_s


**趋肤效应衰减常数**：

α_R(f) = R_DC/(2Z₀) · √(f/f_s) 当 f > f_s

典型数据: • 5-mil Stripguide: R_DC = 7Ω/m, f_s = 43MHz • 30 AWG 双绞线: R_DC = 0.08Ω/m, f_s = 67kHz


### 3.2 介质损耗(Dielectric Absorption)

**物理机理**：

交变电场作用下: • 电介质原子极化滞后于电场变化 • 极化过程吸收能量(以热量形式损耗) • 用损耗角正切tanδ表征

损耗角正切: tanδ = G/(ωC) = 介质电导/容性电纳

α_D = πf·tanδ·√(LC) = πf·tanδ / v_p

特点: • α_D ∝ f (线性增长) • 在高频时成为主要损耗机制


### 3.3 高级PCB介质材料对比

| 材料 | @56GHz损耗 | 25dB损耗对应长度 | 特点 |
|------|-----------|----------------|------|
| Megtron 6 | ~1.1dB/in | 12.5英寸 | 低损耗高速材料 |
| Tachyon | ~1.6dB/in | 15.6英寸 | 平衡性能成本 |
| PTFE (Teflon) | ~2dB/in | 22.7英寸 | 最高性能、成本高 |
| 标准PCB | 更高 | 更短 | 成本优先 |

**电缆背板应用**：

电缆背板 + 短距离子卡: • 可支持 ~1m 距离 @ 224Gb/s • 适合intra-rack互联

## 四、反射与终端匹配

### 4.1 反射系数与Telegrapher方程

**反射系数定义**：

![](lecture2_ee720_channels_images/img-028.jpg)

> 🔍 深度说明：
> 【研究背景】该图解释了反射系数的物理含义。当负载阻抗Z_T与特性阻抗Z₀不匹配时，会产生反射，反射波叠加在入射波上造成信号失真。
> 【核心结论】ρ_r = (Z_T - Z₀)/(Z_T + Z₀)，当Z_T = Z₀时ρ_r = 0（无反射）；反射系数为正时电压同相叠加，为负时反相。
> 【工程价值】在高速链路设计中，所有节点（Tx、Rx、信道、转接器）都需要阻抗匹配以消除反射；差分对的奇模/偶模阻抗匹配尤为关键。
> 【落地注意】实际信道中存在多个不连续点，多次反射会形成振铃（ringing）；使用TDR可以快速定位阻抗不连续的位置。

**格型图(Lattice Diagram)分析**：

多此反射分析工具: • 追踪多次反射的幅度和时序 • 收敛条件分析

不同终端组合的反射模式: ┌─────────────────┬────────────┬────────────────┐ │ Z_S │ Z_T │ 反射模式 │ ├─────────────────┼────────────┼────────────────┤ │ 25Ω │ 25Ω │ kr<0, kT<0 │ 电压收敛 │ │ 25Ω │ 100Ω │ kr<0, kT>0 │ 电压振荡 │ │ 100Ω │ 25Ω │ kr>0, kT<0 │ 电压振荡 │ │ 100Ω │ 100Ω │ kr>0, kT>0 │ 电压攀升 │ └─────────────────┴────────────┴────────────────┘


### 4.2 终端匹配方案

**四种基本终端方案对比**：

  1. 无终端 • 几乎无吸收入射波能量 • 可能产生振荡波形 • 适用条件: t_propagation << n·T_bit • 实际应用: 极其有限的离片场景

  2. 源端终端(Source Termination) R_S = Z_0 • 源端阻抗匹配消除源端反射 • 传输线末端全反射后被源端吸收 • 适合中速点对点连接 • 缺点: 2倍步进电压降

  3. 接收端终端(Receiver Termination) R_T = Z_0 • 接收端无反射 • 需注意中间阻抗不连续 • 驱动器端反射需在信道中衰减

  4. 双端终端(Double Termination) R_S = Z_0, R_T = Z_0 • 最佳反射抑制配置 • 高速SerDes标准方案 • 差分驱动电压幅度 = 单端的一半


### 4.3 终端电阻类型

**无源终端(Poly/N-diffusion/N-well)**：

90nm CMOS电阻选项对比: ┌──────────────┬─────────┬──────────────────┐ │ 电阻类型 │ 方块电阻 │ 寄生电容 │ ├──────────────┼─────────┼──────────────────┤ │ Poly │ 90±10Ω │ 2-3fF/μm² │ │ N-diffusion │ 300±50Ω │ 0.9fF/μm²(面积) │ │ N-well │ 450±200Ω│ 0.7fF/μm²(面积) │ └──────────────┴─────────┴──────────────────┘

选择Poly的原因: • 线性度好 • 精度较高(tighter tolerance) • 缺点: ±30% process/temp variation


**有源终端(主动FET终端)**：

三极管偏置FET架构: • 低电压摆幅(<500mV)适用 • 增加二极管连接FET增大线性范围 • Pass-gate结构实现差分终端

数字可调终端: R_FET = 1/(μC_ox·(W/L)·(V_GS - V_t))

优势: • 数字trimming补偿PVT变化 • 比模拟控制更高的线性度 • 现代SerDes标准配置

## 五、差分传输线

### 5.1 差分信号优势

![](lecture2_ee720_channels_images/img-054.jpg)

> 🔍 深度说明：
> 【研究背景】该图对比了单端和差分信号传输的原理。差分信号以两根线之间的电压差为信息载体，具有共模噪声自抵消的能力，是高速SerDes的标准接口形式。
> 【核心结论】差分信号的优势包括：1）自参考（不需要地平面）；2）共模噪声抑制；3）更大信号摆幅（差分电压=2×单端）；4）电源噪声自抵消。
> 【工程价值】在10Gb/s以上速率，差分接口几乎成为强制要求；差分对布线需严格控制耦合长度匹配（<5μm），以避免skew引入抖动。
> 【落地注意】差分对的shielding和cross-talk控制是关键；相邻差分对之间应保持3W间距（W=走线宽度），以控制串扰。引脚利用率对比：单端需要2根pin（signal+return），差分仅需2根signal（return电流在差分对内流动）。

### 5.2 奇模与偶模分析

**模式定义**：

偶模(Even Mode): • 两根线加相同电压 • 传播一种"偶模"电磁波 • 两线之间无电位差

奇模(Odd Mode): • 两根线加大小相等、符号相反的电压 • 传播一种"奇模"电磁波 • 两线之间存在电位差

物理理解: • 奇模: 两线之间存在虚拟地平面(提供返回电流路径) • 电场垂直于虚拟平面 • 磁场与虚拟平面相切


**奇偶模阻抗计算**：

平衡传输线参数: • C: 单线对地电容 • C_c: 两线间耦合电容 • L: 单线自感 • M: 两线间互感

偶模(公共模式): • C_even = C • L_even = L + M • Z_even = √[(L+M)/C]

奇模(差分模式): • C_odd = C + 2C_c • L_odd = L - M • Z_odd = √[(L-M)/(C+2C_c)]

重要关系: Z_diff = 2·Z_odd Z_cm = Z_even/2


### 5.3 差分终端拓扑

**PI型终端(Pi Termination)**：

奇模阻抗: Z_odd = R_1 || R_2 偶模阻抗: Z_even = R_1

设计公式: R_2 = 2·Z_even·Z_odd / (Z_even - Z_odd) R_1 = 2·Z_even


**T型终端(T Termination)**：

偶模阻抗: Z_even = R_2 + 2R_1 奇模阻抗: Z_odd = R_2

设计公式: R_1 = (Z_even - Z_odd)/2 R_2 = Z_odd

## 六、传输线高级主题

### 6.1 群延迟与色散

群延迟 τ_g = dβ/dω

当β与ω为非线性关系时: • 不同频率分量以不同速度传播 • 脉冲展宽(码间干扰) • 称为"色散"(dispersion)

低损耗近似: β ≈ ω√(LC)·[1 + (R/ωL)²/8 + (G/ωC)²/8]

其中二阶项导致色散: 高速数字信号包含丰富谐波分量 不同频率分量经历不同衰减和相移 → 眼图恶化


### 6.2 TDR测量原理

时域反射计(TDR): • 快速阶跃发生器 + 高速示波器 • 发射阶跃脉冲进入信道 • 观察源端电压波形(含反射)

阻抗计算: Z(x) = Z_0 · (1 + ρ_r(x))/(1 - ρ_r(x))

空间分辨率: Δx > t_r · v_p t_r: 阶跃上升时间 v_p: 相位速度

上升时间越快，分辨率越高 但: 信道会低通滤波，展宽上升时间
