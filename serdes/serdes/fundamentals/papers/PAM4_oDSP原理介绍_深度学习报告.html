PAM4 oDSP 原理深度学习报告¶

文档来源: PAM4 oDSP原理介绍（2026/2/20） 报告撰写: 深度学习分析 日期: 2026年4月

摘要¶

本报告基于原始文档《PAM4 oDSP原理介绍》进行深度技术分析与扩展撰写，系统性地阐述PAM4（Pulse Amplitude Modulation 4-level，四电平脉冲幅度调制）光学数字信号处理（oDSP）的完整技术栈。本报告从PAM4调制的基础原理出发，深入解析发射端预处理（FFE FIR滤波、预编码）、接收端信号处理（CTLE、DFE、CDR）、前向纠错编码（FEC/ KP4 RS-FEC）以及线侧光路损伤补偿等核心模块，并辅以性能分析、与NRZ/PAM2的对比、400G/800G以太网应用场景及未来发展趋势。本报告在原始文档基础上进行了大量技术细节补充与深度理论扩展，力求达到专业学术与工程实践双重标准。

第一章 PAM4调制技术概述¶

1.1 高速光通信的带宽需求与调制演进¶

随着数据中心和AI算力基础设施对带宽需求的爆发式增长，56Gbps乃至112Gbps每通道的传输速率已成为当前主流光互连标准的技术节点。在此速率下，传统的NRZ（Non-Return-to-Zero，非归零）调制即PAM2已接近电气带宽的物理极限。以28Gbps NRZ为例，其奈奎斯特频率为14GHz，而实际高速数字链路中信道插入损耗在14GHz处往往已超过20dB，这对模拟前端和时钟恢复电路构成了极为严苛的设计挑战。

PAM4通过在单个符号周期内编码2比特信息，将有效符号率降低至原来的一半，从而在给定物理带宽条件下将系统容量翻倍。这一技术选择成为从56G PAM4向112G PAM4演进过程中的关键使能技术，并在400G/800G以太网、50G/200G/400G PON以及数据中心内部光互连中得到了广泛应用。

1.2 PAM4信号基础理论¶

PAM4信号采用四个离散的幅度电平来表示两比特二进制数据。四电平分别对应逻辑电平00、01、11和10（采用格雷码映射以最小化相邻电平间的比特错误概率）。四个电平的标称幅度通常归一化为-3、-1、+1、+3（也有实现采用0、1、2、3的线性映射）。

PAM4信号的眼图呈现为三个并行的"眼睛"——上眼、中眼和下眼。与NRZ的单一眼图不同，PAM4的三个眼图在垂直方向上共享总信号动态范围，每个眼的开口高度（眼高）约为NRZ的约三分之一。这意味着PAM4对噪声和串扰的容忍度较NRZ显著降低，对系统的线性度和动态范围提出了更高要求。

从信息论角度分析，PAM4每符号携带2比特信息，其谱效率是NRZ的两倍。然而，PAM4的最小欧氏距离（相邻电平间的幅度差）为NRZ的约1/3（假设同样的峰值功率约束下），这直接导致PAM4的符号间干扰（ISI）容忍度下降约9.54dB（20log10(1/3) ≈ -9.54 dB），这正是PAM4系统需要更强大的均衡技术和纠错能力来补偿的根本原因。

第二章 oDSP系统架构总览¶

2.1 光数字信号处理流程¶

oDSP（optical Digital Signal Processing，光数字信号处理）是高速光传输系统中将模拟光信号转换为可用数字域信号并进行智能处理的核心技术模块。与传统的模拟均衡方案相比，oDSP在数字域实现精确的线性与非线性均衡，具备自适应配置和算法迭代升级的灵活性。

完整的PAM4 oDSP收发链路架构如下：

发射端（TX）链路： 数据流 → Gray编码 → 预编码(Precoder) → FIR/FFE滤波 → DAC → 激光调制器

接收端（RX）链路： 光探测器 → TIA → CTLE → ADC → FFE(前馈均衡) → DFE(判决反馈均衡) → CDR(时钟数据恢复) → Gray解码 → 数据流

上述架构中，TX端与RX端各包含多个关键信号处理模块，原始文档在第3页给出了清晰的系统架构总览（Analog TX architecture中的FFE EQ、RX端的CTLE、FFE adapt、DFE adapt、CDR等）。

2.2 模拟发射架构与DAC架构对比¶

原始文档区分了两种发射端架构：

纯模拟TX架构（Analog TX architecture）：在模拟域完成FIR滤波和均衡功能，适用于对功耗和延迟有极端要求的短距离场景。该架构中FIR滤波器以模拟电路（连续时间滤波器或开关电容网络）实现，配置灵活度低但功耗最优。

基于DAC的TX架构（DAC-based TX architecture）：先将数字信号经过数字FIR滤波器，再通过高速DAC转换为模拟信号驱动调制器。该架构利用数字电路的高精度和可编程性，能够实现复杂的预均衡策略（如预加重/去加重），且同一硬件平台可通过配置适应不同信道条件，是当前主流方案。

第三章 发射端预处理技术¶

3.1 FIR滤波与发送端前馈均衡（TX FFE）¶

3.1.1 FIR滤波器原理¶

发送端FIR（Finite Impulse Response，有限脉冲响应）滤波器是最常用的发射端预均衡技术。原始文档第4页给出了2-tap和3-tap FIR的数学表达：

🔍 深度说明：TX FFE FIR滤波器原图解析 - 图片来源: PAM4 oDSP原理介绍_images/img-004.jpg（55KB）和img-005.jpg（24KB） - 工程价值: 该图直观展示了2-tap（main+post）和3-tap（pre+main+post）FIR的结构差异，以及9.7dB预均衡效果的可视化 - 关键洞察: pre-cursor系数c(-1)用于补偿前一个符号对当前符号的干扰，这在高速背板中尤为关键——相邻通道的远端串扰（far-end crosstalk）会产生pre-cursor ISI，只能通过发送端pre-cursor EQ消除；main-cursor系数c(0)决定信号主体强度；post-cursor系数c(1)补偿当前符号对后续符号的干扰 - 设计启示: 9.7dB的2-tap FIR能在13dB插入损耗链路上实现有效预均衡，说明FIR tap数的选择应基于信道脉冲响应测量——tap系数不为零的个数决定了所需FIR阶数

y(n) = c(-1) * x(n) + c(0) * x(n-1) + c(1) * x(n-2)

其中，c(-1)为前标（pre-cursor）系数，c(0)为主标（main-cursor）系数，c(1)为后标（post-cursor）系数。在3-tap FIR中，x(n)为当前符号，x(n-1)为前一符号（pre-cursor），x(n+1)为后一符号（post-cursor）。

其Z域传递函数为： H(z) = c(-1) + c(0) * z^(-1) + c(1) * z^(-2)

FIR滤波器的核心作用是在发送端对信道的高频衰减特性进行预补偿，使得经过信道后的总体响应趋于平坦。对于典型的PCB背板信道，其插入损耗随频率近似按f^0.5或f^1增长，在14GHz处的损耗可能高达15-20dB。通过在发送端施加与信道频率响应相反的预加重（FIR滤波器的高频系数大于低频系数），可以有效展宽信号的高频谱能量，补偿信道引起的频率相关性衰减。

3.1.2 FIR系数设计¶

FIR系数的确定有两种主要方式：固定系数设计（依据典型信道模型预设）和自适应系数调整（通过链路训练获取最优系数）。原始文档第4页给出的9.7dB 2-tap FIR（仅含main和post）示例，说明了对于一个13dB插入损耗的链路如何通过FIR滤波在发送端实现约9.7dB的预均衡效果。实际设计中，FIR的tap数（2-tap、3-tap、5-tap乃至9-tap）取决于信道的长度、反射特性以及功耗预算。

3.2 预编码（Precoding）¶

预编码是PAM4系统中一项重要的发送端信号处理技术，其核心目的是将信道引入的符号间干扰（ISI）转化为已知的、可通过接收端处理消除的确定性干扰模式。

PAM4信号的四个电平（-3, -1, +1, +3）在经过存在ISI的信道后，相邻符号之间会产生特定模式的码间干扰。预编码器在发送端对数据进行预变换，使得接收端在简单的逐符号判决后即可恢复原始数据，无需迭代ISI消除。

典型的预编码方案为模二加（modulo-2）预编码。对于二进制数据流b_n，预编码输出p_n = b_n XOR p_(n-1)（XOR表示异或运算）。在接收端，恢复数据为 b_hat_n = p_hat_n XOR p_hat_(n-1)。该方案将信道导致的"差错传播"特性转化为"差错累积"特性，使得误码模式更为集中，有利于FEC的纠错处理。

第四章 接收端信号处理¶

4.1 连续时间线性均衡器（CTLE）¶

4.1.1 CTLE工作原理¶

CTLE（Continuous Time Linear Equalizer，连续时间线性均衡器）是接收端模拟域的第一个线性均衡模块，位于TIA（跨阻放大器）之后和ADC之前。CTLE本质上是一个频率响应可调的模拟高通滤波器或带通滤波器，其频率响应可表示为：

H_CTLE(s) = (omega_z / (s + omega_z)) * ((s + omega_p) / (s + alpha * omega_p))

其中omega_z为零点频率，omega_p为极点频率，alpha为直流增益控制参数。通过调节这些参数，CTLE可以在特定频率范围内提升增益（零点处开始提升，极点处达到峰值），从而补偿信道的低通特性。

原始文档第6页指出"CTLE code is determined by FFE frequency spectrum"，这表明CTLE的均衡深度（均衡代码）与FFE的频率响应特性紧密耦合。在实际系统中，CTLE首先对宽频带噪声和信道失真进行初步均衡，降低接收信号的高频噪声含量，为后续的数字FFE和ADC创造更好的信号条件。

4.1.2 CTLE与眼图张开度¶

CTLE的核心工程目标是"打开眼图"——即恢复因信道带宽限制而模糊的PAM4三个眼图的垂直开口度。CTLE的高频提升效果直接增加了信号在高频段的幅度，从而压缩因群时延波动和频率选择性好导致的符号展宽。但CTLE增强高频的同时也会放大高频噪声，因此CTLE的均衡深度需要在信号增强与噪声放大之间取得工程折中。

4.2 前馈均衡器（FFE）与自适应算法¶

4.2.1 FFE数学模型¶

FFE（Feed-Forward Equalizer，前馈均衡器）的核心是一个抽头延迟线滤波器，其输出为：

Y_out = sum(H_k * X_in(n-k)), k从-N到M

其中H_k为各抽头的复数或实数系数，X_in为输入信号序列，N和M分别为前向和后向抽头的数量。FFE在数字域实现，其抽头系数可通过训练序列或盲均衡算法自适应调整。

原始文档第7页给出了FFE均衡的简洁表达： Y_out = sum(X_in * H)

这表明FFE本质上是一个线性卷积过程，通过对输入信号与信道冲激响应进行反卷积来补偿信道的频率选择性衰落。

4.2.2 CMA盲均衡算法¶

CMA（Constant Modulus Algorithm，常模算法）是一种无需训练序列的盲均衡算法，广泛应用于接收端FFE系数的初始收敛阶段。CMA的核心代价函数为：

J_CMA = (|Y_out|^2 - R)^2

其中R为目标幅度常数（对于PAM4，需根据四个电平的幅度分布设定适当的目标值）。CMA通过梯度下降法更新系数：

H(n+1) = H(n) + mu * X_in * Y_out * err

其中err = R - |Y_out|^2为幅度误差，mu为步长参数。

CMA的优势在于不依赖训练序列即可在链路建立初期快速收敛，但CMA存在收敛速度慢、稳态误差大、对载波频率偏移敏感等固有缺陷。因此原始文档指出"CMA performs coarse equalization based on the average energy of signal"——CMA基于信号的平均能量进行粗均衡，主要用于系数初始值的获取。

4.2.3 DDLMS决策导向自适应算法¶

DDLMS（Decision-Directed Least Mean Square，决策导向最小均方）算法是FFE系数精细调整阶段的核心算法。与CMA不同，DDLMS在判决器（slicer）输出硬判决结果后，以判决结果作为参考信号进行有监督的自适应更新：

H(n+1) = H(n) + mu * X_in * err

其中err = Y_out - Y_DDLMS_out，Y_DDLMS_out为判决器的输出（硬判决结果）。

DDLMS的收敛精度远高于CMA，因为判决结果提供了比幅度统计更为精确的参考信息。原始文档指出"DDLMS performs precise equalization based on the decision target of signal"——DDLMS基于信号的判决目标进行精确均衡。这两个算法通常级联使用：CMA先完成粗收敛，DDLMS再进行精确跟踪，形成完整的两阶段自适应均衡体系。

4.3 判决反馈均衡器（DFE）¶

4.3.1 DFE原理与优势¶

DFE（Decision Feedback Equalizer，判决反馈均衡器）是接收端最关键的非线性均衡模块之一。DFE的核心思想是利用已判决符号的信息来消除当前符号所受到的来自已判符号的码间干扰（post-ISI），而不会将这种消除过程产生的噪声放大。

DFE的结构包含两个基本组件：一个前馈滤波器（通常与FFE共享或合并实现）和一个反馈滤波器。反馈滤波器的输入为已判决符号序列，其输出被从当前符号的均衡结果中减去：

Y_out = X_in + sum(X_in * H_k^FB), k从1到N

原始文档第8页给出了DFE的精确数学表达并配有10G背板的脉冲响应实例：h1 = 0.175（相邻符号的ISI系数），h2 = 0.075（次相邻符号的ISI系数）。对于2-tap DFE，位流"11011"中比特"0"的DFE输出为0.375 - 0.075 - 0.175 = 0.1，可被正确判决为"0"。这说明DFE能够有效消除已知的post-ISI分量。

4.3.2 DFE与FFE的关键差异¶

DFE相对于FFE的核心优势在于：FFE在抑制post-ISI时必须同时放大噪声功率（因为FFE是一个纯线性滤波器，提升高频增益的同时必然提升该频段的噪声），而DFE利用已判决符号的确定信息来消除post-ISI，不会额外放大噪声。这一特性使DFE在高损耗信道（如长距离背板或光纤传输）中成为不可或缺的均衡模块。

然而DFE也有其固有限制：一旦发生判决错误，错误会通过反馈滤波器传播到后续符号的均衡结果中，造成错误传播（error propagation）。在PAM4系统中，由于眼图开口较小，判决错误概率高于NRZ系统，因此DFE的错误传播问题更为突出。解决方案包括：采用软判决DFE、限制反馈滤波器的深度（tap数）、以及在DFE前设置足够强的FFE/线性均衡以降低判决错误概率。

4.4 baud-rate CDR与Mueller-Muller定时恢复¶

4.4.1 CDR基本原理¶

CDR（Clock and Data Recovery，时钟数据恢复）是接收端最核心的时序同步模块。CDR从收到的数据流中提取时钟相位信息，并将ADC的采样时刻对准每个符号的最佳采样点（眼图中央）。PAM4系统对CDR的要求比NRZ更为严苛——PAM4的三个眼图中每个眼图都需要在最佳时刻采样，且三个眼图的垂直偏移不同，CDR的相位锁定精度要求更高。

4.4.2 Mueller-Muller算法¶

原始文档第9页介绍了baud-rate CDR中广泛使用的Mueller-Muller（MM）定时恢复算法。MM算法的核心原理是从baud-rate采样（每个符号仅一个采样点）的数据中推断信道冲激响应，然后据此调整采样时钟相位使其对齐到脉冲响应的中心位置。

MM算法的自适应更新规则为：

  • 若h(t_k - T_b) < h(t_k + T_b)，则CDR采样时刻过早（early），需要将采样时钟向后调整；
  • 若h(t_k - T_b) > h(t_k + T_b)，则CDR采样时刻过迟（late），需要将采样时钟向前调整。

第五章 前向纠错编码（FEC）¶

5.1 FEC基本原理¶

FEC（Forward Error Correction，前向纠错编码）通过在数据流中引入冗余比特，使接收端能够在不依赖重传的情况下纠正一定范围内的错误比特。FEC编码将k个信息符号映射为n个编码符号（n > k），形成(n, k)分组码。编码开销率（overhead）为(n-k)/k。

原始文档第11页给出了KP4 FEC（Key Performance 4-level FEC）的标准定义：

  • RS(528, 514, T=7, M=10)：用于IEEE P802.3bj 25G NRZ的KR4 FEC，纠错半径T=7；
  • RS(544, 514, T=15, M=10)：用于28G/56G/112G PAM4的KP4 FEC，纠错半径T=15。

其中RS为Reed-Solomon（里德-所罗门）码，M=10表示符号宽度为10比特，T表示最大可纠正的符号错误数。RS(544, 514)将514个10-bit信息符号编码为544个10-bit编码符号，每个码字可纠正最多15个符号错误。

5.2 KP4 FEC的纠错能力分析¶

原始文档第11页指出KP4 FEC在不同错误分布下的纠错效果差异极大：

最优情况：KP4 FEC在5440比特（544符号 x 10比特/符号）的码字中可纠正多达150个错误比特。这要求错误高度集中分布在少数几个符号内，使得每个符号的错误比特数不超过15（RS码的纠错能力）。

最差情况：KP4 FEC仅能纠正15个错误比特——即当16个比特错误分散在16个不同的10比特符号中时，由于每个符号的错误比特数均不超过1（远低于T=15的阈值），但符号错误数达到16个，超过了RS(544, 514, T=15)的符号纠错上限。

这说明FEC的实际纠错效果不仅取决于总错误比特数，更关键的是错误的分布特性。突发错误（burst error）模式对FEC更为友好，而分散错误（random error）模式对FEC的挑战更大。

5.3 RS解码器架构¶

原始文档第12页给出了RS解码器的四步经典架构：

第一步：伴随式计算（Syndrome Computation） s = [s_1, s_2, ..., s_{2t-1}, s_{2t}] = H x R^T = H x E^T

通过计算接收矢量与校验矩阵的乘积，得到2t个伴随式分量。若所有s_i = 0，则判决无错误；否则伴随式向量将引导后续的错误值查找过程。

第二步：BM算法（Berlekamp-Massey） lambda(x) = (1 - P_1 * x)(1 - P_2 * x)...(1 - P_v * x), v <= t

BM算法从伴随式向量中恢复错误位置多项式lambda(x)和错误值多项式Omega(x)，确定错误发生的符号位置v和错误幅度。

第三步：Chien搜索（Chien Search Algorithm） x = a^n, a^(n-1), ..., a^2, a^1

将有限域GF(2^10)中的各元素逐一代入错误位置多项式，找到使lambda(x) = 0的根，即为错误符号的位置。

第四步：Forney算法 利用错误位置多项式和错误值多项式，计算各错误位置上正确的符号值，完成错误修正。

5.4 编码增益（Coding Gain）¶

编码增益是衡量FEC性能的核心指标，定义为在达到相同BER目标的条件下，有FEC时所需的SNR相对于无FEC时所需的SNR的减少量（dB）。原始文档第13页给出了KP4 FEC的编码增益计算：

C_G = Q_ref - Q_in = 20 * log10(2 * erfcinv(2 * 2e-15)) - 20 * log10(2 * erfcinv(2 * 2.4e-4)) = 7.04 dB

  • KR4 FEC（RS(528, 514)）编码增益约5.63 dB
  • KP4 FEC（RS(544, 514)）编码增益约7.04 dB

KP4 FEC比KR4 FEC高出约1.4 dB的编码增益，这主要来源于KP4更大的纠错半径（T=15 vs T=7）和更大的符号宽度（M=10）。

6.2 激光调制器啁啾（Laser Chirp）¶

啁啾是激光器在调制过程中由于载流子浓度变化引起的瞬时频率偏移现象。原始文档第17页将激光调制器分为三类：

MZM（Mach-Zehnder调制器）：双臂干涉结构，理论上无啁啾（chirp-free），是长距离高速光传输的首选方案。

EML（Electro-absorption Modulated Laser，电吸收调制激光器）：主要受瞬态啁啾（transient chirp）影响。当调制信号使吸收边发生快速变化时，输出光频率随之产生短暂的偏移。瞬态啁啾与调制信号的上升/下降沿强相关。

DML（Directly Modulated Laser，直接调制激光器）：同时受瞬态啁啾和绝热啁啾（adiabatic chirp）影响。绝热啁啾是调制电流持续变化时激光器输出频率的连续偏移，与调制信号的幅度直接相关。DML的啁啾特性最为复杂，在长距离光纤传输中会与色散效应相互作用，产生额外的功率衰落。

在高速PAM4系统中，啁啾会导致信号频谱的相位失真，与色散效应共同恶化信号质量。数字域补偿通常需要引入非线性均衡器（如Volterra滤波器）来建模和消除啁啾效应。

6.3 色度色散（Chromatic Dispersion, CD）¶

色度色散是光纤中不同波长光信号传输速度不同的物理现象。原始文档第18页给出了色散的定义、影响和数学模型：

定义：不同频率的信号具有不同的传输速率。

影响一（符号展宽）：色散导致不同频率分量以不同速度传播，使脉冲在时域展宽，产生符号间干扰（ISI）。这一效应对高速PAM4信号的影响尤为严重，因为PAM4的眼图开口垂直方向本就较NRZ更小，ISI的叠加会进一步闭合眼图。

影响二（功率衰落）：色散导致不同频率分量具有不同的频率域相位。在IM/DD（强度调制/直接检测）系统中，平方律检测将双边带信号的上、下两个边带相乘，产生拍频项。由于色散使上、下边带的相位差随频率变化，平方检测后的信号功率出现周期性衰落：

P(dB) = 10*log10(|sin(phi)/phi|), phi = D_z * f * lambda * pi

其中D_z为色散系数，f为信号频率，lambda为光波长。当phi接近pi的整数倍时，sin(phi)/phi趋近于零，出现深度功率衰落（power fading）。

在数字域中，色散补偿通常通过有限脉冲响应（FIR）滤波器或更复杂的自适应均衡器实现。色散引起的频率相关相位旋转可以通过在数字域引入与频率成正比的相移来精确补偿，即数字色散补偿滤波器。

6.4 多径干涉（MPI）¶

MPI（Multipath Interference，多径干涉）是指光信号沿多条不同路径到达接收端时，各路径信号因时延差和相位差叠加而产生的干涉现象。原始文档第19页特别指出光MPI与电反射的关键差异：

  • 电反射的受影响符号长度通常较短（几十至几百UI，取决于反射点的位置）
  • 光MPI的受影响符号长度特别长，可达约10000 UI（对于10 Gbaud符号率，相当于1ms的时延扩展）

MPI导致的符号间干扰持续时间极长，传统的浮点FIR（有限长度均衡器）无法有效补偿如此长时延的干扰。这要求oDSP系统采用更长的均衡器抽头或引入专门的MPI补偿算法，如迭代干扰消除（Successive Interference Cancellation, SIC）或多输入多输出（MIMO）均衡。

6.5 线侧均衡方案¶

原始文档第20页给出了线侧（Line-side）光路均衡的三种主要技术方案：

DFE（判决反馈均衡器）：利用已判决符号消除当前符号所受的ISI。优点是不放大噪声，缺点是存在错误传播。对于MPI等长时延干扰，DFE需要极多的前向和反馈抽头，实现复杂度高。

FFE（前馈均衡器）：直接对接收信号进行线性滤波补偿。FFE的优点是实现简单、无错误传播，但消除长时延干扰需要极多的抽头数，且抽头系数收敛困难。

MLSE（最大似然序列估计）：在理论上最优的均衡方案，通过动态规划算法（Viterbi算法）在整个接收序列中搜索最可能的发送序列。MLSE能够同时处理短时延和长时延的ISI，但计算复杂度为O(2^M)（M为记忆长度），对于PAM4和长记忆信道实际上不可行。实际系统通常采用Truncated MLSE或Volterra-MLSE等简化方案。

Volterra FFE：基于Volterra级数的非线性均衡器，能够建模和分析包括激光啁啾、光纤非线性、调制器非线性在内的多种非线性损伤。Volterra滤波器的核（kernel）系数通过自适应训练获取，是应对复杂光损伤的关键技术。

第七章 性能分析¶

7.1 信噪比（SNR）分析¶

PAM4系统的性能分析以信噪比（SNR）为核心指标。对于PAM4信号，其信号功率与符号的均方值成正比：

P_signal = (1/4)[(-3)^2 + (-1)^2 + (1)^2 + (3)^2] = 5

噪声功率N包括热噪声、散粒噪声、量化噪声和RIN（相对强度噪声）等各种噪声源的叠加。在总功率约束下，PAM4的信号噪声比相对于NRZ有所降低——由于同样的峰值功率约束下，PAM4的相邻电平差为NRZ的1/3，因此PAM4的SNR代价约为：

delta_SNR_PAM4vsNRZ = 20*log10(3) ≈ 9.54 dB

这意味着在相同的信道噪声功率密度下，PAM4的误码率性能比NRZ差约9.54 dB，需要通过更强的均衡和FEC来弥补这一差距。

7.2 误码率（BER）与Q因子¶

BER（Bit Error Rate，误码率）是衡量通信系统性能的最直接指标。PAM4系统的BER可以通过Q因子来估算：

BER ≈ (3/4)*erfc(Q/sqrt(2))

其中Q = delta/(2*sigma)，delta为相邻电平间的最小间距，sigma为噪声标准差。

KP4 FEC将系统BER从FEC输入侧的约2.4e-4（纠错阈值）降低到输出侧的10^-15量级，这一约11个数量级的改善正是FEC的核心价值所在。原始文档第14页的编码增益数据表明，最强的OFEC方案可达到11.1 dB的编码增益，对应的FEC输入阈值BER高达1.93e-2，说明超强编码方案能够容忍极为恶劣的信道条件。

7.3 眼图分析¶

眼图是评估PAM4信号质量的综合工具。在示波器上叠加多个PAM4符号的眼图可以同时显示三个眼图的开合程度、抖动分布和幅度噪声。

关键眼图指标： - 眼高（Eye Height）：三个眼图中各自垂直开口的高度，直接反映该眼图的噪声容限 - 眼宽（Eye Width）：时域上的采样窗口宽度，反映定时抖动的大小 - 眼图交叉点偏移：反映占空比失真和偏置误差 - 眼图线性度：四个电平之间的间距是否均匀，影响格雷码映射的有效性

PAM4的三个眼图中，中间眼的线性度和张开度通常最差，因为中间眼同时受到上下两个方向上的码间干扰影响，噪声累积最为严重。

第八章 PAM4在400G/800G以太网中的应用¶

8.1 400G以太网光接口标准¶

400G以太网（IEEE 802.3bs）定义了多种光物理层规范，其中PAM4是核心调制方案：

400GBASE-LR8：采用8波长×50G PAM4，总速率400G，使用LWDM（长波长波分复用），传输距离可达10km。

400GBASE-DR4：采用4波长×100G PAM4，使用PSM4（并行单模光纤）技术，传输距离500m。

400GBASE-FR8：8波长×50G PAM4，传输距离2km（细粒度可达到8×50G的灵活栅格）。

在这些标准中，每个波长通道承载56 Gbaud或106 Gbaud的PAM4信号，符号率约为28 GBaud或53 GBaud。oDSP在这些应用中负责完成全部的数字信号处理功能，包括发射端的数字预均衡、接收端的自适应均衡、CDR和FEC。

8.2 800G以太网与112G PAM4¶

800G以太网（IEEE 802.3ck及后续标准）将每通道速率提升至112 Gbps，符号率相应提升至56 Gbaud。112G PAM4对系统的带宽、线性度、均衡能力和FEC性能都提出了更高要求：

112G PAM4的技术挑战： 1. 带宽压力：56 Gbaud的PAM4信号的3dB带宽约为28 GHz，对DAC/ADC、调制器、探测器的带宽要求极为严苛 2. ADC分辨率：PAM4眼图开口仅为NRZ的约1/3，要求ADC有效位数达到6.5-7 bit以上 3. 均衡深度：56 Gbaud下信道的高频损耗更为严重，需要更长的FFE/DFE抽头 4. CDR精度：更高的符号率意味着相同的相位抖动对应更大的时间误差 5. FEC设计：112G PAM4的原生BER更高，需要编码增益更强的FEC方案

原始文档指出RS(544, 514, T=15, M=10)（KP4 FEC）是112G PAM4的标准FEC方案，对应IEEE P802.3ck标准。

8.3 共封装光学（CPO）与oDSP的协同¶

800G及更高速率系统的一个重要技术趋势是共封装光学（Co-Packaged Optics, CPO）。CPO将光引擎（光模块）与交换芯片在物理上紧耦合共封装，大幅缩短电气走线长度，从而将高速电气互连的损耗从传统可插拔光模块方案的20+dB降低到5dB以下。

在CPO架构中，oDSP的功能定位发生了重要变化： - 由于电气互连损耗大幅降低，发射端的预均衡强度可以降低，节省TX功耗 - 接收端的均衡复杂度也可能降低，但ADC采样率和分辨率要求进一步提升 - oDSP的功耗和面积成为系统功耗预算的关键约束，需要在算法性能和实现效率之间进行更深入的优化

第九章 PAM调制技术未来趋势¶

9.1 PAM6/PAM8的探索¶

随着单通道速率向200G乃至400G演进，业界已开始研究更高阶的PAM调制格式，如PAM6（每符号3比特，64电平）和PAM8（每符号3比特，256电平...实际上PAM8为8电平对应每符号log2(8)=3比特）。但更高阶的调制面临严峻挑战：

  • 最小电平间距进一步缩小（以PAM8为例，相邻电平间距仅为NRZ的约1/15），对噪声和线性度的要求几乎不可接受
  • 需要ADC/DAC分辨率达到9-10 bit以上，实际系统实现难度极大
  • FEC的纠错负荷急剧增加，需要接近香农极限的极化码或LDPC码来弥补

因此，PAM6/PAM8更可能作为短距离（如芯片间互连）的特殊应用方案，而非远距离光传输的主流技术。

9.2 概率整形（Probabilistic Shaping）与几何整形（Geometric Shaping）¶

为了在不增加符号率的情况下提升谱效率，概率整形和几何整形技术受到了广泛关注：

概率整形（PS）：通过非均匀地分配各符号的出现概率（使低幅度符号出现概率更高），使信号星座图的有效SNR得到提升。配合64QAM等高阶调制格式，概率整形可在相同的平均功率下实现接近香农极限的性能。

几何整形（GS）：优化各符号点在星座空间中的位置，使各符号点之间的最小欧氏距离最大化。几何整形可以针对特定信道条件定制最优星座图。

这些技术已在有线通信（如CableLabs的DOCSIS 4.0）和部分光通信实验系统中得到验证，预计将在下一代800G/1.6T光互连中与PAM4结合使用。

9.3 数字孪生辅助的自适应oDSP¶

未来的oDSP系统将更多地借助数字孪生（Digital Twin）技术来实现信道状态估计和均衡器的实时优化。数字孪生通过对真实信道进行高精度的数字建模，在离线状态下完成最优均衡器系数的搜索，再将结果部署到实际系统中。这可以显著减少在线训练时间，并支持更复杂的均衡算法（如全数字域的MLSE或神经网络均衡器）。

9.4 神经网络均衡器¶

近年来，深度神经网络（DNN）均衡器和递归神经网络（RNN/LSTM）均衡器在光通信系统中的应用研究日益活跃。与传统线性均衡器相比，神经网络均衡器能够建模任意形式的非线性损伤（如激光啁啾、放大器非线性、光纤Kerr效应等），其表达能力远超Volterra滤波器。实验结果表明，基于DNN的均衡器在长距离相干光通信系统中可获得1-2 dB的额外增益。

在PAM4直接检测系统中，神经网络均衡器已开始被用于替代或增强传统的FFE/DFE结构，其优势在于： 1. 自动学习信道中的复杂非线性特性，无需人工设计均衡器结构 2. 可通过迁移学习将在一个信道上训练的模型快速适配到相似信道 3. 与传统FEC协同设计时，可将均衡器输出直接送入神经网络译码器

第十章 总结与展望¶

本报告系统性地阐述了小鹏PAM4 oDSP的核心技术体系。从调制基础出发，PAM4通过在单个符号周期内编码2比特信息，在相同带宽条件下将系统容量翻倍，是56G/112G乃至224G高速光互连的核心使能技术。然而，更高的谱效率以约9.54 dB的SNR代价换取，这要求oDSP系统具备强大的信号处理能力。

在发射端，FIR预滤波和预编码技术通过预补偿和码间干扰变换，为接收端处理创造有利条件。在接收端，CTLE、FFE和DFE构成三级级联均衡体系：CTLE在模拟域进行初步线性均衡，FFE通过CMA和DDLMS两阶段自适应算法完成精细线性均衡，DFE利用已判决信息消除post-ISI而不放大噪声。Mueller-Muller CDR在baud-rate采样条件下恢复精确的时钟相位。

在纠错领域，KP4 FEC（RS(544, 514, T=15)）以7.04 dB的编码增益将系统BER从10^-4量级改善到10^-15量级，为高速PAM4系统提供了可靠的数据完整性保障。随着速率向112G/224G演进，更高编码增益的OFEC（11.1 dB）和双层编码方案（KP4+Hamming）将成为主流。

在光路损伤补偿方面，色散、啁啾和MPI等损伤的建模与均衡是oDSP算法设计的核心难点。Volterra FFE和MLSE等高阶均衡方案在应对复杂光损伤时展现出独特的优势，但也面临计算复杂度和功耗的工程约束。

展望未来，PAM4将在400G/800G以太网中持续演进，同时更高阶的调制格式（如PAM6/PAM8）和智能均衡技术（神经网络均衡器、数字孪生辅助优化）将成为下一代oDSP的研究热点。随着共封装光学（CPO）和片上光网络技术的发展，oDSP的功耗和集成度将成为决定技术路线成败的关键因素。
